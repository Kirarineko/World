
# 本地泡基础设定

> 本泡内一切地理、政治、社会、历史设定的总索引。


## 行星与星系

| 条目 | 设定 |
|------|------|
| 星系 | 本泡包含 **7 颗恒星**，散布于约 4–15 光年尺度内。其中 1 颗为双星系统，其余 5 颗为单星（含母星） |
| 恒星组成 | ① **母星**（太阳）：主行星所环绕的恒星 ② **双星系统**（2 颗） ③ **其他单星**（4 颗） |
| 恒星间距 | 4–15 光年，构成一片松散的物理星团 |
| 主行星 | 地球（暂用名，可改） |
| 卫星 | 月球 |
| 泡壁位置 | 远在探测器可及范围之外，包裹整个星团。从未有人类造物触碰泡壁 |
| 星空 | **双重光源**：① 真实恒星（7 颗中的 6 颗在夜空中肉眼可见）② 高斯泼溅光点（附着于泡壁内表面，不旋转不移动，昼夜均存在） |
| 星空可见性 | 真实恒星受地区光污染影响，高斯泼溅光点全天存在但被阳光遮蔽 |
| 星空认知 | 人类认为所有星星都是真实恒星。高斯泼溅光点因映射真实恒星，视差、自行、光谱皆正常，天文学层面**无法区分**。无人知晓泡壁或高斯泼溅的存在 |

| 条目 | 设定 |

| 自定位 | **完全不自知身处泡中** |


## 智慧断崖与本地泡

整个泡宇宙存在一条普遍的生物学规律：智力在"猫"附近有一道无法跨越的断层，能建立文明的智慧物种在整个宇宙中屈指可数。但本地泡是这条规律的一个例外。海豚、鸦科、大象等物种在本泡中展现出的智力水平，在其他泡中极为罕见。

> 详细机制见 `镜像机制.md`。此处仅记录本泡的特殊性。

原因来自镜像机制的底层差异。运行镜像需要神经计算资源：引用计数维护、差分层映射、块释放与垃圾回收。本泡的镜像系统相比其他泡是**阉割版**：纠错更弱，引用计数精度更低，块释放更粗糙。这套阉割的镜像节省了大量神经资源——这些资源被重新分配给了通用认知能力。

结果是本泡的生物普遍拥有比其他泡更高的智力上限，代价是更短的寿命和更高的缺陷累积率。人类的纠错"只够活到生育年龄"，在泡宇宙中属于最低的一档。大多数泡的生物活得更久、缺陷更少，但永远无法点出文明这棵技能树。

本泡不是更好的泡。它只是做了一次不同的资源预算：用镜像的完整性换了智力的可能性。

> 大多数泡的生物是长寿但愚钝的文件服务器。本地泡的生物是短命但聪明的瘦客户端。

| 条目 | 说明 |
|------|------|
| 智慧断崖基准 | 猫。智力在此附近存在全宇宙普遍断层 |
| 本地泡例外 | 海豚、鸦科、大象等智力远超断崖基准 |
| 根本原因 | 镜像机制阉割：弱纠错 → 释放神经资源 → 更高智力上限 |
| 代价 | 更短的寿命、更高的缺陷率、更弱的纠错能力 |

### 边陲

| 条目 | 设定 |
|------|------|
| 地理 | 不可居地带：沙漠、冰原、极端山地等 |
| 居民 | 大部分已适应严酷环境，能勉强或正常生存 |
| 政治状态 | 部分聚落自立为国。主导国家未予承认，也未出兵：无资源价值 |
| 详细信息 | 见 `边陲与聚落.md` |


## 力量社会

| 条目 | 设定 |
|------|------|
| 模式 | **公开**。社会知道武者和魔法师的存在 |
| 修炼门槛 | 大部分人无法开始修炼（魔力吸收的生理门槛 |
| 社会感受 | 像职业体育）知道，尊重，但有距离感 |
| 跨越者 | 近20年和平共存，接受度走高。绝对数量极少，普通人一生难遇 |
| 残余阻力 | 洁癖派以碳基纯洁性为底层逻辑，保有政治与文化影响力 |


## 魔力与科技

| 条目 | 设定 |
|------|------|
| 能量名称 | **洋能**：亚空间渗透能量。本地俗称"魔力" |
| 科技基准 | 等同于21世纪 |
| 魔力参与度 | 较低。魔力依赖个人体质，无法标准化量产 |
| 魔力储能 | 能量密度比锂电池高约20%。需人工灌魔，产能受限 |
| 魔力武器 | 存在，但占比极小。特种装备级别 |
