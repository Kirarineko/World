
# 本地泡基础设定

## 行星与星系

| 条目 | 设定 |
|------|------|
| 星系 | 本泡有 **7 颗恒星**，散落在约 4–115 光年的尺度里。其中 1 颗是双星系统，其余 5 颗是单星 |
| 主行星 | 地球 |
| 卫星 | 月球 |
| 泡壁位置 | 远在探测器可及范围之外，包裹整个星团。人类造物还没碰到过它 |
| 星空 | 真实恒星（7 颗中的 6 颗在夜空中肉眼可见）加上泡壁外的高斯泼溅形成的光点 |
| 星空认知 | 人类一直以为看到的星星都是真实恒星。因为映射关系，高斯泼溅光点在视差、自行、光谱上都像真的，天文学上也分不出来。|


## 宇宙认知

| 条目 | 设定 |
|------|------|
| 宇宙模型 | 仍然按传统大爆炸宇宙学理解。泡模型不在任何主流学术框架里 |
| 探测器极限 | 还飞不出太阳系，也没碰到过什么异常边界 |
| 亚空间 | 不知道它存在。|
| 自定位 | **完全不觉得自己活在泡里** |


## 智慧断崖与本地泡

整个泡宇宙里，智力在“猫”附近有巨大的断崖。能把文明搭起来的智慧物种，少得离谱。本地泡偏偏是少数例外之一。

> 详细机制见 `镜像机制.md`。

原因出在镜像。镜像要吃神经资源，得拿去维持引用计数、差分层、块释放这些东西。别的泡大多把资源砸进镜像里，生命力普遍强，但脑子也就那样。本地泡精简了镜像，省下来的资源拿去养认知，提高了平均下限。

代价就是寿命短，更容易衰老。

## 国家

单个巨型国家，约60年前沉渊战争时形成，直到20年前名为人类联合国，此后改名为地球联合国。

社区的态度没有太大变化，能够包容跨越者的城市/社区还是一只手能数过来，大部分的虽然消除了一些偏见，但还是不太能正常生活，包括雾城在内的个位数城市任然仇视跨越者。

虽然法律写明了跨越者有人格尊严，但还是难以约束一整个城市的社区行为，甚至引起了部分洁癖派组织对国家的不满

### 边陲

| 条目 | 设定 |
|------|------|
| 地理 | 大部分不宜居地带和极少的宜居地带 |
| 居民 | 大部分已适应严酷环境，能勉强或正常生存 |
| 政治状态 | 部分聚落自立为国。地联未关注，也未出兵，无资源价值 |

## 力量社会

| 条目 | 设定 |
|------|------|
| 模式 | **公开**。社会知道武者和魔法师的存在 |
| 跨越者 | 近20年开始可以共存，接受度在部分城市缓慢上升。绝对数量比较少 |
| 残余阻力 | 洁癖派还是会拿碳基纯洁性说事，政治上还留有不少权力 |

## 魔力与科技

| 条目 | 设定 |
|------|------|
| 能量名称 | **洋能**：从亚空间渗进来的能量，本地人叫它魔力 |
| 科技基准 | 大致相当于21世纪 |
| 魔力参与度 | 不算高。它太吃个人体质，没法稳定量产 |
| 魔力储能 | 能量密度比锂电池高约20%，但得人工灌魔，产能一直卡着 |
| 魔力武器 | 有，但很少，基本算特种装备 |
