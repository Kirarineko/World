
# 本地泡基础设定

> 本泡内一切地理、政治、社会、历史设定的总索引。


## 行星与星系

| 条目 | 设定 |
|------|------|
| 星系 | 本泡有 **7 颗恒星**，散落在约 4–15 光年的尺度里。其中 1 颗是双星系统，其余 5 颗是单星（包括母星） |
| 恒星组成 | ① **母星**（太阳）：主行星绕行的恒星 ② **双星系统**（2 颗） ③ **其他单星**（4 颗） |
| 恒星间距 | 4–15 光年，构成一片松散的物理星团 |
| 主行星 | 地球（暂用名，可改） |
| 卫星 | 月球 |
| 泡壁位置 | 远在探测器可及范围之外，包裹整个星团。人类造物还没碰到过它 |
| 星空 | **双重光源**：① 真实恒星（7 颗中的 6 颗在夜空中肉眼可见）② 高斯泼溅光点（贴在泡壁内表面，不旋转、不移动，昼夜都在） |
| 星空可见性 | 真实恒星会受地区光污染影响，高斯泼溅光点白天也在，只是被阳光压住了 |
| 星空认知 | 人类一直以为看到的星星都是真实恒星。因为映射关系，高斯泼溅光点在视差、自行、光谱上都像真的，天文学上也分不出来。没人知道泡壁，更没人知道那些光点是怎么回事 |


## 宇宙认知

| 条目 | 设定 |
|------|------|
| 宇宙模型 | 仍然按传统大爆炸宇宙学理解。泡模型不在任何主流学术框架里 |
| 探测器极限 | 还飞不出太阳系，也没碰到过什么异常边界 |
| 亚空间 | 大多数人不知道它存在。只有极少数势力和学者知道一些零碎东西 |
| 自定位 | **完全不觉得自己活在泡里** |


## 智慧断崖与本地泡

整个泡宇宙里，智力在“猫”附近就会断一下。能把文明搭起来的智慧物种，少得离谱。本地泡偏偏是少数例外之一：海豚、鸦科、大象这类物种，在这里都比别处聪明得多。

> 详细机制见 `镜像机制.md`。这里先记住结论就行。

原因出在镜像。镜像要吃神经资源，得拿去维持引用计数、差分层、块释放这些东西。别的泡大多把资源砸进镜像里，活得久、缺陷少，但脑子也就那样。本地泡反过来，镜像做得更粗，省下来的资源拿去养认知，于是上限抬高了。

代价也很直接：寿命短一点，缺陷来得快一点，纠错也弱一点。人类在这里已经是很会算账的那一类了，但本质上还是拿完整性换了智力。

> 别的泡像长寿但反应慢的老机器。本地泡像不太耐用、但转得飞快的旧车。

| 条目 | 说明 |
|------|------|
| 智慧断崖基准 | 猫。智力在此附近存在全宇宙普遍断层 |
| 本地泡例外 | 海豚、鸦科、大象等智力远超断崖基准 |
| 根本原因 | 镜像机制阉割：弱纠错 → 释放神经资源 → 更高智力上限 |
| 代价 | 更短的寿命、更高的缺陷率、更弱的纠错能力 |

### 边陲

| 条目 | 设定 |
|------|------|
| 地理 | 不可居地带：沙漠、冰原、极端山地等 |
| 居民 | 大部分已适应严酷环境，能勉强或正常生存 |
| 政治状态 | 部分聚落自立为国。主导国家未予承认，也未出兵：无资源价值 |
| 详细信息 | 见 `边陲与聚落.md` |


## 力量社会

| 条目 | 设定 |
|------|------|
| 模式 | **公开**。社会知道武者和魔法师的存在 |
| 修炼门槛 | 大部分人连门都进不去，卡在魔力吸收的生理门槛上 |
| 社会感受 | 有点像职业体育：知道、尊重，但离自己很远 |
| 跨越者 | 近20年和平共存，接受度在往上走。绝对数量很少，普通人一生都未必能碰见 |
| 残余阻力 | 洁癖派还是会拿碳基纯洁性说事，政治和文化上都还留着一点力气 |


## 魔力与科技

| 条目 | 设定 |
|------|------|
| 能量名称 | **洋能**：从亚空间渗进来的能量，本地人叫它魔力 |
| 科技基准 | 大致相当于21世纪 |
| 魔力参与度 | 不算高。它太吃个人体质，没法稳定量产 |
| 魔力储能 | 能量密度比锂电池高约20%，但得人工灌魔，产能一直卡着 |
| 魔力武器 | 有，但很少，基本算特种装备 |
