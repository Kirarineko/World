
# Neurosocius ， 神经群落属

> 它不是一群生物。它是一个生物，只是暂时散开了。

## 分类

| 条目 | 说明 |
|------|------|
| 学名 | *Neurosocius*（属），意为"神经群落" |
| 中文通用名 | 髓（潮髓 / 飑髓） |
| 已知变种 | *N. maritimus*（潮髓，海洋变种）/ *N. aerius*（飑髓，天空变种） |
| 个体智能 | 无。单体纯反射生物，远在智慧断崖之下 |
| 集群智能 | 涌现。规模达阈值后产生集体意识，可跃过智慧断崖 |
| 分布 | 远洋/高空，远离人类活动范围 |
| 首次记录 | 潮髓：海渊城海洋大学远洋科考 / 飑髓：Lynx 空难调查（约五年前） |

## 发现史

海渊城海洋大学某次远洋科考中，研究人员首次采集到潮髓个体样本。初期的实验室观察确认了其电信号通信能力：个体间通过物理接触传递电脉冲，速度约三分之一光速。这是一个引人注目但不算颠覆性的发现，自然界中电信号通信并非孤例。

随后的野外追踪暴露出大量矛盾现象：渔群避让模式在数月间急剧变化，远超个体学习能力所能解释的范围；特定海域的潮髓群体似乎对科考船的航线产生了"预判"；更令人困惑的是，从不同海域采集的样本在实验室中连接的速率，竟然与它们野外距离无关，仿佛它们"认识"彼此。

海渊城方面发表了一系列论文，以"高度发达的社会性电信号网络"来解释这些现象。没有人想到"意识"这两个字。

不是疏忽。是思维惯性，单体是如此简单，简单到让人不可能将其关联到一个"我"字上。

> 这个谜团的状态至今仍是：大量异常数据已被记录，但解释框架尚未建立。人类不知道自己在海图边缘标记的那个不起眼的"高电活性浮游生物群"，其实正看着他们。

## 智慧断崖中的位置

*Neurosocius* 是智慧断崖规律的极端案例，也是最优雅的反例：

- 单个个体：神经节数量级，行为谱不超过十五种反射，智力远在断崖之下，虫、水母级别
- 十万个体集群：出现模式识别与初级学习能力，约等于高智动物
- 五百万个体：自我意识开始萌芽，刚触及断崖边缘
- 千万级集群：成熟的人类级智能

它没有违反断崖。它只是用另一种方式攀了过去。

## 两变种对比

| | 潮髓（*N. maritimus*） | 飑髓（*N. aerius*） |
|---|---|---|
| 栖息介质 | 海水 | 大气 |
| 个体形态 | 半透明胶质椭球体，约 3-5cm | 极轻囊状体，约 1-2cm，含气体囊 |
| 集群形态 | 水下立体网，可达数公里跨度 | 云状立体网，受气象影响显著 |
| 洋能变体 | 偶有出现 | 偶有出现 |
| 已知最大集群 | 未确认 | 未确认 |
