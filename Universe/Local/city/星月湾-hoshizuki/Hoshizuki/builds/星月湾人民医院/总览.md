# 星月湾人民医院

> 市级综合医院 · 北湾老城区 · 樱巷以东，潮蚀区以西

## 位置

医院坐落在北湾老城区东缘，正门朝西：从樱巷骑车过来五分钟，从潮蚀区走路十分钟。这是一条有意味的距离：樱巷的咖啡馆和旧书店是生活，潮蚀区的违章建筑和压低的争吵声也是生活，而医院恰好站在两者之间。

选址没有"恰好"：建院时周边全是普通居民区。后来北湾往西发展出了樱巷的文艺街区，往东残留了潮蚀区的破落角落，医院落在这条线上就成了一个沉默的见证者：它见过樱巷的游客中暑，也见过潮蚀区的"锻炼事故"。

## 规模

| 条目 | 数据 |
|------|------|
| 床位 | 约 900 张（含魔力相关科室 120 张） |
| 日门诊量 | 约 2500 人次 |
| 职工 | 约 1800 人 |
| 魔力相关科室 | 4（中医科 / 魔力损伤科 / 跨越者门诊 / 心智评估与沉渊科） |

比同人口城市的综合医院略大。不是因为星月湾人不健康：是因为周边几个中小城市的魔力伤病都会转诊过来。本地区只有这一家医院有魔力损伤科和跨越者门诊。

## 跨科室协作

| 组合 | 说明 |
|------|------|
| 急诊科 × 魔力损伤科 | 急诊接到魔力相关外伤，稳定后转魔力损伤科收治 |
| 急诊科 × 中医科 | 暗礁社成员的第二选择（"去中医科调理一下"比"去急诊看魔力反噬"体面 |
| 魔力损伤科 × 中医科 | 联合"魔力康复门诊"）西医管急性期，中医管康复期 |
| 心智评估科 × 中医科 | 沉渊者病区，西医诊断+中医安神方剂，中西医联合查房 |
| 跨越者门诊 × 检验科 | 常规检验对跨越者无效，检验科给跨越者门诊留了一台备用设备专做替代方案 |
| 检验科 × 研究园区 | 魔力增强剂中毒的血检异常，非正式通知 R4 |

## 对暗礁社的态度

医院没有"暗礁社收治流程"这种正式文件。但急诊科的护士长有一个抽屉，里面有几张手写的便签，不是病历，是经验：

- "自称健身房摔伤的，如果瞳孔有银光反应，先查血清魔力残留"
- "魔力增强剂中毒的补液速度比一般药物中毒慢一半，原因不明"
- "潮蚀区送来的患者陪护人不要多问，问多了人就不来了"

这不是保护暗礁社。这是保护患者不因恐惧而逃避就医。

> 星月湾人民医院真正的底色是：救人不问来路。在急诊科，一个洁癖派议员和一个跨越者理论上有平等的优先分诊权。实际上洁癖派议员不会来：他有私人医生。但原则本身是真的。
