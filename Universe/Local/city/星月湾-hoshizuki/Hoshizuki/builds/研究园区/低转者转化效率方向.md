# 低转者转化效率提升方向

> 园区方向三。R4 转化楼。三个方向中规模最小、伦理最敏感、与城市阴影面联系最深的课题。

## 背景

魔力存在于环境中，但大多数人（所谓的"无感者"）身体无法吸收。有感者中的多数又是"低转者"：能吸收，但转化效率极低。以十年为单位的修炼速度，魔力积累终生不足以支撑玄二。

园区想问的是：低转者的转化效率能不能被外部手段提升？

这不是学术好奇。这是现实需求。一个低转者如果能获得哪怕翻倍的转化效率（也仍然远不如正常转化者）人生的可能性就多了一种。

## 研究内容

### 一、代谢通路筛

| 条目 | 说明 |
|------|------|
| 目标 | 找出低转者和正常转化者之间在生物化学层面的差异。为什么张三维生素B12吃下去转化效率是李四的十几倍？问题出在吸收、转运、代谢的哪一步？ |
| 方法 | 志愿者提供血清和细胞样本，用高分辨质谱做代谢组学对比 |
| 进展 | 已发现低转者血清中几类特定的氨基酸衍生物浓度持续偏低。补充这些化合物是否能提升转化效率：尚未进入临床试验 |
| 合作 | 与 HBI 生科共享代谢通路数据库。生科对魔力在神经网络层面的研究为这个方向提供了一些分子靶点的假说 |

### 二、外部供魔辅助

| 条目 | 说明 |
|------|------|
| 概念 | 不只是"给低转者补充魔力"：如果体内魔力浓度本来就低，灌也灌不进去。方法是：在高魔力环境中，低转者的体内魔力是否会被动地达到更高平衡浓度？如果是，能不能在"高浓度浸泡"期间进行转化效率的训练，利用浓度差驱动更多的魔力进入细胞？ |
| 设施 | R4 有一间"高魔室"：密封房间，内部魔力浓度维持在正常环境的约 8 倍。志愿者在室内进行轻度魔力引导训练，监测体外魔力摄取速率的变化 |
| 结果 | 短期浸泡确实能提升魔力摄取速率，但效果在离开高魔室后 3 至 6 小时内消退。长期反复浸泡是否会产生持续效应，数据尚不足以得出结论 |

### 三、魔力增强剂（管控外）

园区不做药物开发：至少官方不做。但园区的研究人员知道外面有人在做。

暗礁社（城市缝隙里那群自主修炼的低转者）在服用未经测试的魔力增强药物。这些药物的成分、来源、制备方法，园区无从得知。但偶尔有暗礁社成员被送进医院时，急诊科医生会联系园区："你们懂魔力是怎么回事，能过来看一眼这个人的血检报告吗？"

园区没有直接介入暗礁社。但有几位研究人员私下对暗礁社使用的"野路子增强剂"非常感兴趣，不是支持，是恐惧。万一哪一天某个暗礁社成员突破的那扇门，不再是常规的从低转到正常，而是一次跃迁到一个谁都没见过的异常状态：园区理论上负有理解这件事的潜在责任，但目前毫无准备。

### 四、伦理约束

R4 的研究活动受严格的伦理框架限制：

| 约束 | 内容 |
|------|------|
| 志愿者 | 必须是年满 20 周岁、签署过知情同意书的成年人。不招募 HBI 在校学生（利益冲突） |
| 动物实验 | 不用。动物的魔力吸收机制与人类不同：这是园区和 HBI 魔力-神经接口课题组共同确认的基本事实：动物研究的数据不适用于人类转化效率 |
| 高魔室安全 | 志愿者在高魔室内佩戴实时生理监测。任何一项指标偏离基线超过预设阈值，实验立刻中止。中止按钮在志愿者手边 |
| 不做基因编辑 | 改造人类转化效率的基因编辑不在任何计划中。这不是技术问题：是园区所有人同意"先不谈这个"的默契 |

## 园区对暗礁社的立场

| 条目 | 说明 |
|------|------|
| 知情程度 | 知道暗礁社的存在，知道他们中有少数人因为"锻炼事故"被送医。但不掌握暗礁社的具体规模、人员构成或活动地点 |
| 官方态度 | 不干预。园区没有执法权，也没有调查义务 |
| 少数研究者的私下关注 | 个别研究员会追踪医院急诊记录的"魔力相关不明原因入院"趋势。目前趋势平稳，但他们知道这组数据理论上可能在一夜之间改变 |
| 若暗礁社有人突破 | 没有预案。R4 没有"异常魔力生物效应处置小组"这种东西。这是园区自己也知道的一个缺口，但没人有权限去填 |

## 速查

```
代谢通路 → 低转者氨基酸衍生物偏低，未入临床
外部供魔 → 高魔室浸泡，短期有效，长期待验证
魔力增强剂 → 园区不做药，但关注暗礁社的野路子
伦理底线 → 不用动物，不做基因编辑，高魔室有中止按钮
暗礁社关系 → 官方不干预，私下有人追踪急诊数据
预案缺失 → 若暗礁社有人突破异常状态，园区没有应对方案
```
