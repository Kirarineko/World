# 海洋科学学部

> 简称：海科。HBI 的历史根基：早在"泛生命科技"这个词出现之前，这里就有研究海湾的人。

## 学科定位

海洋科学学部不研究深海或全球洋流：那是国家级海洋研究所的事。海科的研究半径大约是从海湾开口处向外延伸 100 公里，向内覆盖整个湾区。尺度的"小"是主动选择的：与其追逐广阔，不如把一小片海域研究到极致。

这座海湾因为近乎封闭的地形和稳定的亚热带气候，提供了一个天然的可控研究场。海科学部的核心资产不是设备，是几十年的连续观测数据：水温、盐度、浮游生物种群、污染物浓度。数据本身不稀缺，但连续性稀缺。

## 核心方向

### 一、近岸生态系统

| 条目 | 说明 |
|------|------|
| 研究区域 | 海湾全境及近海大陆架 |
| 内容 | 浮游生物种群动力学、底栖群落结构、海湾与开阔海域的物质交换 |
| 独特性 | 圆形海湾的狭窄开口形成了一个"半封闭海洋盆地"：湾内的生态群落与湾外有显著差异，但又不是完全隔离。这种半连通状态在海洋学上不常见，为物种扩散和种群遗传学提供了天然对照实验 |
| 长期观测站 | 海湾内有三处固定的水质/生态监测浮标，数据每分钟回传。灯塔岛底部也有一个潮汐和波浪传感器，与灯塔共用电源 |

### 二、海洋生物资源

| 条目 | 说明 |
|------|------|
| 方向 | 不是渔业养殖。是海洋天然产物（海藻多糖、海绵生物碱、软体动物活性肽等）的分离提取与功能研究 |
| 应用 | 一种本地马尾藻提取的硫酸多糖被证实具有显著的重金属吸附能力，已作为生物修复材料进入生科的联合课题。另一种从海湾底泥中分离的放线菌菌株，其代谢产物对魔力场存在微弱但可重复的响应：目前机理不明 |
| 样本库 | 海科楼地下有一个冷冻样本库，存储着近二十年来从海湾采集的生物组织、海水和沉积物样本。库内温度常年维持在 -80°C。它不对外展示，但所有三系课题都可能从中调用实验材料 |

### 三、洋流与扩散建模

| 条目 | 说明 |
|------|------|
| 内容 | 海湾内水流循环的数值模拟。聚焦两个问题：其一，从海湾开口处进入的污染物（或浮游生物幼体）在湾内的扩散路径是怎样的？其二，海湾与近海的水体交换效率在不同季节和潮汐条件下如何变化？ |
| 与计科合作 | 模型依赖计科的高性能计算资源。一次高精度的三维海湾环流模拟大约吃掉 C4 三分之一节点半天 |
| 应用案例 | 曾在一次小型渔船燃油泄漏事件中，用模型预测油污扩散方向，为海警部署围油栏提供了24小时的提前量。预测偏离实际漂移中心约 200 米：对一次突发泄漏来说已足够好 |

### 四、环境 DNA (eDNA)

| 条目 | 说明 |
|------|------|
| 方法 | 从海水中提取游离的 DNA 片段，通过高通量测序识别其中包含哪些物种的遗传信息。不需要捕捞、不需要看到目标生物：从一桶水里就能知道这个海湾里住着谁 |
| 与计科的深度绑定 | 计科团队负责序列拼接和物种注释的算法管线。两者之间的日常是：海科的人把样本和 Excel 表格扔过来，计科的人把一堆基因序列 ID 扔回去，两边都觉得对方看不懂自己这半段工作的难度 |
| 发现 | eDNA 监测在 2017 年前后检测到一种不在本地记载中的裸鳃类（海蛞蝓）DNA。实地潜搜三个月后在灯塔岛礁石间找到实物：体长约 4 厘米，半透明蓝白色。目前尚未正式命名，实验室内部叫它"灯塔蓝"。它在分类学上的位置可能意味着它迁徙路线的终点比已知的任何裸鳃类都更靠北，但这只是假说 |

## 与其他学部的交点

| 交叉方向 | 合作学部 | 简述 |
|----------|----------|------|
| 海洋环境感知生物体 | 生科 | 将魔力/污染物感知菌株植入发光藻类，在海水中可视化环境异常 |
| 海洋生物基材 | 生科 | 海藻多糖 → 生物修复材料；贝类壳蛋白 → 建筑裂缝矿化 |
| eDNA 生信管线 | 计科 | 序列拼接、物种注释、种群遗传学分析 |
| 海湾环流数值模型 | 计科 | 三维 CFD 模拟，污染扩散预测 |
| 藻类光电薄膜的海水版 | 生科 | 用海洋微藻替代淡水藻，开发可在海面漂浮的光伏薄膜 |

## 教学特色

| 条目 | 说明 |
|------|------|
| 出海实践 | 必修。每学期至少两次。乘学校的科考船"海月号"（一艘改装过的 12 米渔船，可搭载 8 人）出海采样。大二的学生第一次在船上被晃吐是常事：指导老师会递一瓶水然后说"你吐你的，温度计递我一下" |
| 样本处理 | 海科学生对"泥沙"的感知和普通人不在一个层面。他们可以在一坨海底沉积物里用镊子挑出比芝麻还小的甲壳类幼体：然后告诉你这个种只在春季的第二周出现。这种技能在校园外完全没有用，但在海湾研究里是基本功 |
| 与渔港的关系 | 渔港的海鲜早市是海科的一个非正式采样点。教授和渔民之间有一种默契：渔民把混在渔获里的非经济物种（怪鱼、奇怪甲壳类、形态异常的贝类）留着，海科的人早晨去收。报酬通常是一条烟或一瓶酒，没有正式的科研经费走这条线 |

## 学部氛围

海科楼总是有股微咸的海水味：来自一楼的开放式水槽系统。大楼的通风井设计成潮间带水道的形态，海水在管道中缓缓流动，既是实验水循环系统的一部分，也给建筑降了温。如果你把耳朵贴在通风井管道上，能听到微弱的水声：海科学生管它叫"大楼的心跳"。

海科学生和其他两个学部存在温和的文化隔阂：他们晒得更黑，指甲缝里有洗不掉的沉积物，上课穿拖鞋，不是因为懒，是因为实验课经常踩水，穿运动鞋是浪费。计科的学生觉得他们"野"，生科的学生觉得他们"糙"，但所有人毕业前都得和他们上一组船。

> 海科楼外墙不是故意做旧的。是海风，是盐雾，是三十年没刷过外墙的预算缺口。但没人想刷：这栋楼的新生入学照和毕业照永远不一样，因为盐蚀的痕迹每年在变。
