# 生命科技学部

> 简称：生科。HBI 最年轻的学部，也是最具辨识度的标签。

## 学科理念

生命科技学部不研究"如何治疗疾病"：那是医学院的事。它研究的是**如何用活的系统替代或超越传统机械/电子系统**。学部的基本信念是：经过数十亿年演化的生物系统，在复杂性、能效和自修复能力上，比人类设计的任何机器都更优雅。与其从零造机器，不如理解并重新编织生命的原理。

学部对魔力的态度是功能性的：魔力是一种能量输入，生物系统对它存在响应，这种响应可以被研究、可以被利用。至于魔力是什么、从哪来：不在本学部学术范围内。

## 核心研究方向

### 一、湿件计算机

湿件（Wetware）指以活体细胞或生物大分子为基底的运算系统。这是生命科技学部的旗舰方向，也是让HBI在全国学术圈有了识别度的课题。

| 条目 | 说明 |
|------|------|
| 定义 | 利用神经元、心肌细胞或微生物群落的电化学信号进行信息处理 |
| 当前进度 | 已实现单层神经元网络的小规模逻辑运算。速度远不及硅基芯片，但功耗仅为后者的万分之一量级，且具备自适应重构能力：同一个培养皿在训练前后能执行不同任务 |
| 瓶颈 | 稳定性。湿件在连续运行 72 小时后开始出现不可预测的信号漂移。目前的最优方案是"周期性休眠"：让细胞休息几个小时，就像让生物睡觉 |
| 长远目标 | 不是替代硅基计算机，而是在特定场景（环境监测、长期低功耗边缘计算）中作为硅基的互补方案 |
| 校园可见度 | C1 实验楼一层有一间"湿件展示室"，陈列项目从早期的神经元网格到最新的多层培养阵列。玻璃培养箱上方挂着实时信号显示：一组缓慢波动的曲线。参观者通常盯着看十几秒，然后承认自己看不懂 |

### 二、蛋白质折叠电路

| 条目 | 说明 |
|------|------|
| 概念 | 利用特定蛋白质在环境中自发折叠成精确三维结构的特性，在纳米尺度构建"分子电路" |
| 与湿件的区别 | 湿件用活的细胞，蛋白质折叠电路用死的分子，但它继承了生物系统在分子层面的自我组装能力 |
| 当前进度 | 已实现简单的逻辑门（与/或/非）在试管内运行。单次运算消耗的能量约为同等硅基逻辑门的十亿分之一 |
| 应用前景 | 分子级传感器、体内可降解的微型计算装置 |
| 代表性实验 | "黑白反应"：一种在试管里用蛋白质逻辑门驱动的显色反应。输入两个分子信号，根据逻辑运算结果，溶液变色或不不变色。原理简单，但对大一新生有奇效：这几乎是HBI开放日最受欢迎的展示项目 |

### 三、菌丝体信号网络

| 条目 | 说明 |
|------|------|
| 概念 | 真菌菌丝体在土壤中天然形成庞大分支网络，单株真菌的菌丝总长可达数公里。菌丝体通过离子流、钙波和电化学梯度在网络上传递信息，其行为在某些方面类似神经网络的脉冲传导 |
| 研究内容 | 在受控介质中培养菌丝网络，用微电极阵列记录菌丝对外部刺激（温度、化学物质、魔力浓度、物理触碰）的响应模式 |
| 发现 | 菌丝体会"记住"反复出现的刺激模式：同一点被触碰多次后，菌丝对该方向的电信号传导速度明显加快。这是否构成"学习"尚无定论，但数据是真实的 |
| 潜在应用 | 活体土壤传感器、分布式环境监测网络。一个菌丝网络可以同时感知数平方米范围内的温度/湿度/魔力梯度变化，且不需要供电 |

### 四、藻类光电转换薄膜

| 条目 | 说明 |
|------|------|
| 概念 | 将光合藻类封装在半透光聚合物薄膜中，利用光合作用过程中电子传递链产生的微弱电流 |
| 效率 | 远低于硅太阳能板：约为商用光伏板的 3%–5%。但藻类薄膜的优势不在效率：它自修复（藻类活着就会持续生长），可生物降解，生产成本接近于零 |
| 校园应用 | 校园保留林地边缘有几盏试验性的藻类薄膜路灯。白天藻类光合蓄能，夜间通过埋设的微型储能电容点亮低功耗 LED。亮度约等于月光，但如果在深夜经过，你会注意到它确实亮着 |

### 五、魔力-神经接口

> 这是HBI最危险的课题方向。不因为魔力本身危险，因为没人知道魔力在神经层面到底做了什么。

| 条目 | 说明 |
|------|------|
| 研究问题 | 当一个人体内有魔力流动时，他的神经系统发生了什么？魔力为何可以被意念引导？ |
| 合作方 | 与计科共享 C2 实验楼的魔力测量设备 |
| 当前发现 | 魔力的流动模式与神经电信号之间存在耦合，但耦合关系不是严格线性的。同一组神经信号在不同魔力浓度的环境中，会诱发不同强度的魔力响应。换句话说，神经信号像是"拨动"魔力的那根手指，但手指拨出的力度取决于它周围有多少魔力可拨 |
| 伦理边界 | 由于本课题无法用动物实验替代（动物体内本就无魔力），所有涉及活体魔力响应的实验必须在体外培养的神经组织上进行。HBI 伦理委员会在批准第一批实验时花了 18 个月。目前该项目的研究规模很小：2 名教授 + 3 名研究生 |
| 理论猜想 | 如果有一天，有人能同时掌握神经信号编码和魔力响应的全部规律，理论上可以制造一个完全由生物神经网络驱动的魔力释放装置。这不是魔法：这是一个生物设备用正确的神经信号拨动了魔力。HBI 目前离这一步极远，但有人在想这件事 |

### 六、环境感知生物体

| 条目 | 说明 |
|------|------|
| 概念 | 通过基因改造简单生物（细菌、微藻、苔藓），使其在检测到特定环境因子时产生可读信号：变色、荧光、生长速度改变 |
| 已实现 | 一种工程菌株在魔力浓度超过阈值时产生绿色荧光蛋白。响应时间约30分钟，精度粗糙但肉眼可见。常用于实验室的魔力泄漏快速检测，本质上是"魔力验孕棒" |
| 与海科的交叉 | 海洋浮游生物版本正在研发。目标是将感知菌株植入发光藻类，让海湾水质的异常（包括魔力浓度异常）在肉眼可见层面现形。这个项目的难点不是基因工程，是让菌株在海水里不漂走 |

### 七、生物修复与韧性材料

| 条目 | 说明 |
|------|------|
| 概念 | 利用微生物或酶降解特定污染物；或利用细菌诱导的矿化过程修复建筑材料裂缝 |
| 已实现 | 一种石油烃降解菌的本地适应株：能在星月湾的海水温度下有效工作。曾用于一次小型渔船泄漏事故的后续处理 |
| 建筑材料方向 | 在混凝土裂缝中注入尿素分解菌：细菌代谢产生的碳酸钙会逐渐填充裂缝。修复速度以月为单位，但它确实在修复。丘麓步道一段老旧挡土墙曾在五年前接受此方法的现场测试：至今未见新的裂缝扩展 |

## 学部氛围

生命科技学部不是医学院。没有白大褂走廊，没有消毒水味道：实验楼闻起来更像潮湿的泥土、海藻、和培养基里的琼脂。学生在培养箱旁边吃盒饭（违规但没人管），把失败的菌丝样本做成杯垫当毕业礼物。

生科楼一楼走廊的"活体材料展示廊"是HBI唯一一个市民友好型空间：附近的家长带小孩来看发光藻和变色的试管，觉得"科学还挺酷"。教授们对此的态度是：只要不摸培养箱，不碰试剂架，随便看。

> 生科楼一楼女厕所有一面墙，学生自发在上面贴实验失败的便签："第47次，藻类全死了。原因：周四忘了调光周期。"下一行是别人的笔迹："换LED阵列三号位，那个灯泡偏暖。"这不是正式的知识管理，但可能比正式的好用。

## 速查

```
湿件计算机 → 神经元/M细胞做运算，功耗极低，瓶颈在稳定性
蛋白质折叠电路 → 分子级自组装逻辑门，纳米尺度
菌丝体信号网络 → 真菌菌丝的"神经网络"，活体分布式传感器
藻类光电薄膜 → 低效率但自修复的光伏方案，校园有试验路灯
魔力-神经接口 → 神经信号如何驱动魔力，伦理敏感，进展缓慢
环境感知生物体 → 基因改造生物对特定因子变色/发光
生物修复材料 → 细菌降解污染物或修复混凝土裂缝
```
