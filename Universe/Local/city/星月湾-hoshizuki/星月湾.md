# 星月湾

## 概述

星月湾是座人口约 172 万的沿海城市，躺在一座近乎圆形的海湾里。名字来自夜里那一眼看过去的景象：海面上的船灯、岛上的灯火、沿着月牙形平地铺开的城区，拼成一弯很安静的月色。"星"和"月"都在，于是就叫星月湾。

## 地理

### 海湾

形态近乎完美的圆形海湾，仅有一处开口连通外海

边缘有一圈低矮山丘，海拔不高但足以将城市与内陆隔开，形成天然的围合感 

山丘到海滩之间有大片大致呈月牙形的平坦地带，城市即建于此

东偏南，朝向开阔海域

海湾水域平静，时有台风暴雨，但少有不可预测的大风浪。 

海湾内散布少量小型岛屿

### 气候

| 条目 | 说明 |
| - | - |
| 类型 | 亚热带海洋性。冬无严寒，夏时有酷暑 |
| 特征 | 海雾常见于春秋清晨。夏季偶有短时雷雨 |
| 台风 | 海湾地形大部分时候可削弱风力，除非台风角度刚刚好在海湾内形成了气旋，百年间少形成严重灾害 |

## 城市格局

城市沿月牙形平地展开，从海湾最深处向外延伸。没有严格的功能分区，居住、商业、文化交错分布。

### 核心分区

| 区域 | 位置 | 特征 |
| - | - | - |
| **月心** | 海湾最深处，市区正中 | 真正的市中心。市政厅、图书馆和剧院都在这里 |
| **北湾** | 月牙北翼 | 老城区。原木风格建筑最密，街巷曲折，店铺也小 |
| **南湾** | 月牙南翼 | 新一些，住宅和小店混在一起，绿化很好。HBI 在南湾和丘麓交界那一带 |
| **丘麓** | 山丘脚下 | 城市边缘，密度最低，很多房子直接依山而建，站高一点就能看见整片海湾。研究园区也在这边 |
| **港区** | 月牙两端及部分沿岸 | 私人码头、游艇港、中小型货运港散着分布，没有重工业港 |

## 港湾

### 港口设施

| 设施 | 说明 |
| - | - |
| 私人小船码头 | 散布于湾内多处。停泊小型帆船、钓鱼艇为主 |
| 游艇港 | 南湾有一处专业游艇港。规模不大，但设施齐全 |
| 货运港 | 中小型。处理本地消费品和建材，非工业港 |
| 渔港 | 多处小渔港。每日清晨有渔获上岸，旁边就是海鲜早市 |


### 海上交通

| 类型 | 说明 |
| - | - |
| 游船 | 日间有环海湾观光船，夜间有"星月航线"|
| 货轮 | 普遍中小型，几十米级。每日可见七八艘进出 |
| 渔船 | 清晨出，午后归。从市中心月心广场就能望见归港的渔船桅杆 |

## 旅馆标语

星月湾的每一家旅馆，从北湾老城区的家庭旅馆到市中心的精品酒店，大堂或前台上方都挂着那一句话：

> **"若是繁星还没来，烟火也可映沧海。"**

## 人口与社会

### 人口

| 条目 | 数据 |
| - | - |
| 总人口 | 约 172 万 |
| 人口密度 | 中等 |
| 年龄结构 | 均衡。不是老年城市，也不是大学城式的年轻城市 |

### 社会氛围

总体基调开放包容

对魔力与魔法的接受度高。魔法师不是传闻，是街上偶尔会碰见的职业 

对跨越者的态度是只要不惹事，跨越者照样上街、坐公交、去咖啡馆。星月湾算是本地泡里最能让跨越者正常生活的地方

武者、魔法师、跨越者都只是“生活方式不同的人”，星月湾大体是这么看的


> 星月湾不是没有偏见。但在全国范围内，这里的人更倾向于"先看这个人做了什么，再决定怎么看他"。这种态度使星月湾自然成为了跨越者选择定居的城市之一。

### 洁癖派在星月湾

碳基纯洁性运动（"洁癖派"）在星月湾存在，但影响力微弱。他们在市中心偶尔组织小型集会，路人大多绕过，偶尔有人停下争论几句。本地媒体对他们的态度是"记录但不放大"。星月湾的洁癖派更像是一个被宽容存在的异见群体，而非政治力量。

星月湾不是童话。它有自己不想被游客看见的部分。

### 潮蚀区

北湾老城区最靠近海湾开口的一小段（本地人叫"潮蚀区"）是这座城市最破旧的角落。这里的建筑大半建于数十年前的风暴之后，用料廉价，地基受潮。墙面常年有水渍，梅雨季墙角长霉，海风裹着盐分把木窗框啃得斑驳。

潮蚀区住着渔港的临时工、无证小贩、以及那些"不太想被人找到"的人。白天安静得反常，入夜后巷子里偶尔有压低的争吵声和不知来源的撞击声。北湾其他区域的居民路过这里会不自觉地加快脚步。

### 废弃矿道

环绕海湾的山丘内部有多条废弃矿道，最早可追溯到星月湾建城之前。多数入口已被封堵，但仍有部分被重新"发现"并投入使用：用途不一。已知的包括：无证修炼者的秘密训练场、走私货物的中转仓、洁癖派极少数激进成员的集会点。也有传闻说某些矿道深处存在地下拳台，观众买票入场，胜负没有规则，打死不算新闻。

市政厅的官方说法是"所有矿道均已封闭，进入即有生命危险"。后半句是真的：矿道结构失修，地质早已不稳定。

### 暗礁社

魔力修行天然筛选出极少数"正常转化者"，但筛选的另一面是（大量"低转者"被挡在真正的修行门外）他们有感应，有渴望，但没有效率。这些人中的一部分聚集在城市的缝隙里，自称"暗礁社"。

暗礁社不是组织，更像是一种身份标签。他们交换粗劣的自主修炼方法，服用未经测试的魔力增强药物，偶尔利用废弃矿道进行"实战测试"。大部分人没什么威胁，只是在自己折腾自己。但每年总有几起事故（魔力反噬、药物中毒、走火入魔）被匆匆送往医院时，病历上总写着"锻炼事故"。

### 雾夜

星月湾的海雾在春秋两季是常态，但真正危险的是"浓雾夜"：能见度不足五米，海陆不分。本地渔民航海靠设备和经验，但雾夜里什么事都慢半拍，包括救援。

每年总有几艘小船在浓雾夜没有回来。有些是事故。有些据说是"越过了不该越过的界限"。

> 海警的记录显示近十年失踪船只数量在正常范围内。但那些航行者说的是另一回事。

### 岛屿上的人

海湾内的离岸礁和少量小型岛屿上并非完全无人。有些岛屿上住着独居者：懒得和任何人打交道的人，或是不能和人打交道的人。他们没有在市政厅登记，靠小船往返，有时候连着几个月不露面。偶尔有登山者在丘麓步道上用望远镜看到远处某座礁岛上有炊烟。
