# 茉璃 · 能力与习惯

---

## 能层能级：炽·散（N·4d）

### 散态特征

4d 范围型。洋能展开为多瓣扩散态，覆盖面广，控制精细。不适合单体爆发（那是贯态的领域），但适合精细操作、大面积感知和轻柔干预。

| 用法 | 说明 |
|------|------|
| 广域感知 | 洋能散开后感知范围内的生命体位置、形态、状态。范围取决于洋能投入量，日常轻松感知半径约数十米，主动扩展可达更远 |
| 精细操作 | 散态的洋能可以拆成极细的"手指"，在微观层面干预活体系统。这就是她能修湿件的原因，不是学来的技能，是自然感知的延伸 |
| 轻柔推动 | 推开鸟群、拨开水面的浮萍、把掉进海里的仪器轻轻托起：散态不是打出去的拳头，是铺开的薄纱 |
| 防护 | 散态可以形成大面积的浅层防御，不是硬壳，是减缓冲击的缓冲场。够用于日常意外，不够用于战斗 |

### 战斗定位

炽层是"人类路线的最高完整能层"，但茉璃不是人类：对她而言炽只是她当前的状态，不是修炼的终点。4d 在攻击性上不如同层的 4f（构态），但范围和控制有优势。她不喜欢战斗，不是不能打，是没必要。

## 洋能依赖

| 条目 | 说明 |
|------|------|
| 能量来源 | 亚空间洋能。本地泡渗透率有限，环境魔力太稀薄，无法支撑炽层消耗 |
| 补充方式 | 回亚空间。通道藏在北湾后山地洞：去撕开裂缝，回到亚空间吸一口饱的，再出来关闭裂缝 |
| 补充频率 | 日常低消耗时约一两周一次。连续大量使用（比如长途奔袭、大规模感知）后立刻需要 |
| 不进食 | 纯洋能生物不需要食物。但她会吃东西：不为营养，为味道。偏好鲜味重的、咸味淡的（味觉偏移详见 `../Foods/饮食.md`） |

---

## 通道

藏在北湾后山一个天然地洞里。通道是亚空间与本地泡之间的一扇"门"：不映射到现实空间，需要她手动撕开裂缝才能进出。

| 特征 | 说明 |
|------|------|
| 携带 | 曾经随身。但太累：通道是一个挂在她意识上的锚点，离本体越近越重。藏在地洞里轻松很多 |
| 安全性 | 只有她能打开。裂缝关闭后没有痕迹，没有洋能泄漏 |
| 退路 | 她不常用：最近几年一个月才回去一两次，纯粹是补充洋能。亚空间的领地还在，但她已经很久没认真打理过了 |

## 易容

| 条目 | 说明 |
|------|------|
| 用途 | 湿件维修等需要"不是猫娘"的场合。她不想让猫窝变成电脑城，不是怕人知道能力，是怕人上门打扰 |
| 方式 | 4d 散态洋能覆盖全身，微调身体轮廓和面部结构。不是幻术：是肉体层面的短期重塑。舒适时长约 1-2 小时，超过三小时会感到不适，需要变回原形休息 |
| 瑕疵 | 猫耳和猫尾可以缩短和变色，但不能完全消失：这是她身体的一部分，不是饰品。一般把猫耳压进头发，尾巴藏在宽大外套里或缠在腰间。近距离仔细观察可能暴露，所以她只接不需要面对面长谈的维修单 |

## 假身份

| 条目 | 说明 |
|------|------|
| 数量 | 两三个，交替使用 |
| 来源 | 不是伪造的：是这些年认识的熟人帮忙注册的。主要是一个HBI计科的学生帮她弄了一套基础身份链 |
| 用途 | 湿件维修接单、租借设备、在不想被当作"那个二次元猫娘跨越者"时使用 |
| 茉璃的真身 | 只在她当吉祥物、晒太阳、去咖啡馆、和熟人相处时出现 |

## 日常习惯

| 习惯 | 说明 |
|------|------|
| 句末带喵 | 不是卖萌。亚空间没有语言，她的第一套声音表达是猫叫。学人类语言后猫叫变成了语气助词（经常在句末不自觉地加上，"不知道喵""饿了喵"）自己有时意识到，有时不能 |
| 晒太阳 | 不管季节，只要有太阳且没有别的事，她会找一个有光的地方安静坐着。樱巷的石板路、丘麓步道的长椅、南栈桥尽头那张长椅：都是她的老位置 |
| 尾巴反应 | 尾巴是她情绪最诚实的地方。压下去是紧张，轻微摆动是想问题，僵直是警觉。她自己知道这点但控制不了 |
| 帮大学生捞东西 | 起源是某个HBI海科学生的仪器掉进潮间带，她在旁边晒太阳。现在海科学生之间有一个不成文的传统："仪器掉海里了，去南栈桥看看有没有一只猫娘在发呆。"：成功率比打捞队高，且不收钱 |
| 漫展吉祥物 | 每年本市漫展她都去。不是去逛：是去坐镇的。主办方会给她安排一个靠窗的座位和一杯不加糖的热可可。游客可以和"真猫娘"合影。她一般会配合，除非当天心情不好或者对方试图拽她尾巴 |
|  | 她不是极简主义者。衣柜里塞满了各种风格的衣服：和风、休闲、连衣裙、卫衣、冬季的大衣。出门前换衣服是她为数不多的仪式感。厨房有囤零食的习惯（偏鲜味：海苔、鱼干、虾片），冰箱里常备布丁。有手机也有电脑：手机主要用来刷漫展资讯和跟熟人聊天，电脑用来接湿件维修的单和偶尔打游戏。最私人的物品是一本翻旧了的星月湾植物图鉴：那是她学语言时用的第一本"课本" |
