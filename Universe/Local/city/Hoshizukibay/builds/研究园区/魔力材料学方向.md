# 魔力材料学方向

> 园区方向二。R3 材料楼。

## 定位

如果魔力储能方向的问题是"怎么把魔力存起来"，魔力材料学的问题是"魔力跟不同材料相处得怎么样"。这是一个基础研究占比较高的方向，离商品化最远，但也是三个方向中最可能产生意外的领域。

## 核心课题

### 一、铭文载体

铭文（Enchantment Substrate）是已知唯一能使魔力在脱离活体后仍保持特定结构的功能材料——本质上是对魔力的"定型"介质。

| 条目 | 说明 |
|------|------|
| 现有材料 | 一种复合陶瓷聚合物，在表面刻以特定纹路后，可将魔力约束在纹路路径上 |
| 瓶颈 | 高温稳定性差——超过约 80°C 后铭文开始失效。这意味着铭文不能上发动机、不能进工业高温环境 |
| 方向 | 正在测试两种替代方案：① 金属基铭文——耐高温但加工成本飙升，且有些金属本身就能屏蔽魔力（铝就是之一）；② 生物基铭文——用角蛋白/丝蛋白等生物大分子做载体，灵活且可能在活体组织上临时附着 |
| 应用 | 当前铭文主要用于介质船（亚空间中稳定介质密度的船体——但园区并不知道亚空间的存在，他们只知道这种铭文"似乎能在某些特殊流体场中保持结构"）、微量魔力工具的表面赋魔 |

> 园区对亚空间的存在毫不知情。他们研究铭文是因为"魔力能维持结构"这件事本身值得研究，实验结果偶尔与维度界面附近的物理现象发生共振——这种时候，数据会莫名超出误差范围。发生过的次数不多，但每个材料组的资深研究员至少亲眼见过一次"不太对的数据"。没有人能解释。报告上没有提。

### 二、魔力屏蔽材料

| 条目 | 说明 |
|------|------|
| 问题 | 魔力会渗透大多数天然和合成材料。高浓度魔力环境中，精密电子设备会受干扰——类似电磁干扰但不是电磁干扰。需要能阻隔魔力穿透的材料 |
| 已知 | 金属铝是最有效的魔力屏蔽材料之一。一层 2mm 铝箔就能显著削弱魔力渗透。原因不明——魔力不是电磁波，铝为什么管用，没人知道。但这不妨碍所有人用铝箔包实验设备 |
| 方向 | 正在寻找比铝更轻、更柔韧、更容易加工成型的屏蔽材料。候选包括：石墨烯层压膜（石墨烯层之间的电子云可能对魔力产生类似铝的效应）、生物源复合材料（某些海藻的细胞壁似乎有轻微的魔力阻挡特性——这是园区从 HBI 海科的样本库中偶然注意到的） |

### 三、传魔介质

| 条目 | 说明 |
|------|------|
| 概念 | 与屏蔽材料相反——寻找让魔力"流畅通过"的材料。用于魔力传输管道、储能单元内的能量迁移路径 |
| 现状 | 空气本身是够用的传魔介质，但衰减快、覆盖范围短。液体介质（如特制的硅油）传输效率更高但成本和维护不可接受 |
| 候选 | 一种掺杂了微量镧系元素的有机凝胶。能将魔力的有效传输距离从空气中不足两米延伸到约十五米。目前离实用化还很远——凝胶的保质期不到一周 |

## 与其他方向的关系

| 关联 | 说明 |
|------|------|
| 储能方向 | 铭文载体是储能单元内部结构的关键组件——如果没有铭文维持魔力在单元内的分布，魔力会在几小时内从高密度区扩散到低密度区，最终均匀衰减。铭文的品质直接决定储能单元的可用性 |
| 转化方向 | 低转者体内魔力浓度往往不足以触发任何可测量的生物效应。传魔介质的研究如果能反向应用于生物环境，可能提供"外部供魔辅助自身转化"的理论基础——不过这仅仅是假说 |

## 速查

```
铭文载体 → 陶瓷聚合物为主，耐热差，在测试金属基和生物基
魔力屏蔽 → 铝最有效但原因不明，石墨烯/海藻在研
传魔介质 → 镧系有机凝胶，2米→15米，保质期仅一周
隐藏事实 → 某些实验数据莫名异常，可能与维度界面共振有关（园区不知情）

```
