# 校园布局

## 选址

南湾与丘麓的交界地带。地块北侧缓缓抬升融入山丘，南侧是南湾平坦的住宅区，东侧可以远眺海湾开口方向。校园占据约 0.8 平方公里的缓坡地，其中约四分之一是原始植被保留区。

选址理由很简单：南湾是城市最新的开发区域，地价适中，有足够的空地容纳一个校园；丘麓的低密度住宅在校园建成后自然形成了一个环绕的"教职工居住带"。

## 边界

HBI 是半开放校园。边界由三样东西界定：

| 边界 | 形式 |
|------|------|
| 北侧（丘麓方向） | 一条宽约 4 米的浅水渠，水流引自山丘上的自然溪流。渠中常年有水，水浅可涉，但底部铺了大小不一的鹅卵石——物理上可跨越，心理上它划了一条线 |
| 南侧（城市方向） | 低矮的绿篱和花坛。没有围栏，没有门岗。这是主入口方向 |
| 东侧 | 与丘麓步道自然衔接，有一条校属滨水步道 |
| 西侧 | 隔一条双向两车道的小路与教职工住宅区相邻 |

水渠内侧属于校园，但渠上的几座矮桥是公共通道——市民和学生共享。穿过校园的几条主要步道在夜间 23 点后关闭路灯，但不设物理阻拦。

## 分区

校园大致分四个板块：

```
北（丘麓侧）
  实验楼群 + 保留林地
          ↑
西 ← ← 核心区（图书馆/广场/食堂） → → 东（生活区+滨水步道）
          ↓
       南湾住宅区方向
```

### 核心区

| 地点 | 说明 |
|------|------|
| **图书馆** | 六层，是现代建筑但大量使用当地石材和木材饰面。顶层是玻璃幕墙阅读室，可以看到海湾。馆藏以电子为主，纸质馆藏集中在 4–6 层。负一层有 24 小时自习室 |
| **星月广场** | 图书馆前方的小型下沉广场，半圆形阶梯看台，中央是一方浅水镜面池。池底嵌有光纤，入夜后亮起暖光，排列模仿海湾的轮廓——是一个微缩的星月湾地图。不下雨的日子里，学生坐在台阶上吃午饭是常态 |
| **食堂群** | 三个食堂分散在核心区边缘。第一食堂（最大）靠近教学区，第二食堂靠近实验室，第三食堂靠近生活区。彼此在菜品上存在温和的竞争关系 |
| **行政楼** | 五层，低调地缩在核心区角落。砖红色外墙，被一排香樟遮得几乎看不见。HBI 学生普遍认为这是学校故意的 |

### 教学区

| 地点 | 说明 |
|------|------|
| **海洋科学楼** | 五层，靠东侧。底层有大型水槽实验室，顶层有遥感天线阵列。建筑的通风井被设计成模仿潮间带水道的形态——海水在管道中循环，同时降温、教学、造景三用 |
| **计算机科学楼** | 六层，在核心区西侧。玻璃和灰色铝板外墙，看起来像"学院里唯一没什么特色的建筑"。但内部有一个小型数据中心的冷却管道穿过建筑中庭——管道外壁覆盖着从校园保留林地移植的苔藓，冬天苔藓干燥呈褐色，夏天恢复墨绿。学生叫它"绿肺" |
| **生命科技楼** | 四层，位于教学区北侧、紧邻实验楼群。生物安全等级最高达 BSL-2。一楼全层是公开的"活体材料展示廊"——沿走廊排列的培养箱里展示菌丝体生长、藻类光电反应等。对外开放，常有市民带小孩来"看会发光的绿藻" |
| **综合教学楼** | 两栋错落的四层建筑。通用教室分布于此，三个学部的通识课、基础课都在这里上。走廊墙面上挂着历年学生课题的海报，层层叠叠，最底层那张可能是十几年前的 |

### 实验楼群

教学区以北、靠近水渠的缓坡上一字排开五栋三至四层的实验建筑。编号 C1–C5，按生物安全级别和学科分配使用：

| 编号 | 主要用途 | 特点 |
|------|----------|------|
| C1 | 生科·湿件计算与合成生物学 | BSL-2。有独立通风系统 |
| C2 | 生科·魔力-神经接口 | 与计科共用。屏蔽室内放置魔力测量设备 |
| C3 | 海科·海洋生物与水质分析 | 与校园水槽系统直连 |
| C4 | 计科·高性能计算 | 隔音、恒温、独立的供电冗余 |
| C5 | 三系共用·交叉课题 | 空间可灵活重组的开放式实验室。设备和分区随当前在研课题变化 |

实验楼群之间由玻璃连廊连接，连廊外侧的支架上爬满藤本月季——四月开到十月。选择藤本月季的原因有两个：其一，它在星月湾的气候下几乎不用管；其二，连廊里的学生可以透过花叶间隙看到水渠对面的丘麓步道。

### 生活区

| 地点 | 说明 |
|------|------|
| **学生宿舍** | 东侧。四至六层的小型公寓楼，围绕五个小型庭院分布。每个庭院种一棵不同的树：樱花、银杏、香樟、桂花、紫荆。学生给宿舍的称呼方式是"那棵树下那栋" |
| **滨水步道** | 宿舍区东侧边缘，与丘麓步道平行。贴校园内侧有一条铺木屑的跑步径，每隔一段有木椅面朝水渠和远处的山丘 |
| **学生街** | 也称"南栈后街"。不严格在校园内——是紧贴南侧绿篱外的一条短街，但学生比市民多。长度仅约 200 米，分布着：打印店（兼卖旧教材）、两家便利店、一间文具/耗材店、三间咖啡馆、一间深夜营业的盖饭店、一间甜品店。街上没有酒吧——最近的酒吧在南湾住宅区深处 |

### 保留林地

校园北端、水渠以北是一块约 0.2 平方公里的原始植被保留区——实际上是建校时规划留给未来的预留地，但这"未来"从未到来。林内没有铺装路面，只有几条被人踩出来的土径。生科的学生偶尔在这里采集土壤菌种样本，其他学生主要在这里：散步、发呆、偷偷吸烟。
