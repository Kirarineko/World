# 魔力损伤科

> 三楼西翼。全院最安静的科室，不是因为病人少，是因为病人都在疼，没力气说话。

## 定位

魔力损伤科处理的是魔力对活体组织造成的直接物理或功能性损伤。这是全市唯一的魔力损伤专科：周边几个中小城市转诊过来的魔力伤病全部收在这里。

科里没有"常见病"这个概念。魔力的作用方式取决于施用者的手法、受害者的体质、接触时长、环境魔力浓度：变量太多，两个表面症状相似的病例可能是完全不同的损伤机制。住院医生入职第一年最大的感受是：教科书上的标准病因分类，在真实魔力损伤面前大概只覆盖了六成。

## 核心病种

### 一、魔力回路撕裂

| 条目 | 说明 |
|------|------|
| 病因 | 魔力在体内过量聚集或失控流动，超过经脉/血管/神经周围组织的承载上限，发生微观层面的组织撕裂 |
| 好发人群 | 玄境及以上武者（归境尤为高发）、魔法师突破能级时失控、低转者用劣质魔力增强剂强行提升 |
| 临床表现 | 撕裂部位剧痛（常被描述为"体内有刀片在刮"），局部皮肤出现银白色线状纹路（魔力沿撕裂路径逸出，与表皮角蛋白反应变色），受累肢体运动障碍 |
| 诊断 | 魔力造影：在患者体内注入微量无害的魔力示踪剂，观察示踪剂在组织中的扩散路径。正常人的示踪剂沿经脉自然扩散，回路撕裂者会在裂口处"漏"出示踪剂，形成喷溅状影像 |
| 治疗 | **轻中度**：制动 + 魔力禁绝（停用一切魔法/武力修炼）+ 组织再生药物。愈合周期约 2–4 周。**重度**：手术修复。但魔力损伤科的外科医生必须自身具备至少玄二以上的魔力修为（缝合魔力回路需要术者能感知魔力在组织中的精确位置，这是手术刀看不见的东西 |
| 后遗症 | 撕裂处愈合后的瘢痕组织是魔力的天然屏障）魔力永远不会再流过那个位置。武者如果关键经脉点被瘢痕覆盖，修为可能永久退阶 |

### 二、灌魔过量

| 条目 | 说明 |
|------|------|
| 病因 | 外部魔力注入超过个体承受上限。最常见于（魔力储能设备事故（灌魔操作者被反向冲击）、不当使用魔力增强药物、修炼者贪进 |
| 临床表现 | 全身性魔力水肿）魔力渗入组织液，造成全身肿胀但不伴炎症反应。患者皮肤触之温热（魔力在皮下降解产热），严重者意识障碍 |
| 治疗 | 脱离魔力环境 + 利尿剂加速魔力排泄 + 必要时透析。魔力分子量小且不与血浆蛋白结合，透析效率较高。严重病例联合中医科的泄火排毒方剂，效果优于单用西医 |
| 预防 | R2 储能楼的灌魔操作室有实时魔力浓度报警：这个安全标准不是国家标准，是园区自己加装的。原因就是灌魔过量的头两例事故都发生在园区实验室 |

### 三、修炼反噬

| 条目 | 说明 |
|------|------|
| 病因 | 广义概念，涵盖所有修炼过程中产生的自我损伤。具体次型包括（能级突破时意识失控导致魔力暴走、吸入被污染的魔力（某些环境中的魔力含未知杂质）、归渊过程中魔力回流冲撞 |
| 临床表现 | 高度异质）从单纯性头痛到全身魔力系统崩溃不等 |
| 治疗 | 无统一方案。核心原则：停修、监控、支持治疗。部分病例自愈，部分留永久性魔力回路损伤 |

## 职业病谱

金玄归天四境武者，各自有自己的"职业病"：

| 境界 | 典型损伤 | 原因 |
|------|----------|------|
| 金境 | 肌肉/骨骼劳损 | 纯肉身极限训练。本质上和运动员伤病一样，只是程度更重 |
| 玄境 | 魔力回路不完全撕裂 | 魔力觉醒期的适应过程。玄三至玄二是发生率最高的阶段：身体第一次被魔力撑开经脉，很多人这一步就没走稳 |
| 归境 | 归渊后遗症 / 魔力融合异常 | 归渊是武者的心理临界，也是生理临界。跨过去的人有一部分身体对魔力的免疫识别紊乱（开始排斥自己体内的魔力，表现类似自身免疫病 |
| 天境 | **几乎不在本院就诊** | 天一武者远比跨越者稀少）纯人类使用洋能的效率极低，修炼到天境比跨越N→O难得多。天一受伤通常直接去一线城市顶级医院。本院建院以来仅收治过两例天一患者，一例是路过星月湾时旧伤复发，一例是本市出生但拒绝去外地就医。两次都让全科如临大敌：天一的碳基身体已被洋能深度改写，常规药物剂量全部失效，麻醉剂无法通过改造后的血脑屏障，手术中可能出现无意识的洋能脉冲反击。术后，两例病历被装订成册放在主任办公室，封面手写"天境"二字，没有编号，因为至今没有第三例 |

## 魔力造影室

科里的核心设备。一间经过铅+铝双层屏蔽的房间，中央是一张环形传感器平台。患者在平台上方接受微弱魔力场扫描。铅层屏蔽外部电磁干扰，铝层，因为没人知道的原因：能显著降低包络外魔力噪声。

> 铝为什么屏蔽魔力？魔力损伤科的主任在学术会议上被问过这个问题。他的回答是："我不知道，但如果你有更好的选择，请务必告诉我。"

造影的图像不显示器官或骨骼，显示的是魔力在人体内的分布。正常的魔力影像是沿着经脉的柔和光带。异常的影像（撕裂处的喷溅、水肿区的弥散晕、瘢痕处的暗区）对诊断的参考价值远超任何血液指标。

## 与中医科的康复联合门诊

魔力损伤科治疗急性损伤，中医科管康复。一个武者或法师的典型就诊路径：

```
受伤 → 急诊科 → 魔力损伤科收治（急性期，1-2周）→ 出院前转中医科评估 → 康复联合门诊（针灸/方剂/导引，2-8周）→ 复功
```

住院期间中医科医生会来魔力损伤科查房，主要看脉象：在魔力损伤患者身上，寸口脉的变化与魔力回路状态存在粗略但可重复的对应关系。魔力损伤科年轻医生最初对脉诊持怀疑态度，被中医科老主任一句话堵住了（"你可以不信，但你可以看看造影图再摸摸脉）它们不一样的时候你再来找我。"

目前没人找过他。
