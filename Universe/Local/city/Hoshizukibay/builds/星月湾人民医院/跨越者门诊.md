# 跨越者门诊

> 五楼东南角。门牌上写着"跨碳基代谢专科门诊"——这个名字是故意起的。跨越者知道这是什么科，洁癖派看了不知道。

## 定位

跨越者门诊是全市——极可能是全地区——唯一为跨越者提供医疗服务的专科门诊。"跨越者"指完成了能层 N→O 跨越的个体：不再是碳基生命，不再被认为是人类，但仍然是需要看病的人。

门诊每周开放三天（周一、周三、周五上午），其余时间接诊普通内分泌和代谢病患者——这是维持科室运营的妥协。跨越者的绝对数量极少，单独支撑不起一个全日制门诊。

## 为什么跨越者需要专门门诊

常规医疗对跨越者的适用性：

| 常规检查 | 对跨越者是否有效 | 原因 |
|----------|-----------------|------|
| 血常规 | 否 | 跨越者的循环液体不是血液。严格来说它是一种含能量的半透明流体——成分接近血浆但魔力饱和度高到能让血细胞分析仪报错 |
| 生化全套 | 否 | 代谢途径变了。血糖、血脂、肝肾功能这些概念对碳基生命有意义，对非碳基生命来说不存在对应的生物化学过程 |
| CT / X 光 | 部分有效 | 跨越者体内仍有物理结构，但密度分布与碳基生命完全不同。放射科医生看到跨越者的 CT 影像第一反应通常是"这人是金属做的吗？"——不是，是高密度魔力积累区域在影像上和金属钙化影没有区别 |
| MRI | 禁用 | 跨越者体内的高强度魔力场与 MRI 的强磁场存在不可预测的交互。仅发生过一次事故就足以让所有跨越者禁用 MRI——那次事故的后果是跨越者从磁场中被弹出去，连带撞翻了在场的两个技师 |
| 药物 | 几乎无效 | 跨越者的体内环境使大多数药物在吸收前就被降解或转化。剂量也无法计算——没有肝肾功能就没有半衰期 |

> 一个跨越者感冒了怎么办——这个问题没有答案。不是感冒治不了，是跨越者不感冒。非碳基生命不感染人类病原体。他们有自己的"病"，但这些病和《内科学》第八版没有任何关系。

## 诊断手段

跨越者门诊不能用常规检验，只能靠间接手段：

| 手段 | 说明 |
|------|------|
| 魔力场谱分析 | 科室最核心的设备。用低频魔力场扫描患者全身，生成魔力场谱——类似人类的血检，但不是测化学指标，是测能量分布和振荡模式。一个健康的跨越者有一套稳定波谱，"生病"时波谱出现特征性偏移 |
| 自体报告 | 跨越者最可靠的症状来源是他们自己。科室有详细的症状问卷——不是选择题，是自由描述。跨越者对自身状态的主观感知通常比任何仪器都准 |
| 功能测试 | 看患者"能做什么"而非"指标是多少"。能量输出是否稳定？形态维持是否正常？维度感知是否清晰？——这些是跨越者健康的核心维度 |
| 同侪参照 | 不同的跨越者之间因跨越路径不同，身体状态无标准"正常值"。最有效的方法是让患者与自身基线做对比——每次就诊留一次场谱，建立个人健康档案，偏离基线即为异常 |

## 常见"病种"

跨越者不生病，但他们有自己的异常状态：

| 状态 | 描述 |
|------|------|
| 能量代谢失衡 | 跨越者的能量摄入（吸收环境中魔力）与消耗不平衡。表现类似人类的疲劳综合征——但原因不是缺觉，是魔力补充跟不上消耗。处理方式：建议去魔力浓度更高的区域待几天 |
| 形态失稳 | 跨越者的身体形态开始出现不受控的波动——手指透明度变化、轮廓边缘模糊、局部区域短暂"闪烁"。轻微且偶尔发生属正常，频繁发生时通常意味着心理压力过大或近期跨越了某种状态边界 |
| 维度感知错乱 | 跨越者可能感知到普通人感知不到的维度——这不是病，是虚层以上的被动能力。但当感知与实际发生冲突时（比如"看到"一个门在它不存在的地方），就是问题了。就诊时常被描述为"最近总觉得身边还有一个人，但我转身看什么都没有"。无法用药，靠认知行为调整和与同类交谈来矫正 |
| 魔力沉积结节 | 魔力在体内局部过度积累形成物理性结节。本质上是"魔力结石"——跨越者的废物代谢产物。通常无自觉症状，在魔力场谱扫描中偶然发现。极少需要手术 |

## 跨越者社区与门诊的默契

| 条目 | 说明 |
|------|------|
| 就诊登记 | 跨越者在医院不登记为"跨越者"——系统里用"特殊代谢体质"作为编码。这不是歧视，是保护。任何涉及"跨越者"三个字的正式记录都走纸质，不入电子系统 |
| 医生 | 门诊两名主治医生自己就是跨越者。这在全国范围内恐怕是独一份——两个非碳基生命体穿着白大褂在诊室里讨论另一个非碳基生命体的魔力场谱偏移。这份工作的招聘启事没有对媒体公开，候选人靠跨越者社区的内部推荐 |
| 社区服务 | 门诊不仅是医疗机构，也是跨越者社区的非正式联络点。新跨越者（刚跨越 N→O 不久）会在社区中被引导来这里做一个"基准体检"——建立基线场谱，了解自己的新身体，以及被告知"别去拍 X 光，别做 MRI" |
| 与洁癖派的关系 | 洁癖派曾试图在市议会削减门诊经费，理由是该科室"占用了为人类服务的医疗资源"。失败了，因为跨越者门诊的经费大部分来自跨越者社区的专项捐赠，不是政府拨款 |

## 与检验科的协作

检验科给跨越者门诊留了一台备用设备——一台改造过的魔力浓度测定仪，调低灵敏度后可以粗测跨越者的"体循环状态"。这不是标准诊断，是急诊替补。遇到跨越者急性发病又来不及做全面场谱时，先扫一遍看魔力分布有没有明显异常。

> 这台机器的屏幕边缘贴着用胶带固定的便签，检验科主任的笔迹："仅限跨越者。人类勿用。用了数值也不对，回来找我我也不解释。"
