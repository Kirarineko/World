# 跨越者门诊

> 五楼东南角。门牌上写着"跨碳基代谢专科门诊"——这个名字是故意起的。跨越者知道这是什么科，洁癖派看了不知道。

## 定位

跨越者门诊是全市——极可能是全地区——唯一为跨越者提供医疗服务的专科门诊。

"跨越者"指完成了能层 N→O 跨越的个体：碳基仍是其生命基础，但不再纯粹。他们的代谢、免疫、器官功能仍沿续碳基逻辑，同时被高强度洋能深度重塑——常规医学手段对跨越者而言时灵时不灵，就像在一个被反复改写过的硬

盘上运行旧版操作系统。

真正的"非碳基生命"——完全由洋能构成的存在——是更高能级的跨越者（5d曲态、6p坍态、7s寂·浑以上）。他们通常不会出现在门诊候诊室。他们出现在门诊的几率，大约等同于一座城市需要为一阵风设立挂号窗口。

门诊每周开放三天（周一、周三、周五上午），其余时间接诊普通内分泌和代谢病患者。跨越者的绝对数量极少，单独支撑不起一个全日制门诊。

> 一个更稀少的事实：天一武者比跨越者还少。纯人类使用洋能的效率极低，修炼难度比跨越者高得多。天一武者通常是去一线城市顶级医院就医的——星月湾人民医院没有天一病区，偶尔收治一个就是全院的学术讨论病例。

## 为什么跨越者需要专门门诊

常规医疗对跨越者的适用性——取决于跨越者处于碳基-洋能谱系的哪个位置：

| 常规检查 | 对低阶跨越者（O·虚~P·极） | 对高阶跨越者（5d曲/6p坍以上） |
|----------|--------------------------|-------------------------------|
| 血常规 | 部分有效。循环系统仍以碳基血液为基础，但洋能饱和度升高导致血细胞形态漂移，分析仪报参考值范围偏离警告 | 无效。循环介质已不再是血液 |
| 生化全套 | 部分有效。碳基代谢仍在运作，但重要指标的参考范围已显著偏移。血糖和电解质勉强可用，肝肾功能标志物不可靠 | 无效 |
| CT/X光 | 部分有效。碳基部分正常显影，高密度洋能积聚区在影像上呈现异常亮斑，容易误判为钙化或金属异物 | 几乎无效 |
| MRI | 禁用。跨越者体内的洋能场与MRI磁场存在不可预测的交互。有过一次事故，后果是跨越者连带两个技师被弹飞 | 禁用且不可想象 |
| 药物 | 部分有效但剂量不可预测。碳基代谢仍能处理部分药物，但洋能会加速或阻滞特定代谢路径，效应因人而异 | 无效 |

> 对低阶跨越者而言，最大的临床困境不是"这个检查用不了"，而是"这个检查显示的值我不知道它代表什么"。血常规报告上那个偏离的数值，是正常的跨越者生理变异，还是病理信号——没有人做过足够大的样本量来回答。星月湾的跨越者总数可能不到两位数，做不了统计。

## 诊断手段

跨越者门诊不能完全依赖常规检验，需要混合策略：

| 手段 | 说明 |
|------|------|
| 洋能场谱分析 | 科室最核心的设备。用低频洋能场扫描患者全身，测量洋能分布和振荡模式。一个健康的跨越者有一套稳定波谱，异常时出现特征性偏移。对低阶跨越者，场谱分析与碳基检查并行使用——两者交叉比对是当前最可靠的诊断策略 |
| 自体报告 | 跨越者对自身状态的主观感知通常比仪器更敏感。问卷以自由描述为主——"最近身体哪里不对劲"比标准化量表更有效 |
| 功能测试 | 能量输出是否稳定？碳基-洋能平衡是否倾斜？形态维持是否正常？——这些是跨越者健康的核心维度 |
| 个人基线对照 | 不同跨越者之间因跨越路径不同，无标准"正常值"。每个跨越者在首次就诊时建立个人场谱基线，此后每次就诊与自身基线做偏离分析 |

> 低阶跨越者（O·虚、P·极层，如5d曲态以下）仍保留大量碳基生理功能。他们的医疗可以在跨越者门诊和常规科室之间交叉进行——心内科可以看他们的心电图（虽然T波形态有点怪），骨科可以处理他们的骨折（虽然愈合速度是常人的两到三倍）。高阶跨越者（5d及以上的曲态、坍态、寂·浑）在任何意义上都不再是碳基医学的适用对象。但他们极稀少，极少出现在门诊中。

## 常见"病种"

| 状态 | 多发群体 | 描述 |
|------|----------|------|
| 碳基-洋能代谢失衡 | 所有跨越者 | 洋能摄入与碳基营养摄入比例失调。偏洋能则碳基组织萎缩（体重下降、肌力减退），偏碳基则洋能场不稳定。处理方式：调整生活方式——不是吃药 |
| 形态失稳 | 低阶跨越者为主 | 身体局部出现不受控的形态波动——手指透明度变化、皮肤纹理短暂扭曲。轻微属正常，频繁发生通常提示洋能场振荡或心理压力 |
| 维度感知错乱 | O·虚层以上 | 感知到普通人感知不到的维度信息，当感知与物理现实冲突时就诊。无法用药，靠认知行为调整和与同类交谈矫正 |
| 洋能沉积结节 | 低阶跨越者较多 | 洋能在碳基组织间隙局部过度积累形成物理性结节。碳基代谢无法完全清理洋能废物。通常无症状，场谱体检中发现。极少需手术 |

> 跨越者不感冒——不是不会生病，是人类病原体对他们的碳基组织附着力已大幅降低。但他们有自己的脆弱性：一个洋能场紊乱的跨越者，其碳基部分会像被忽视的花园一样缓慢衰败。

## 跨越者认证

在本地泡，跨越者能否获得武者认证因地而异——这个世界没有统一的"跨碳基武力资质法"，各城市自行决定。星月湾属于对跨越者开放的少数地区之一，跨越者可以在本市参加武者等级考试。不过实际上来考的跨越者极少——不是考不过，是不屑于。一个O·虚层的跨越者不需要一张执照来证明自己能打。

## 跨越者社区与门诊的默契

| 条目 | 说明 |
|------|------|
| 就诊登记 | 跨越者在医院不登记为"跨越者"——系统里用"特殊代谢体质"作为编码。涉及"跨越者"的正式记录走纸质，不入电子系统。这是保护 |
| 医生 | 门诊两名主治医师自己是低阶跨越者（O·虚层）。星月湾可能是全国唯一拥有跨越者主治医师的城市——他们能理解患者的洋能场谱偏移是因为自己也在同一张谱上。招聘走跨越者社区内部推荐，未对媒体公开 |
| 社区服务 | 门诊是跨越者社区的非正式联络点。新跨越者（刚跨过N→O不久）会被引导来做基准体检，建立场谱基线，被告知"别做MRI、别拍增强CT、告诉牙医你是特殊代谢体质——拔牙打麻药的剂量是别人的三分之一" |
| 与洁癖派的关系 | 洁癖派曾试图在市议会削减门诊经费，理由是"占用为人类服务的医疗资源"。失败了——门诊经费大部分来自跨越者社区的专项捐赠 |

## 与检验科的协作

检验科给跨越者门诊留了一台改造过的洋能浓度测定仪，调低灵敏度后可粗测跨越者的体循环洋能分布。急诊替补用——遇到跨越者急性发病又来不及做全面场谱时，先扫一遍。

> 这台机器的屏幕边缘贴着用胶带固定的便签，检验科主任的笔迹："仅限跨越者。人类勿用。用了数值也不对，回来找我我也不解释。"
