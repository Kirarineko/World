# 急诊科与暗礁社

> 一楼正中。全院唯一昼夜不关灯、不走安静路线的科室。

## 定位

急诊科本身只是一个常规城市医院的急诊科：心肌梗死、车祸外伤、小儿高烧、酒精中毒。但它同时也是星月湾最沉默的信息节点：暗礁社的成员受伤后，不去诊所，不去社区卫生站，穿过潮蚀区的巷子，走十分钟路，来急诊科。因为这里不问。

## 暗礁社的"锻炼事故"

| 条目 | 说明 |
|------|------|
| 暗礁社是什么 | 低转者：有感应的魔力，有渴望，但没有转化效率。被挡在真正魔法修行门外的人。他们中的一部分聚集在废弃矿道里，交换粗劣的自主修炼方法，服用未经测试的魔力增强药物，做一些"实战测试" |
| 他们对急诊科意味着什么 | 每年总有几起事故。魔力反噬、药物中毒、走火入魔（被匆匆送来时，陪同者常说）"他健身的时候受伤了" |
| 急诊科知道什么 | 知道这不是健身事故。知道瞳孔银光反应、血清魔力残留浓度、皮下银白色线状纹路：这些特征不属于任何健身运动损伤 |
| 急诊科不做什么 | 不追问。不是因为同意暗礁社的行为：是因为如果追问，下次他们就不来了。一个不来急诊的魔力反噬患者可能死在潮蚀区某个废弃矿道里 |

## 不成文的默契

急诊科内部有一套靠经验累积的口传知识，没人写进任何手册：

| 便签内容 | 来源 |
|----------|------|
| "自称健身房摔伤的，如果瞳孔有银光反应，先查血清魔力残留。不要只查一次：魔力增强剂在体内的代谢曲线不是线性的，前三小时残留量可能正常，四到六小时出现第二次峰值" | 护士长的手写便签，锁在分诊台左边第二个抽屉 |
| "魔力增强剂中毒的补液速度比一般药物中毒慢一半。原因不明。快了的都心率失常了" | 不知是谁写的，字迹潦草，像是夜班凌晨记的 |
| "潮蚀区送来的患者，陪护人不要多问。问多了下次人就往南湾那边送了。南湾没有急诊，只有社区卫生站，到那边人可能就没了" | 护士长的便签，被另一个人的笔迹加了一行："不是夸张。去年真没了一个。南湾卫生站值班医生打的120，送到我们这儿已经晚了" |
| "如果患者主动问'我是不是魔力中毒'，告诉他是。不要回避。回避了他心里有数但不敢再问。直接告诉他了，他反而更可能来复诊" | 急诊科主治 |

这些便签的纸已经卷边，不是被翻烂的：是有人经常拿出来往里面添一行。没有署名，没有日期。

## 送诊模式

暗礁社成员的送诊有自己不成文的流程（1. 伤者被同伴送来（"他健身受伤"））夜间或凌晨最多，因为暗礁社的活动也多在夜间
2. 同伴在分诊台签一个假名，留下一个不存在的电话号码，然后消失
3. 急诊科护士按"魔力相关急性损伤"模式接诊（包括但不限于）血清魔力残留抽血、心电监护、预备魔力造影（如果需要转魔力损伤科）
4. 伤情稳定后，根据损伤类型决定去向（魔力外科属性（回路撕裂）→ 转魔力损伤科；内科属性（中毒、反噬）→ 急诊留观或收治；如果患者本人或陪同提到"想调理一下"）这句话的潜台词是"我知道自己怎么回事，我不想去魔力损伤科，因为去了会被问修炼资质，而我拿不出来"（急诊护士会给中医科打电话）能不能过来看一眼？中医科一般会说能

## 与 R4 的非正式通道

暗礁社成员服用的是未经测试的魔力增强药物。这些药物的成分、来源、制备方法，园区无从得知。但当某个服药者被送进急诊，血清里出现了园区从未见过的代谢产物：这就是一条非正式的数据源。

| 路径 | 内容 |
|------|------|
| 触发 | 急诊科检验报告出现无法解释的魔力残留代谢物 |
| 通知 | 急诊科主治私下联系 R4 转化楼某位研究员，不是体制内的联络人，是某次学术会议上交换过手机号的私交。"你们上次说看到奇怪的血清报告想看一眼？这儿有一个" |
| 园区来的人 | 不是正式访问。不穿白大褂，不碰患者，不记录。站在检验科屏幕前看数值，记在脑子里，然后离开 |
| 信息流向 | 单向。急诊科给园区提供观察机会，园区不反过来指导治疗：园区不做临床 |
| 频率 | 不高。一两个月一次。每一次都让 R4 的人更确认一件事：暗礁社吃的那个药，不是简单把什么药用魔力泡一泡。它的某些代谢副产物，看起来像是某个人在用本科化学实验的设备尝试有机合成 |

## 其他相关线

### 急诊收治的魔力损伤

不是所有魔力损伤都和暗礁社有关。正经的武者修炼受伤、魔法师能级突破失控、灌魔操作事故：也走急诊。急诊科是医院所有魔力相关科室的第一道门。

| 损伤类型 | 急诊处理 | 后续去向 |
|----------|----------|----------|
| 魔力回路撕裂（轻中度） | 止痛 + 初步稳定 + 转魔力损伤科 | 魔力损伤科 |
| 灌魔过量 | 脱离魔力环境 + 补液 + ICU（重症） | ICU 或魔力损伤科 |
| 走火入魔（轻中度） | 针刺急救（急诊护士中有人考过中医急救针灸证）+ 转中医科 | 中医科 |
| 走火入魔（重度） | 急诊稳定 + 通知魔力损伤科和中医科联合接诊 | 联合收治 |
| 暗礁社药物中毒 | 补液 + 心电监护 + 血清魔力残留监测 + 留观 | 急诊留观或中医科 |

### 雾夜的渔民

雾夜那几条线（浓雾夜不归的渔船、海湾开口外不属于任何已知船只的灯光）急诊科偶尔也接到这些事故的后续。被找到的渔民，如果还活着，送来的通常是低温溺水。但急诊科最年长的护士提过一次：曾经有一个雾夜送来的伤者，皮肤上附着一层干燥的、泛蓝光的粉末。不溶于水，不溶于酒精，用胶带粘掉后，皮肤是完好的。患者醒来后不记得雾夜发生了什么。护士长把粉末取样交给检验科：检验科查了所有的毒理学和化学数据库，没有匹配。那管样本之后被园区魔力材料组借走了，再没还回来。
