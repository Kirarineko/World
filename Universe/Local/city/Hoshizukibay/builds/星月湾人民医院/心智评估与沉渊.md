# 心智评估与沉渊科

> 四楼东翼尽头的走廊。出了电梯要拐三个弯，路过消防器材箱和两扇常年锁着的储物间。这是全院最不好找的科室。

## 定位

心智评估与沉渊科是全市唯一处理魔力心智代价的临床科室。它有两个面向：

1. **心智评估**：对魔法师、武者、特殊案例进行魔力心智影响的临床评估——执念放大到什么程度构成临床问题？共情偏移是否需要干预？
2. **沉渊病区**：收治归渊失败者——身体完好，意识不完整的患者。这不是精神病区。这些人不是疯了，是有一部分被留在了归渊里。

科室常年安静。心智评估的患者来一次就走。沉渊病区的患者不出门。

## 背景：魔力心智代价的临床框架

魔力影响心智——这是本地泡的客观事实。但这种影响如何与"个性""情绪""精神疾病"区分，学术界和临床界仍在摸索。

| 现象 | 临床定义 | 诊断困境 |
|------|----------|----------|
| 执念放大 | 修炼者的核心性格特质被魔力持续强化，超出正常个性范围。临床标准：对日常生活或社会功能产生可观察的负面影响 | 如何区分"一个本来就固执的人被魔力放大了"和"一个本来就极度固执的人没被魔力影响"？没有基线就没有诊断 |
| 共情偏移 | 对非修炼者的情绪共鸣能力持续减弱。临床标准：患者本人意识到并报告了这一变化（如果他自己不觉得，诊断几乎不可能） | 本人不求助就不会被发现。高阶修炼者中有多少共情偏移未被识别——没人知道 |
| 魔力回声 | 停止使用魔力时意识深处残留的异常感知。临床标准：回声持续时间超过魔力停用后 24 小时，或内容令患者痛苦 | 回声的"内容"高度私人化。它可能是金属摩擦声，也可能是已故师长的声音。无法标准化 |
| 归渊后心智改变 | 归渊过程中意识基底结构重组导致的人格变化。程度从轻微性格偏移到彻底的性格重塑不等 | 归渊之后的人还是"同一个人"吗——这不是临床问题，但它是所有临床问题背后的幽灵 |

目前科室使用的诊断标准是六年前发布的《魔力心智影响评估指南（试行）》。全院只有这本指南上没有"第X版"的字样——它从来没被修订过。不是不需要，是没攒够数据。

## 心智评估流程

```
初筛问卷（自填）→ 结构化访谈（心智评估科医生，~90分钟）
→ 魔力暴露史量化（患者自行估算近一年魔力使用频率/强度/时长）
→ 对照：患者家属/密切接触者的第三方观察问卷
→ 结案：正常范围 / 观察随访 / 临床干预建议
```

整个过程大概三小时。来的人不多。大多数患者是被家人推来的——"我觉得他没变，但他队友说变了，我不知道该信谁"。

最让医生头疼的不是诊断，是患者来了就开始用魔力。因为紧张，因为习惯。医生反复说"今天不要用魔力，会影响基线"，然后患者在访谈中无意识地在指尖召了一小团魔力去搓——只是一个下意识动作。医生在表格上划掉刚才写的"魔力暴露：无，近24h"，重写"24h内有微量使用，患者不承认——不是撒谎，是不记得"。

## 沉渊病区

### 何为沉渊

归渊是武者突破归三时必须经历的心理临界。魔力占比越过某个阈值，意识的基底结构开始重组。撑过去的人踏入归境——魔力与肉身深度融合，每一拳都是物理打击也是魔力释放。

没撑过去的人叫沉渊者。

### 沉渊者的状态

| 条目 | 说明 |
|------|------|
| 身体 | 完好。在生物医学层面，沉渊者是完全健康的人类。他们能走路、能吃饭、能睡觉。一些沉渊者甚至保留着修炼获得的体能——一个沉渊的归二武者可能在肌肉力量上仍然碾压普通金境 |
| 意识 | 不完整。意识仍然在运作，但像是在一个少了一部分的拼图里运作。这块缺失不是生理性的脑损伤（CT 和 MRI 正常），是魔力层面的——归渊过程中意识基底结构重组到一半卡住了，停在一个"既非旧也非新"的中间态 |
| 行为 | 高度异质。有些沉渊者只是"反应迟缓"——能听懂你的话，但要过十几秒才点头。有些会重复一句话一整天的每个小时。有些看起来完全正常，直到你发现他们从不主动发起任何行为——不饿、不渴、不说话、不移动，不是不能，是"不"——意识里缺了"发起"这个功能 |
| 情感 | 残损。一些沉渊者对情绪仍有感知但非常迟钝——家人哭泣他们会看着，然后转头去看窗外的树。不是冷漠，是"感受到悲伤"这个体验在他们意识里只完成了接收，没完成理解 |

### 收治规模

病区有 24 张床。常年住着约 18–20 位患者。不是星月湾有这么多沉渊者——周边几个城市的沉渊者也转到这里。本市自己也只出过寥寥几个沉渊病例。沉渊的发生率极低，但一旦发生，没有地方去——普通医院的精神科不懂魔力，魔法师/武者社区不会回访一个"失败了"的人。

### 治疗方案

| 层级 | 内容 |
|------|------|
| 基础照护 | 营养支持、生活护理、预防并发症。沉渊者和长期卧床的重症患者不同——他们能走能动，但需要别人替他们决定什么时候走、走到哪。护士的工作量不在体能力面，在"持续为患者做每一个微小决定"——上午几点起床、午饭吃什么、下午要不要去走廊散步。沉渊者自己不会做这些决定 |
| 中医安神 | 中医科的安神定志方剂对沉渊者有可观察的镇静止颤效果。部分沉渊者的重复性言语在服药后频率下降。机制不明——可能是方剂中的某些成分对残存的魔力回路有温和的调节作用 |
| 魔力微刺激 | 最轻微、最安全的外部魔力干预。在患者经脉特定穴位施加极低浓度的魔力脉冲，观察意识反应。这不是治疗——这是诊断性试探。如果一个沉渊者对特定穴位刺激有微弱的瞳孔反应或呼吸变化，说明他的意识残存部分仍然能对魔力做出响应。这扇窗的存在本身就是一个方向 |
| 家庭参与 | 家属被鼓励定期探视，与患者进行单向交谈——说家里最近发生的事，说今年春天樱花开得怎么样。不期待回应。但有记录的案例中，一位沉渊三年的患者在听到"你女儿考上大学了"之后，确实抬头看了说话的人一眼。只有一眼。之后恢复原状。但这说明有些东西还在里面 |

> 沉渊病区没有"出院"这个词。没有治愈出院的先例。偶尔有家属接回家的——但通常一两个月后又送回来，因为在家照护一个不需要照顾也不需要陪伴的亲人，在心理上比照顾一个不能自理的人更难。

## 与中医科的协作

中医科老主任每周来沉渊病区查一次房。不是会诊——会诊意味着西医提出问题中医来回答。老主任的查房是另一种格式：坐在患者床边号脉，号很久。然后写方子。

西医问过他号脉号出什么。他说："正常的脉是气血运行的痕迹。沉渊者的脉里有一部分是空的——不是血管空了，是那部分血里缺了某种'活着的冲动'。你把这叫意识，我叫神。叫法不一样，摸到的东西一样。"

魔力损伤科的年轻医生对这段话的评价是"诗意但不科学"。心智评估科的主任没说话。他没说话的原因是他不知道该站在哪边——他受过西式临床训练，但沉渊者的磁共振影像确实看不出问题，而老主任的方子确实让一些患者的重复性言语频率降下来了。

## 速查

```
心智评估：执念放大 / 共情偏移 / 魔力回声 / 归渊后改变
诊断流程：问诊 → 访谈 → 魔力暴露史 → 家属问卷
沉渊病区：24床，常年住18-20人，全市唯一
沉渊者：身体完好，意识不完整，无出院先例
治疗：基础照护 + 中医安神方剂 + 魔力微刺激诊断性试探
核心困境：MRI正常但人不是原来的那个人——没有临床证据，但有临床事实
```
