# 中医科

> 一楼东翼。紧挨着草药房的窗下种了一排薄荷和紫苏，不是绿化，是应急。护士偶尔掐一片薄荷叶子泡茶，"比休息室那些茶包好喝。"

## 定位

中医科是星月湾人民医院最老的科室之一：比魔力损伤科老，比跨越者门诊老得多。在全市还没有"魔力医学"这个概念的时候，中医科已经在治走火入魔了。不是因为他们懂魔力：是因为经脉理论和魔力回路的天然映射，让中医在魔力医学诞生之前就已经在"魔力医学"里工作了几百年。

科室有住院病床 40 张，门诊日接诊约 80-100 人次。就诊者以本地中老年居民为主，但武者和魔法师的比例在逐年上升：魔力损伤科出院的患者，约一半会转入中医科做康复。

## 经脉即魔力回路

中医科的底层认知框架是：古人描述的经脉，本质上是对体内魔力自然流通路径的经验性映射。

| 证据 | 说明 |
|------|------|
| 经脉走向与魔力扩散路径吻合 | 在低魔力浓度环境下，被注入人体微量的魔力示踪剂会沿着经典十二经脉的路径扩散，而非随机渗透。同一个示踪实验在不含魔力的对照组（无感者，体内无一丁点魔力）身上，示踪剂无此倾向 |
| 针刺得气与魔力波动 | 在正确穴位施针时，针尖附近可检测到局部魔力浓度的瞬时波动。针刺偏位时无此反应。魔力损伤科的造影室曾为一位接受针灸的患者做过实时监测：针尖刺入足三里的瞬间，示踪剂影像上出现了一个直径约 3 厘米的环形亮斑，像一滴水落入水面 |
| 灸法的热-魔力耦合 | 艾灸燃烧产生的热量中，检测到极微量的魔力释放。这意味着艾叶（一种特定产地的植物）可能在生长过程中从土壤中富集了微量魔力。热量释放时，这部分魔力以特定频率散入皮肤，对被灸的经络穴位产生温和的魔力刺激 |
| 中药归经 | 部分中药的活性成分在人体内分布时，确实会优先富集在某些特定经络沿线组织。这不是"某一个成分去了肝经"：是某个成分的代谢产物在肝经沿线的组织液中浓度更高。为什么？目前的理论是这些成分的分子构象对低浓度魔力有某种亲和作用，而魔力的自然分布路径正是经脉 |

> 中医科的年轻医生在入职培训时被要求读一篇论文：HBI 计科和生科联合发表的《人体经脉走向与低浓度魔力流场的一致性建模》。论文最后一句是："本课题不建议对经络的本质做任何结论性判断，仅报告流体力学层面的观测数据。" 中医科老主任的评价是："你没下结论，但你把数据给做了。剩下的我自己会说。"

## 走火入魔

| 条目 | 说明 |
|------|------|
| 现代定义 | 魔力在非修炼状态下误入非经脉路径，造成局部软组织魔力灼伤 |
| 传统描述 | "火"是魔力过量，"魔"是魔力失控。火入魔走即为魔力在非预期的路径上失控 |
| 分级 | **轻度**：四肢末端魔力刺麻感，局部皮肤发热，无器质性损伤。针灸调理 1-3 次即可。**中度**：魔力沿手三阴或足三阳逆行，伴随沿经脉灼痛和肌肉痉挛。针灸 + 方剂，疗程 1-2 周。**重度**（魔力冲击丹田或心俞，可能累及魔力损伤科）此时已是跨科室联合处理，"炼丹走火""冲关反噬"均属此类 |
| 中医科的独特优势 | 重度走火入魔需要魔力损伤科处理急性损伤，但中度和轻度的走火入魔：全市只有中医科能系统处理。纠偏针灸的核心原理是："引导魔力回到该走的路径上。"这不是比喻 |

## 灵药与魔力本草

| 条目 | 说明 |
|------|------|
| 定义 | 在自然生长过程中从土壤或环境中富集了微量魔力的药用植物 |
| 确认品种 | 目前经检测确认含魔力活性的品种：特定产地的灵芝、黄芪、人参、茯苓、艾叶。所有确认品种的魔力浓度均极低：远不足以"催动魔法"，但足以对次生代谢产物的合成路径产生催化作用 |
| 催化效应 | 以黄芪为例。含魔力活性样本中的黄芪甲苷 IV 含量比普通种植品高约 35%。机制假说：魔力作为一种外来能量，作用于植物细胞的次生代谢酶系，降低了某些合成路径的活化能。用中医科老主任的话说："好黄芪本就好，沾了点魔力的更好，但前提是它本来就该好" |
| 药圃 | 中医科药房在院区最东侧有一块约 200 平方米的露天药圃，种植了十几个"怀疑有魔力活性的品种"。药圃的土壤来自海湾周边的山丘：星月湾的土壤本身魔力含量极低，但比市区土壤略高。药圃不是正式的研究项目，是个别医生的个人兴趣。但 HBI 生科的人最近开始来取土样了 |

## 低转者与经脉壅塞假说

中医科有一派观点认为（低转者的魔力转化效率低下，本质是经脉壅塞）经脉路径上存在某种"瓶颈"，阻碍了魔力在体内的自然流通。壅塞越重，可吸收的魔力越少。

| 立场 | 人物 | 陈述 |
|------|------|------|
| 中医科内部支持者 | 老主任 | "有的低转者号脉时，关脉以下有明显涩感：不像是你们说的什么'转化效率'，像是有东西堵着。我这里针了二十年的魔力损伤病例，壅塞是可以疏通的" |
| 中医科内部质疑者 | 年轻主治 | "您每次给低转者针灸完人家都觉得'身体轻快了一些'，但血清魔力残留量没有变化。'轻快'和'转化效率'是两件事" |
| 魔力损伤科的态度 | — | "没有客观指标证明经脉壅塞存在。但我们也没有客观指标证明它不存在。" |
| R4 转化楼的态度 | — | "私下对这假说有兴趣，但它不是可量化的生物标志物。没有标志物就没法做实验。没法做实验的事我们做不了" |

## 药房

中医科药房在科室走廊最东侧尽头。走进去先闻到的是混合的药味，不是消毒水味，是含挥发油的草药香。几百种饮片分装在贴着毛笔手写标签的抽屉里。标签上的字不是一个人的：老主任写了几十年，最近几年手开始抖，年轻医生接过笔。笔迹变了但抽屉没变。

> 药房有一个不成文的规定：所有涉及魔力活性本草的处方必须由副主任医师以上签字。不是危险：是这些品种目前没有经过任何国家的药典收录。处方的法律风险由签字者承担。
