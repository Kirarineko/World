
# TCC · Timeless Colorful City

> 永恒缤纷之城。他们做了一个聊天工具，然后把这座城市的名字写进了公司名里。

## 概况

| 条目 | 说明 |
|------|------|
| 全称 | Timeless Colorful City（永恒缤纷之城） |
| 简称 | TCC |
| 成立 | 约 10 年前（重建纪中期） |
| 创始人 | 墨白，HBI 计算机科学学部毕业生 |
| 员工 | 约 280 人 |
| 总部 | 星月湾南湾，HBI 南栈后街以北约 300 米 |
| 业务 | 社群聊天平台 ColorFul |

TCC 是星月湾第二家具有全国影响力的科技企业。与雷云科技不同：雷云做的是算力基础设施和 AI 模型，TCC 做的只是一个聊天工具。但这个聊天工具已经是该类目的主流产品。

它的创始人墨白不是明星创业者。HBI 计科普通毕业生，成绩中上，毕业设计做的分布式即时通信系统的原型。毕业后在月心区一家小型软件公司写了两年后端，然后辞职，在自己租的公寓里写出了 ColorFul 的第一个版本。

早期用户就是 HBI 的学生。第一批服务器是几个宿舍楼的群组：同学之间传文件、约实验、讨论课题。没有人把它当"创业项目"。墨白自己也没有。

> 他不是那种"改变世界"的人。他就是觉得市面上没有好用的社群工具，自己缺一个，就写了一个。写得不错，别人也开始用。

## ColorFul

### 定位

ColorFul 是一个社群式聊天平台。它的核心结构是**服务器（Server）+ 频道（Channel）**：每个服务器是一个独立社群，内部按主题拆分文字、语音、视频频道。用户可以自由创建或加入服务器。

| 条目 | 说明 |
|------|------|
| 模式 | 官方托管 + 用户自部署 |
| 官方托管占比 | 约 60% 的活跃服务器 |
| 自部署占比 | 约 40% |
| 客户端 | 开源，功能不限 |
| 后端 | 官方服务器运行于 Raiun 钢云 |

### 架构

ColorFul 的服务器架构分为两类（**官方托管服务器**）由 TCC 在 Raiun 钢云上统一运维。用户按订阅计划付费，TCC 负责稳定性、存储、备份。这是大多数普通用户的选择。

**自部署服务器**：ColorFul 的服务端代码开源。任何组织或个人可以在自己的硬件上搭建 ColorFul 实例。自部署版本的功能与官方版本完全一致，但不享受 TCC 的运维支持。

> 自部署模式是 ColorFul 区别于同类产品的关键设计选择。TCC 不强制任何人使用官方托管：你想自己搭、自己管，代码拿去。这不是商业策略，是墨白个人偏好的延续：他当初写 ColorFul 就是为了自己用。自己写的工具，没道理不让人自己跑。

### 与 Raiun 的合作

ColorFul 的官方托管服务器运行在 Raiun 钢云上：虚拟服务器、托管数据库、消息队列、对象存储，全部走钢云产品线。TCC 是 Raiun 在星月湾本地的重要客户之一。

| 层面 | 内容 |
|------|------|
| 计算 | 钢云 VPS 集群，承载官方服务器的实时消息路由 |
| 存储 | 钢云对象存储，存放文件、自定义表情、日志 |
| 峰值 | 节日期间（尤其是樱祭和灯塔节），ColorFul 的流量会给钢云带来显著负载波动：Raiun 的数据中心运维和 TCC 的后端团队在每年三月末和八月末有一个非正式的"战时联络窗口" |
| 关系 | 纯客户-供应商。没有投资关系，没有独家协议。TCC 选择 Raiun 的原因很简单：钢云性价比高，且两家公司总部隔了不到两公里，出问题时墨白可以走路去拍桌子 |

IRIS 的典型用户不是普通社群：是企业内部协作平台、大型游戏公会、全国性粉丝社区，以及少数"不方便透露名称的机构客户"。

## 客户端

ColorFul 的客户端完全开源。桌面端（Windows / macOS / Linux）、移动端（iOS / Android）、Web 端三端覆盖。功能不设限：任何用户，不论是否付费、不论在哪个等级的服务器上，都可以使用客户端的全部功能。

> 客户端仓库的 README 第一行写着："这是我们用的客户端。如果你觉得哪里不好用，issue 开着，PR 也开着。"最后一句话是墨白亲手写的。

客户端开源的副作用是：市面上存在多个 ColorFul 的第三方客户端。有些做减法（极简模式、低配机适配），有些做加法（主题美化、插件系统）。TCC 对此的态度是"只要不冒充官方，随便改"。

## 自部署生态

ColorFul 自部署版本的服务端代码同样开源。部署要求极低：一台最低配的钢云 VPS 就能跑一个几百人的小服务器。这意味着：

- **学生社团**可以自己搭一个内部交流平台，不花一分钱
- **暗礁社**可以在自己的服务器上协调活动，不需要经过 TCC 的官方基础设施
- **跨越者社区**可以拥有完全私密的讨论空间，不依赖任何外部托管

TCC 不审核自部署服务器上的内容：也审核不了。自部署意味着服务器完全在用户自己的控制之下，数据不经过 TCC。这不是"宽容"：是技术架构的客观结果。

> 墨白在唯一一次公开采访中被问到"如果有人用自部署版本做坏事怎么办"。他的回答是："菜刀可以切菜也可以砍人。我不卖菜刀给砍人的人，但菜刀本身没有错。"

## 与城市的关系

TCC 的根扎在星月湾，比 Raiun 更深。

**命名**："Timeless Colorful City"直译是"永恒缤纷之城"。这是一个外来者会觉得浮夸、本地人会觉得理所当然的名字，因为星月湾本身就是一座缤纷的城市。墨白说这名字是他在樱巷咖啡馆里想的，"本来想叫星月湾聊天室，被朋友拦住了"。

**办公室**：南湾一栋四层小楼，外墙刷成暖灰色，窗台种满茉莉。步行到 HBI 南栈后街约五分钟，到 Raiun 总部约二十分钟。办公室没有前台：进门是一面贴满 ColorFul 用户画作的墙，画的内容从猫到机甲到风景，来源从五岁小孩到职业插画师。这面墙叫"缤纷墙"，是公司唯一的装饰预算。

**与节日**：ColorFul 是星月湾六大节日在线上最集中的组织平台，没有之一。樱祭的"樱巷纸灯"筹备群、渔获祭的"船队投票"频道、灯塔节的"登塔排队实时通报"、星月夜的"今晚哪个望台风最小"：都在 ColorFul 上。这不是 TCC 的商业推广，是本地人自己用起来的。墨白自己每年都在樱祭筹备群里，不发消息，只看。

**与 HBI**（墨白每年回 HBI 做一次分享，内容不是"怎么创业"，是"计科楼三楼男厕左边第二个坑位的wi-fi信号最好"）以及"不要把全部心思放在刷 LeetCode 上，如果有想写的东西，现在就写"。分享会结束后照例去南栈后街盖饭店，点的还是十年前的四种盖饭之一。

> TCC 的员工里，约三分之一是 HBI 毕业生。不是招聘偏好：是投简历的人本来就多。

| 层面 | 说明 |
|------|------|
| 商业 | TCC 是 Raiun 钢云的客户。合作关系，没有交叉持股 |
| 体量 | Raiun 约 1200 人，全泡客户；TCC 约 280 人，主要国内 |

> 两栋楼隔着二十多分钟的步行距离。一家是算力的源头，一家是社群的水渠。在这座城市里，它们各自在各自的赛道上安静地运转，偶尔在对方的账单上留下自己的名字。
