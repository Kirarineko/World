# 饮食

> 星月湾人不太谈"料理"——这个词太大了。他们谈"今天上岸了什么"。

---

## 海鲜——海湾的底味

星月湾的饮食没有"代表作"。如果非要挑一样接近这个定义的东西——那就是海鲜。不是因为做得特别厉害，是因为出门左转就是渔港。食材决定一切，剩下的只是别把鱼烧焦。

### 本地渔获

| 品种 | 季节 | 常见做法 |
|------|------|----------|
| 真鲷 | 春-夏 | 刺身、清蒸、盐烤。春天的鲷鱼脂肪刚好，入夏后转瘦但仍然好吃 |
| 鲭鱼 | 秋 | 盐烤、味增煮。秋天的鲭鱼油脂最厚，烤出来皮脆肉裂，是星月湾人认定的"秋天来了"的信号 |
| 枪乌贼 | 全年 | 刺身（活造，鱼肉透明脆弹）、炭烤（整只串烤，刷酱油，海鲜早市即食档的招牌）、煮付け。不炸——本地人觉得炸浪费了甜味 |
| 虾蟹 | 冬-春 | 甜虾刺身、清蒸蟹。冬天的甜虾腹中带籽，是本地产妇坐月子的传统食材——科学上没道理，传统不听科学 |
| 贝类 | 全年 | 味增汤、黄油烤、酒蒸。蛤蜊味增汤是星月湾家庭餐桌的固定开场 |
| 海胆 | 夏 | 生食、海胆盖饭、蒸蛋。初夏的海胆最肥 |

### 海鲜早市——不仅是买菜

海鲜早市的核心功能不是买菜，是吃饭。市场边缘有几排即食摊，凌晨五点出摊、渔获卖完收摊——通常中午之前。摊主多半是渔民家属，摊位是固定的，但菜单每天有一个菜是"今天打上来的那块东西"——不写在招牌上，靠老板口头通知。

| 摊位 | 常驻菜单 |
|------|----------|
| 炭火档 | 整只炭烤鱿鱼（刷酱油）、烤鱼下巴（鲷鱼或鲭鱼，看当天渔获）、烤贝柱串 |
| 鱼骨汤档 | 用杂鱼骨和昆布长时间熬煮的浓汤，加豆腐和葱花。一碗两百日元不到。早市最暖手的东西 |
| 刺身档 | 比目鱼薄造、真鲷厚切、枪乌贼活造。蘸料不是芥末是当地产的柑橘醋（ポン酢） |

> 海鲜早市是星月湾唯一一个"不认识也可以坐同桌"的地方。一张长凳上可能挤着一个渔民、一个HBI生科的学生、一个刚下夜班的护士。三个人吃的东西不一样，都不说话。不是不想说——是嘴里有鱼。

## 市井小吃

星月湾的小吃没有"小吃街"这种集中形态。摊贩散落在樱巷入口、海鲜早市边缘、月心广场周末小集、和丘麓步道入口。以下是各季节的常见面孔：

| 季节 | 小吃 | 说明 |
|------|------|------|
| 春 | 樱饼 | 樱花叶包糯米团，内馅是红豆沙。樱祭期间樱巷入口至少有六个摊在卖，味道大同小异——但每个摊都有忠实顾客。忠实顾客不是看口味，是看摊主是谁——"那是我小学同学他妈" |
| 春 | 萤乌贼佃煮 | 春天的萤乌贼捕捞季短，用酱油和糖煮成咸甜口，配白饭或清酒。过了三月就没了 |
| 夏 | 海藻凉粉 | 石花菜熬制的凉粉，浇柑橘醋和姜末。口感接近果冻但不甜。解暑首选 |
| 夏 | 冰镇海胆 | 海胆连壳冰镇，撬开直接吃。海鲜早市旁边卖的海胆是不撒酱油的——撒酱油在老渔民的认知里等于"不新鲜才遮味"。本地海胆不需要遮 |
| 秋 | 烤秋刀鱼 | 秋天的秋刀鱼油脂丰沛，整条串在炭火上烤，挤柠檬或酸橘。海鲜早市炭火档的秋季限定 |
| 秋 | 银杏饭 | 丘麓步道两侧的银杏树进入观赏期后被游客拍完照的银杏落在地上——会被本地人捡起来。银杏果去壳煮软，拌入白饭加酱油和味淋。苦中回甘，小孩通常不喜欢 |
| 冬 | 牡蛎煎 | 小颗本地牡蛎裹薄面糊煎至边缘焦脆，配甜辣酱。冬天海鲜早市旁边的移动摊，寒风中排队的人手插在口袋里原地跳 |
| 冬 | 鱼形烧 | 鲷鱼形状的烤饼，内馅是红豆沙或卡仕达奶油。来源是更北方的"鲷鱼烧"的本地变体——星月湾人觉得鲷鱼是菜不是点心，所以改叫"鱼形烧" |
| 全年 | 味增鱼骨汤 | 早市鱼骨汤档的全年供应。冬天喝一碗，夏天也喝一碗——夏天喝得很慢，因为烫。但还是要喝 |

## 咖啡馆与甜点

### 樱巷的咖啡馆

樱巷的咖啡馆菜单大同小异——咖啡豆的供应商就本地两家烘焙商，设备也差不多。区别不在菜单，在环境：

- 靠巷口那家窗台朝东，早上有光，常客是退休老人和上早班前坐二十分钟的上班族
- 中间那家有两层，二楼阳台正对樱巷，春季一座难求
- 靠北湾中学方向那家最安静，常客是高中生——不是来做作业的，是来聊天的。做作业的高中生都在星月湾旅馆大堂，因为那里有免费的WiFi

三家都卖咖啡，三家都卖芝士蛋糕。区别是：靠巷口那家的芝士蛋糕偏酸（老板喜欢放柠檬皮），中间那家的偏甜，靠中学那家的芝士蛋糕是老板自己烤的——形状不稳定，有时塌，有时不塌，但比供应商送来的好吃。

### 南栈后街的学生咖啡馆

HBI南栈后街的三家咖啡馆菜单比樱巷更简单——美式、拿铁、可可，三样打底。杯量大，价格低。学生的需求很明确：能坐两个小时不被赶，有WiFi。其中一家深夜营业到凌晨两点，主要客流是自习室抢不到座的学生和实验楼出来需要咖啡因的人类。

### 甜品

| 甜品 | 说明 |
|------|------|
| 柚子芝士挞 | 冬季限定。本地柚子——不是很大很甜的那种柚子，是酸香浓烈的小柚子——加在芝士蛋糕上。全市甜品店都会做，做法相差无几 |
| 海盐焦糖冰淇淋 | 全年。手工冰淇淋店在南湾住宅区有一家。海盐从海水晒制——不是噱头，只是"刚好在海边" |
| 樱饼（甜品版） | 樱祭之外也能买到。北湾老城区的和果子铺常年供应，不是季节性限定。但春天之外的樱饼用的是腌渍樱花叶（前一年腌的），味道略重 |

## HBI食堂群

| 食堂 | 位置 | 主力菜品 | 学生评价 |
|------|------|----------|-----------|
| 第一食堂 | 核心区，近教学区 | 定食为主：烤鱼定食、炸鸡定食、咖喱饭。每天有一个"不确定菜"——食堂阿姨早上逛海鲜早市买的当天便宜渔获，做成什么取决于阿姨的心情 | 最稳定。不能说出彩，但不会踩雷。适合天天吃 |
| 第二食堂 | 核心区，近实验室 | 偏清淡：沙拉吧、豆腐料理、蒸鱼套餐。实验楼出来的学生不想闻油炸味——第二食堂的炸物窗口在角落，不炸的时候窗口是关的 | "健康。但吃了一周之后会想吃炸鸡。然后你就去了第一食堂。" |
| 第三食堂 | 生活区，近宿舍 | 最便宜：盖饭、炒面、蛋包饭。夜间盖饭窗口开到十点 | 深夜从C4下来的人的第一选择。"不是最好吃，是最近的。而且便宜。" |

三个食堂之间的竞争温和且不成文。没有一家会复制另一家的主力菜单——不是因为协议，是因为三家食堂的采购和人员彼此知道，抄菜单属于"可以做但不体面"的事。食堂阿姨之间的信息交换比任何正式渠道都高效：早市哪个摊的鱿鱼今天特别好，三家食堂的人到了中午都知道了。

> HBI没有"美食街"——学生街太短，撑不起。但第一食堂门口每周四中午有一个海鲜早市的分摊——是渔港那边一个摊主和食堂阿姨的私人交情。卖的是不定期的随机渔获：可能是鱿鱼，可能是贝类，可能是某种食堂阿姨也不认识的鱼。认识不认识不重要。"不认识说明我们不常吃——不代表不好吃。"

## 家常与季节

星月湾的家庭餐桌没有固定"菜系"。烹饪方式受渔业和气候驱动而非文化传统。

| 季节 | 常见家庭菜 |
|------|-----------|
| 春 | 萤乌贼冲漬（生萤乌贼用酱油和酒渍半小时即食）、竹笋煮鲷鱼、樱虾什锦饭 |
| 夏 | 海藻凉粉、冷奴（冰镇豆腐配生姜酱油）、素面、生海胆拌饭 |
| 秋 | 盐烤秋刀鱼（配萝卜泥和酸橘）、银杏饭、鲭鱼味增煮 |
| 冬 | 海鲜寄世鍋（随手边的鱼贝蔬菜扔进锅里煮，蘸柑橘醋吃）、牡蛎味增汤、柚子胡椒烤鸡 |

星月湾家庭的厨房常识：
- 每家有一个"鱼盘"——椭圆形陶瓷盘，专门盛整条烤鱼。没有鱼盘的家庭要么是刚搬来的，要么是不吃鱼的——第二种情况在这个城市很少见
- 味增汤的配料随季节变化：春天是蛤蜊和蕨菜，夏天是豆腐和裙带菜，秋天是滑子菇，冬天是萝卜和油豆腐
- 酱油消耗量在全国人均水平中偏高——不是因为口味重，是因为海鲜蘸酱油。但海鲜早市的刺身档不用酱油——用柑橘醋——这说明"海鲜蘸酱油"更多是家庭习惯，不是本地的"正宗"做法

## 酒与夜食

### 居酒屋

北湾老城区深处藏着三家小居酒屋，都不显眼——门面窄，招牌小，全靠老客带新客。菜单写在墙上的黑板上，鱼的部分每天换——根据老板早上去海鲜早市买了什么。三家店的共同菜只有一道：烤鱼下巴。区别在酱汁——一家偏咸，一家偏甜，一家加了一味老板不肯说的香料。

### 海藻酒

星月湾渔家自酿的低度酒。用本地马尾藻和糯米发酵，酒精约十几度的浊酒，带微甜和若隐若现的海味。甜度每批不一样——取决于发酵季节的温度和藻的品种比例。这个不稳定性被当作"年份特征"而非质量缺陷。在渔获祭期间，渔民互相交换去年的酒；交换的意义不是品鉴，是"你去年那批比我的好——今年帮我调一下"。

> 市面上没有海藻酒的商业品牌。有人提过，酿酒的人说"量太少，自己喝都不够"。实际原因可能是——一旦标了定价，就要保证每一批味道一致。而没人想为此付出努力。

### 深夜食堂

南栈后街有一家深夜营业的盖饭店，菜单上只有四种盖饭（亲子丼、炸鸡丼、海鲜丼、猪肉丼），开到凌晨两点。顾客构成：HBI晚自习出来的学生、C4实验楼关了灯还不想回宿舍的人、偶尔有研究园区下夜班的研究员（R2和R4都是二十四小时实验室）。这家店的存在不是因为星月湾的夜间经济——是因为HBI的存在。

## 跨越者吃什么

| 群体 | 进食情况 |
|------|----------|
| 低阶跨越者（O·虚/P·极） | 正常进食。碳基代谢仍占主导，但味觉有偏移：咸味感知钝化，鲜味感知敏感。可能是洋能对味蕾细胞的微改造。常见偏好：偏爱淡味，不爱腌制食品，对海胆和贝类的天然鲜味有异常热情 |
| 高阶跨越者（5d曲/6p坍） | 不需要进食——洋能直接供给能量。但"不需要"不等于"不吃"。高阶跨越者会出于社交、怀旧或纯粹的口腹之欲进食，就像人类不需要甜点但会吃。味觉与人类差异更大：酸甜苦咸四基味极弱，鲜味保留，对食物质地的敏感度远超人类——一个高阶跨越者可能因为"这口牡蛎的口感曲线不对"而皱眉，但旁边的普通人类觉得完全一样 |
| 纯洋能生物（7s寂·浑以上） | 理论上可以进食（洋能构成的感官并非完全闭塞），但几乎不存在这方面的记录。已知的寂·浑存在个位数，没人研究过他们吃不吃东西。如果吃了，大概也不是为了营养 |

星月湾没有专门的跨越者餐厅。跨越者人数太少，支撑不起。但跨越者社区内部聚餐时会调整调味——海鲜锅底不加盐，靠昆布和贝类自带的鲜味出汤。跨越者版本的味增汤用的是减盐味增，普通人类喝了会觉得淡如水，跨越者觉得刚好。

人民医院跨越者门诊的候诊室茶几上有一盒饼干。盐味极淡。没人记得是谁放的。一直有人在补。

海鲜：真鲷、鲭鱼、枪乌贼、虾蟹贝、海胆——做法清蒸盐烤为主，不搞复杂调味
早市：凌晨至中午，买菜+吃饭混合。必吃：炭烤鱿鱼、鱼骨汤、刺身（蘸柑橘醋）
小吃：樱饼(春)、海藻凉粉(夏)、烤秋刀鱼(秋)、牡蛎煎(冬)、鱼形烧(全年)
