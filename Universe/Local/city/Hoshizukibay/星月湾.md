# 星月湾

> 类型：中型沿海城市 · 本地泡主角活动主舞台

## 概述

星月湾是一座人口约 72 万的沿海中型城市，坐落于几近完美的圆形海湾内。海湾得名于夜幕下独特的视觉奇观——海面船灯与岛屿灯火星星点点，市区沿月牙形平地铺展，万家灯火宛如一弯蛾眉月倒映海中。"星"与"月"俱在，故名星月湾。

这座城市不追求高度，不追求密度。它追求的是"刚好"。刚好的海风，刚好的树影，刚好的烟火气。

## 地理

### 海湾

| 条目 | 说明 |
| - | - |
| 形态 | 近乎完美的圆形海湾，仅有一处狭窄开口连通外海 |
| 边缘 | 一圈低矮山丘，海拔不高但足以将城市与内陆隔开，形成天然的围合感 |
| 平地 | 山丘到海滩之间的月牙形平坦地带，城市即建于此 |
| 开口方向 | 东南，朝向开阔海域 |
| 海湾水域 | 平静，少大风浪。天然良港 |


### 岛屿

海湾内散布少量小型岛屿。

| 类型 | 说明 |
| - | - |
| 近岸岛 | 距岸数百米，有桥或堤道连接。部分建有公园、茶室、私人别墅 |
| 离岸礁 | 无建筑，植被覆盖，鸟类栖息。小型游艇偶尔泊于附近 |
| 灯塔岛 | 海湾开口处有一座小岛，建有灯塔。已自动化，不再驻人，但灯塔本身是城市标志之一 |


### 气候

| 条目 | 说明 |
| - | - |
| 类型 | 亚热带海洋性。冬无严寒，夏无酷暑 |
| 特征 | 海雾常见于春秋清晨。夏季午后偶有短时雷雨，雨后天晴极快 |
| 台风 | 海湾地形天然削弱风力，百年间极少形成严重灾害 |


## 城市格局

城市沿月牙形平地展开，从海湾最深处向外延伸。没有严格的功能分区，居住、商业、文化交错分布。

### 核心分区

| 区域 | 位置 | 特征 |
| - | - | - |
| **月心** | 海湾最深处，市区正中 | 市中心。较现代化，但高楼屈指可数。市政厅、图书馆、剧院集中于此 |
| **北湾** | 月牙北翼 | 老城区。原木风格建筑最密集的区域，街巷蜿蜒，多小型店铺 |
| **南湾** | 月牙南翼 | 较新开发。住宅与小型商业混合，绿化率极高。南湾-丘麓交界处坐落着星月湾泛生命科技学院（HBI） |
| **丘麓** | 山丘脚下 | 城市边缘。密度最低，部分住宅依山而建，可俯瞰海湾。丘麓步道中段内侧有星月湾研究园区 |
| **港区** | 月牙两端及部分沿岸 | 私人码头、游艇港、中小型货运港散布。没有重工业港 |
| **丘麓** | 山丘脚下 | 城市边缘。密度最低，部分住宅依山而建，可俯瞰海湾 |


## 建筑

### 居民区

| 条目 | 说明 |
| - | - |
| 风格 | 原木风。大量使用浅色木材、石材和白墙。坡屋顶，多有门廊和露台 |
| 层数 | 多为二至三层。独栋或联排为主，少量低层公寓 |
| 色调 | 暖色系。原木色、米白、浅灰为主，偶有淡蓝或淡绿墙面 |
| 细节 | 窗台花箱几乎家家都有。栅栏低矮，院门常开 |


### 市中心

| 条目 | 说明 |
| - | - |
| 风格 | 现代，但克制。玻璃幕墙被限制使用，建筑外层大量覆盖绿植立面 |
| 层数 | 最高建筑不超过二十层。天际线被严格控制 |
| 公共空间 | 小型广场和口袋公园散布。没有巨型购物中心，商业沿街分布 |
| 夜间照明 | 暖色路灯为主。霓虹极少，招牌多为内透光的木制灯箱 |


### 整体画风

> 日常动漫风格。不是赛博朋克的霓虹雨夜，不是蒸汽朋克的铜管齿轮。是夏日的海风、春日的樱吹雪、秋夜的灯影、冬日安静的港口。是你可以闻到海盐味、听见自行车铃铛声、看得见远处灯塔光晕的那种城市。

### 特色

北湾老城区有一条"**樱巷**"，两侧樱花树在春季形成花廊。路面是石板，车进不来。樱巷两侧是咖啡馆、旧书店和手作工坊。本地人经过时不会抬头看——不是看腻了，是这本身就是日常生活的一部分。

## 港湾

### 港口设施

| 设施 | 说明 |
| - | - |
| 私人小船码头 | 散布于湾内多处。停泊小型帆船、钓鱼艇为主 |
| 游艇港 | 南湾有一处专业游艇港。规模不大，但设施齐全 |
| 货运港 | 中小型。处理本地消费品和少量建材，非工业港 |
| 渔港 | 北湾端有一处小渔港。每日清晨有渔获上岸，旁边就是海鲜早市 |


### 海上交通

| 类型 | 说明 |
| - | - |
| 游船 | 日间有环海湾观光船，夜间有"星月航线"——一小时的夜游，看岸上灯火和岛屿剪影 |
| 货轮 | 中小型。不频繁，但每日可见三两艘进出 |
| 渔船 | 清晨出，午后归。从市中心月心广场就能望见归港的渔船桅杆 |
| 私人船只 | 周末海湾上星星点点的白帆是常态 |


### 夜色

夜幕下的星月湾是这座城市最核心的视觉名片：

- **海面上**：岛屿上的零星灯火、游船的暖色舷灯、灯塔的间歇光束

- **岸线上**：市区沿月牙形展开的万家灯火，从北湾到南湾，勾勒出一道弯月般的暖色弧线

- **山丘上**：丘麓住宅的灯光稀疏错落，像撒在山坡上的碎星

- **倒影中**：所有光都复刻在海面上，风来碎成万点金鳞

> "星月湾"不是形容词。站在灯塔岛回望，你看到的确实是一弯月亮，和满海繁星。

## 旅馆标语

星月湾的每一家旅馆——从北湾老城区的家庭旅馆到市中心的精品酒店——大堂或前台上方都挂着同一句话：

> **"若是繁星还没来，烟火也可映沧海。"**

这句话的来历已不可考。有人说是某个老船长留下的，有人说是最早那批定居者刻在码头木桩上的。没有官方出处，但也没有人想去查证。它就是在那，像海风，像樱树，像每天傍晚灯塔的第一束光。

关于这句话的含义，本地人各有解读：

| 解读 | 版本 |
| - | - |
| 旅人说 | 没赶上好天气看不到星星，港口的灯火倒映在海里的景色也不会让你失望 |
| 老人说 | 年轻时的理想没实现，但你好好过的这一辈子，也不比任何星辰黯淡 |
| 不说的人 | 只是把它刻在旅馆门前的木牌上，等住客自己抬头看见 |


在这个世界观里，这句话还有第三层——高斯泼溅的光点不是真正的恒星，真正的"繁星"从未到来。但这座城市的人早已给出了回答：**没关系。我们有自己的光。**

## 人口与社会

### 人口

| 条目 | 数据 |
| - | - |
| 总人口 | 约 72 万 |
| 人口密度 | 中偏低。城市不追求容积率 |
| 年龄结构 | 均衡。不是老年城市，也不是大学城式的年轻城市 |
| 外来人口 | 有一定比例。因风景和生活节奏迁居至此的人不在少数 |


### 社会氛围

| 条目 | 说明 |
| - | - |
| 总体基调 | **开放、包容、不紧不慢** |
| 对魔力与魔法的态度 | 接受度高。魔法师不是都市传说，是街上偶尔能遇到的职业 |
| 对跨越者的态度 | 行为没有危险性的跨越者完全正常上街生活。星月湾是本地泡内对跨越者接受度最高的城市之一 |
| 对修炼者的态度 | 武者、魔法师、跨越者都只是"生活方式不同的人"——星月湾大致做到了这一点 |
| 邻里关系 | 松散但友好。没人过度关心你，但你需要的时候隔壁会送来一盒多余的草莓 |


> 星月湾不是没有偏见。但在全国范围内，这里的人更倾向于"先看这个人做了什么，再决定怎么看他"。这种态度使星月湾自然成为了跨越者选择定居的城市之一。

### 洁癖派在星月湾

碳基纯洁性运动（"洁癖派"）在星月湾存在，但影响力微弱。他们在市中心偶尔组织小型集会，路人大多绕过，偶尔有人停下争论几句。本地媒体对他们的态度是"记录但不放大"。星月湾的洁癖派更像是一个被宽容存在的异见群体，而非政治力量。

## 阴影面

星月湾不是童话。它有自己不想被游客看见的部分。

### 潮蚀区

北湾老城区最靠近海湾开口的一小段——本地人叫"潮蚀区"——是这座城市最破旧的角落。这里的建筑大半建于数十年前的风暴之后，用料廉价，地基受潮。墙面常年有水渍，梅雨季墙角长霉，海风裹着盐分把木窗框啃得斑驳。

潮蚀区住着渔港的临时工、无证小贩、以及那些"不太想被人找到"的人。白天安静得反常，入夜后巷子里偶尔有压低的争吵声和不知来源的撞击声。北湾其他区域的居民路过这里会不自觉地加快脚步。

> 有意思的是，潮蚀区和樱巷只隔了两条街。春天樱花飘不过去——不是风向问题。

### 废弃矿道

环绕海湾的山丘内部有多条废弃矿道，最早可追溯到星月湾建城之前。多数入口已被封堵，但仍有部分被重新"发现"并投入使用——用途不一。已知的包括：无证修炼者的秘密训练场、走私货物的中转仓、洁癖派极少数激进成员的集会点。也有传闻说某些矿道深处存在地下拳台，观众买票入场，胜负没有规则，打死不算新闻。

市政厅的官方说法是"所有矿道均已封闭，进入即有生命危险"。后半句是真的——矿道结构失修，塌方不罕见，加上某些区域被魔力使用产生的残余能量影响，地质已经不稳定。

### 暗礁社

魔力修行天然筛选出极少数"正常转化者"，但筛选的另一面是：大量"低转者"被挡在真正的修行门外——他们有感应，有渴望，但没有效率。这些人中的一部分聚集在城市的缝隙里，自称"暗礁社"。

暗礁社不是组织，更像是一种身份标签。他们交换粗劣的自主修炼方法，服用未经测试的魔力增强药物，偶尔租用废弃矿道进行"实战测试"。大部分人没什么威胁，只是在自己折腾自己。但每年总有几起事故——魔力反噬、药物中毒、走火入魔——被匆匆送往医院时，病历上总写着"锻炼事故"。

警察知道他们是做什么的，但懒得管——暗礁社不出大事，不偷不抢。真正令当局不安的是另一种可能：某一天，某个暗礁社成员赌对了，真突破了那扇门。那就不是小事了。

### 雾夜

星月湾的海雾在春秋两季是常态，但真正危险的是"浓雾夜"——能见度不足五米，海陆不分。本地渔民航海靠设备和经验，但雾夜里什么事都慢半拍，包括救援。

每年总有几艘小船在浓雾夜没有回来。有些是事故。有些据说是"越过了不该越过的界限"——老航行者偶尔会压低声音提起，雾最浓的时候，海湾开口外的水面偶尔能看到不属于任何已知船只的灯光。

> 海警的记录显示近十年失踪船只数量在正常范围内。但那些航行者说的是另一回事。

### 岛屿上的人

海湾内的离岸礁和少量小型岛屿上并非完全无人。有些岛屿上住着独居者——懒得和任何人打交道的人，或是不能和人打交道的人。他们没有在市政厅登记，靠小船往返，有时候连着几个月不露面。偶尔有登山者在丘麓步道上用望远镜看到远处某座礁岛上有炊烟，但不会有人去确认。

灯塔岛的塔灯下方，值班日志上偶尔会出现不是值班员写的记录。笔迹不一，内容琐碎——"今晚浪高""风向偏东""鱼群过了"。无人追究。星月湾人对于"古怪但不能证实的事"有一种默契：不去查，不会少块肉。

## 地标与场景

| 地点 | 位置 | 说明 |
| - | - | - |
| **灯塔** | 海湾开口处灯塔岛 | 城市标志。已自动化，底部设小型观光平台。黄昏时分的灯塔剪影是星月湾被拍摄最多的画面 |
| **月心广场** | 市中心 | 圆形广场，中央有一座浅水池。冬日放干后用作滑冰场。四周是市政厅、图书馆和两座剧院 |
| **樱巷** | 北湾老城区 | 石板路，樱花廊道。两侧为咖啡馆、旧书店、手作工坊。不通车 |
| **海鲜早市** | 北湾渔港旁 | 每日清晨开市，中午收摊。最新鲜的渔获和最地道的本地生活 |
| **南栈桥** | 南湾突出部 | 一座伸入海湾约两百米的木栈桥，尽头无建筑，只有一张长椅。公认的城市最佳发呆地点 |
| **丘陵步道** | 丘麓沿线 | 沿山丘蜿蜒的步行与骑行道。沿途可俯瞰海湾全貌 |
| **星月旅馆** | 北湾老城区 | 星月湾最早的旅馆之一。据说"若是繁星还没来"这句话最早就是挂在这里的大堂 |
| **HBI** | 南湾-丘麓交界 | 星月湾泛生命科技学院。半开放校园，三大学部：海洋科学/计算机科学/生命科技 |
| **研究园区** | 丘麓步道中段 | 独立产学研集群。三个方向：魔力储能/魔力材料/低转者转化效率。与HBI人员交叉 |
| **人民医院** | 北湾老城区东缘 | 市级综合医院。设有魔力损伤科、跨越者门诊、心智评估与沉渊科、中医科 |


## 交通

| 方式 | 说明 |
| - | - |
| 市内 | 公交为主，线路覆盖全面但班次不密集（密度低，需求不大）。自行车道贯穿全城 |
| 市外 | 公路连通邻近城市。有一座小型机场，航班有限 |
| 海上 | 游船、渡轮和私人船只。无大型客运港 |
| 市内特色 | 北湾老城区部分路段为石板路，仅限步行和自行车 |


> 星月湾的节奏不需要地铁。公交车上不会挤满人，自行车篮里放着刚买的菜和花，你骑过樱巷时花瓣刚好落在车篮里——这大概就是本地人理解的"通勤幸福感"。
