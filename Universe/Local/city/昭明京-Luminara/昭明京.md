# 昭明京

## 概述

承明大陆首都，联合国总部所在地。人口约1520万。600余年城市史，80年前成为联合国首都。面积约2538平方千米（市区1138平方千米，郊区1400多平方千米）。

## 地理

位于承明大陆中部偏东，山脉边缘。依山而建，地势较高，受中央山脉气候影响。

## 城市格局

昭明京采用"古都中心—古都外—郊区"三层结构。

### 古都中心

约占城市一成五面积。80年前联合国选定昭明京为总部后，对古都进行了重建改造，建筑风格保持原貌。

**联合国总部行政集群**占据古都七成地带，包括
- 皇城/总部大殿：古都中心，占地2.56平方千米，约占古都总面积1.5%。高耸城墙隔绝，平民不可进入。从古至今保留的皇城，现为联合国总部大殿
- 其他联合国办公区
- 世界级组织驻地：卫生组织、机械标准组织等

非联合国办公区为旅游区，夜晚灯火通明，歌舞升平。

### 古都外

约占城市八成半面积，包括普通市区和新区。

普通市区约占六成半大部分为老公寓、30余层高层住宅，无数各种规模的商城，还有许多小型企业的安身之所

*新区约占两成，不连续分布，多个区块夹杂在普通市区之间，区内高楼林立，写字楼冷白LED光与楼底路灯和商贩的暖色灯光交融，住宅建筑为房价不小于10万的豪华公寓

### 郊区

约1400多平方千米，城市外围区域。

## 交通

### 道路

主干道路除古都外呈环形分布，共5环。每圈环之间有直通道路，呈漩涡状，两个相反旋向叠加，满足双向换环需求。

### 公共交通

| 类型 | 距离 | 说明 |
|------|------|------|
| 地铁 | 远距离 | 线路很多，主流远距离公共交通 |
| 有轨电车/公交 | 中距离 | 地面公共交通 |
| 共享单车 | 近距离 | |
| 出租车 | 中/近距离 | |

### 铁路

市区内45座火车站，其中6座为昭明京市政府直属：
- 邵明中站
- 邵明东站
- 邵明南站
- 邵明西站
- 邵明北站
- 承天门站

高铁不太普及，线路不多，多为普通绿皮车线路，但速度不算慢，速度最高约210km/h。

### 航空

郊区有两座机场：
- 邵明长宁国际机场
- 邵明承天国际机场

## 政治态度

### 官方立场

对跨越者必须保持中立。跨越者拥有人格尊严权的法律在昭明京制定，歧视跨越者就是打自己的脸。作为政治中心，官方不会公开支持或反对任何一方。

他们不认为星月湾、满月城一类对跨越者开放的城市有问题。这些城市的政策在联合国框架下属于地方自治范畴。

### 民间态度

各个流派看法不同。首都人口多元，既有保守派也有开放派，但作为政治中心，极端言论较少公开表达。
