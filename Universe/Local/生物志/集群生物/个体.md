
# 个体

> 它什么都没有。没有思考，没有记忆，没有"自己"。它只是一个会动的小东西，本能地寻找另一只小东西，触碰到的那一刻，送出微弱的一闪。

## 形态结构

*Neurosocius* 的单体形态因变种而异，但遵循共同的结构原则：极简。

核心组成：
- 一层柔性外膜，允许电信号沿表面传播
- 一套原始神经节，产生自发脉冲并接收外膜传递的信号
- 一套基态运动系统（纤毛或气体囊调节）
- 接触面分化：体表存在若干信号密集区——物理接触发生时，信号优先从这些区域发出

个体不含消化系统、不含循环系统、不含感觉器官——它不"感觉"世界。它只接收同类的电信号。

## 电信号机制

这是 *Neurosocius* 最核心的生物学特征。

| 参数 | 说明 |
|------|------|
| 信号类型 | 电脉冲，类神经动作电位但更简单 |
| 传播介质 | 必须物理接触。个体间外膜贴合时，信号跨膜传播 |
| 传播速度 | 约 100,000 km/s（光速的三分之一） |
| 信号内容 | 极简。没有符号编码能力——单个信号仅携带"兴奋/抑制"级信息 |
| 衰减 | 信号在个体间每经一次跨膜传播，强度下降。个体本身可作为中继放大 |

在电学意义上，一个集群是一个巨型传导网络：每个个体是网络中的一个节点兼放大器。信号可以穿越数百个个体而不完全消散，前提是路径上的个体密度足够高。

这意味着：
- 集群的"脑电波"就是它的形态。个体排列的拓扑结构直接决定信号流向。
- 当集群分散成若干孤立子群时，子群之间的通信立即中断。
- 集群不存在"无线"通信——不接触，即不通。

## 本能行为谱

个体仅具备固定行为集，无学习能力：

| 本能 | 说明 |
|------|------|
| 接触趋向 | 向邻近同类发出的电信号方向移动。这是集群凝聚的核心驱动力 |
| 排斥反应 | 对过高频/过强电信号短距离回避，防止信号过载 |
| 趋光/避光 | 依变种和个体状态而异 |
| 洋能感知（仅变体） | 对环境中洋能场产生趋向或回避 |
| 分裂繁殖 | 个体体积超过阈值后自发分裂为二 |

不存在觅食行为——个体直接从环境介质中吸收溶解态有机物（潮髓）或大气微粒（飑髓）维持代谢，能耗极低。

## 寿命与繁殖

个体自然寿命约 30-60 天。衰老个体的电信号频率逐步降低，最终停止脉冲并被集群排斥。

繁殖为无性分裂。分裂后的两个子体初始体型相同，但信号特征可在此后的环境适应中产生微妙分化——这是集群功能分化的基础。

个体的"死亡"对集群不构成影响，正如单个神经元凋亡不影响人类思考。

## 洋能变体

绝大多数 *Neurosocius* 个体不含可感知的洋能，它们对魔力场无任何反应。

但在某些集群中，会出现一种极为特殊的个体：它的外膜蛋白结构发生变异，允许微弱地吸附和释放环境洋能。这种个体本身不做任何思考——它只是多了一种感知通道。

- 变体的数量占集群总体的比例未知
- 这个比例直接决定了集群整体对洋能场的感知程度——变体越多，集群越能"看到"魔力
- 对洋能的操控能力则是另一回事，与感知不完全相关

变体的成因不明。已知环境洋能浓度偏高时，新生个体的变异率似乎有所上升，但相关性尚未被系统验证。

> 假设一个千万级集群中含有一万个洋能变体——它对洋能场的感知便能达到中等水平跨越者的程度。但这个集群仍然不能操控洋能——它只能感知，不能运用。
