
# 潮髓 —— *Neurosocius maritimus*

> 远洋深处的暗流中，它舒展成一张数公里宽的无形之网。渔船的声呐照过它的身体时，它正思考着某件无人知晓的事。

## 栖息环境

潮髓分布于开阔大洋中层至深层水域，深度通常在 200m - 1500m 之间。偏好低温、贫营养的稳定水体，避开大陆架和高生物量沿岸海域——自然避开了人类渔业活动密集区。

## 个体形态

| 特征 | 说明 |
|------|------|
| 体型 | 半透明胶质椭球体，长约 3-5cm，宽约 1-2cm |
| 触须 | 6-12 根，可延伸至 15-25cm（体长的 5-8 倍）。丝状，极细（约 0.05mm），水中几乎不可见 |
| 比重 | 略高于海水。借助体表纤毛的微弱摆动能维持深度 |
| 外膜 | 约 1μm 厚，含信号密集区（接触斑）。膜表面分布离子通道，承担跨膜信号传递 |
| 代谢 | 膜表面直接吸收溶解态有机物。能耗极低，单体静息状态仅需维持离子梯度 |

潮髓个体的形态分化程度有限——“信号平稳型”个体寿命稍长、分裂更慢，易成为记忆区的组成单元；“信号活跃型”个体反应更快，常见于集群边缘感受区。

## 集群形态

在海水中，潮髓集群呈现为三维立体网络。个体间通过触须交织连接——触须伸展、触碰、缠绕同伴的触须或体表，编织成由无数微细丝线构成的活网。这张网在水中近乎透明，只有近距离才能被肉眼察觉：仿佛一团悬浮的、微微闪烁的蛛丝。

水下没有遮蔽，集群暴露于洋流中。为了对抗水流的撕裂力，集群需要维持一定的结构张力——这由个体间的贴合强度和快速重新连接的能力共同保证。强流冲击下，集群可能被撕成多个子群；水流平缓后，子群会在数小时内重新聚合。

一个超过千万个体的潮髓集群，在海中的信号最远传输距离可达数十公里——集群边缘的"手"触碰到的信息，到达核心的延迟约几百微秒。

## 已知记录

海渊城海洋大学拥有最多的潮髓野外观测数据，但大部分处于未发表状态。已知的记录显示：

- 某次远洋航次中，声呐探测到一个跨度约 6km 的悬浮物体，没有实体成像。事后比对的潮髓样本暗示可能是一个中等规模的集群
- 数个不同海域的潮髓样本在水槽中接触后，信号交换速率远超预期——暗示这些样本可能来自同一个大集群的碎片
- 星月湾东南海域偶有零星个体出现，但从未形成可观测的集群——洋流方向和陆架地形不利于此

## 天敌与威胁

已知天敌包括某些滤食性鲸类和大型浮游摄食鱼类。对潮髓而言，被吃掉几万个个体不是致命伤害——但对集群智力而言，相当于丢失了几个皮层区域，需要时间重新调整功能分配。

更致命的威胁是地震引发的浊流和海底滑坡——大规模的泥沙羽流可以在数十分钟内掩埋整个集群。
