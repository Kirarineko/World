
# 飑髓 —— *Neurosocius aerius*

> 雷暴云上方，极光带下方。它翻卷在平流层底部的无声风中，用了十二年来想明白一朵云的形状。

## 栖息环境

飑髓分布于高空大气层，常见于对流层上层至平流层下部（海拔 10km-25km）。偏好稳定气团，避开强对流区域。大部分集群位于航运航线和人类活动范围之上——地面上的人永远不知道头顶几十公里处有一片正在思考的活云。

## 个体形态

| 特征 | 说明 |
|------|------|
| 体型 | 极轻囊状体，直径约 1-2cm |
| 浮力 | 内含微量氢气的半透膜囊。比重轻于空气，不耗能即能悬浮 |
| 外膜 | 与潮髓同源但更薄，约 0.3μm。接触斑分布更稀疏，因为气流中个体间距天然大于水中 |
| 代谢 | 膜表面吸附大气微粒和少量有机气溶胶。极低能耗 |

飑髓个体的浮力囊不自主调节——个体无法主动上升或下降。它们在气流中被动漂流，仅靠纤毛微调方向以趋近同伴。

这带来一个独特的问题：气流中个体间距远大于水中，物理接触的频率天然更低。飑髓需要更高密度才能维持与潮髓同等的连通度。

## 集群形态

在大气中，飑髓集群呈现为三维薄雾状——肉眼看来像一片淡色的高云，在气象学上却不符合任何已知云类。

气流对集群形态的影响比洋流对潮髓的影响更为剧烈：
- 风切变可以将集群横向撕成多层
- 上升/下沉气流可以将集群垂直拉伸数千公里
- 一台积雨云的过境可以在十分钟内将一个千万级集群冲散成数百个碎片

飑髓因此发展出比潮髓更强的重聚能力。它们的个体向同伴信号方向的趋近更积极，重聚速度更快——代价是重聚期间的信号噪声显著更高。一个刚被雷暴撕碎的飑髓集群在重新聚合的最初几小时里，思维是断断续续的、充满杂音的——就像一个在爆炸声浪中刚恢复听力的人。

## 气象敏感

飑髓对天气系统极度敏感。千万级集群会主动回避强对流区——它们的感知网络可以在数百公里外检测到气压场的微妙变化，并在数日内完成整体偏移。

这种偏移是缓慢的——数公里的集群以步行速度移动。它不能"逃离"风暴，只能"躲开"。

> 天文学家曾观察到，在某些流星雨极大夜，飑髓集群会下降高度——从平流层沉入对流层上层。原因不明。也许它们在看。

## 已知记录

可靠的飑髓观测记录远少于潮髓。高空气象气球偶尔采集到零星个体，但从未捕获能揭示集群全貌的样本。

海渊城海洋大学——尽管名为"海洋大学"——曾在一项大气-海洋耦合研究中意外记录到高空存在一个疑似飑髓集群。雷达回波显示为一层厚度不足 50m、水平跨度约 4km 的弱反射面。该数据后来被归入"不明大气现象"档案，未有后续跟进。

星月湾上空未见飑髓集群的确凿记录。但湾区的老渔民中流传着一个说法：某些夏夜的云层中会出现不规律的发丝状电光，不是闪电，也不打雷。还没有人把这和飑髓联系起来。
