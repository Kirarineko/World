
# 镜像机制

> 每个人脑中都有一个"最初的自己"——只读快照。每一次成长、愈合、衰老，都在上面叠加一层写时复制。只要原始镜像还在，"我"就还在。

## 原始镜像

人类大脑中保存着一份"生命镜像"——对全身组织的完整只读快照。这份镜像定义了"完整的自己"：骨骼的排列、器官的结构、神经回路的初始拓扑。它不是基因——基因是设计图，镜像是竣工时的验收照片。

镜像以块为单位映射全身。每个块对应某个特定组织或器官区域——比如一块骨骼的特定节段、一块心肌的特定分区。

镜像块是只读的——只要还有一个引用指向它。

## 写时复制（CoW）

身体的每一次自然生长、受伤后的疤痕形成、肌肉锻炼的痕迹——每一次改变都生成一个新的"差分层"。差分层不覆写原始块：它写入新的数据块，然后将对应的虚拟地址重新映射到新块上。

原始块的引用计数减一。但不会归零——除非所有引用它的差分层都已经被更上层的 CoW 重定向替代。

```
原始镜像块 ──→ 引用计数=3
   ├── 差分层A（童年生长）  ──→ 引用该块
   ├── 差分层B（骨折愈合）  ──→ 引用该块
   └── 差分层C（肌肉锻炼）  ──→ 引用该块

当差分层C 的更高层 CoW 替代了它对原始块的引用：
  原始镜像块 ──→ 引用计数=2
  差分层C 不再指向原始块
```

## 块释放与镜像的变动

当一个数据块的所有引用都已移走（被更高层的 CoW 重定向到新块），该块的引用计数归零。块被释放。释放后的空间可以被新数据覆写。

这意味着原始镜像不是永恒不变的。足够的生长、伤愈、改变——足够多的差分层叠加——会逐渐释放原始镜像中的块。一个老人的原始镜像并非完全消失，但可能有相当比例的块已被释放并覆写。这就是衰老——不是镜像损坏，是镜像的逐步归零。

但这种释放在正常情况下是极其缓慢的。人类的引用计数管理粗糙，很多差分层会保留不必要的旧引用——这恰好防止了原始块的过快释放。

> 镜像的保护机制不是锁。是懒得删。

## 治疗魔法 = fsck

镜像可以作为一致性检查的基准——治疗魔法本质上就是一次 `fsck`。

以镜像中对应块的校验和为基准，修复物理损伤：骨折对齐、组织愈合、血管重接。`fsck` 能修复的，一定是"曾经完好的状态"被破坏后的结果。

它不能修复的东西也正好对应：
- **先天残疾 / 遗传病**：镜像本身就有这个"错误"——不是损坏，是源数据如此。`fsck` 跑一遍，校验通过——那个"错误"就是正确的
- **病毒感染**：外来代码不在镜像中。`fsck` 不识别为损坏，不予清除

> `fsck` 不是万能医生。它是校对员——只纠对在源数据基础上发生的偏移。

## 跨越者：洋能覆写原始镜像

当一个人的原始镜像中有块已经引用计数归零——即所有正常的差分层都不再需要这个块——该块的空间处于"已释放，待覆写"状态。

如果此时这个人长期接触洋能，身体中的魔力回路和洋能代谢生成了一组新的差分层——这些差分层依赖洋能来维持自身。当它们恰好写入了那些已释放的原始块位置，那块数据就被洋能覆写了。

从那一刻起，这个人是一个跨越者。

不是因为吸收了多少魔力。是因为原始镜像的某个部分不再由镜像自身保护——而是由洋能"撑着"。一旦洋能撤走或耗尽，那些位置的数据立即失效——镜像出现缺失。缺失的镜像无法通过 `fsck` 修复——因为源数据已经没了。这不是损伤，是消失。

> 跨越 = 底层引退，洋能入主。

## 沉渊

沉渊不是心智"承受不住"魔力。是更底层的结构性崩溃。

当跨越者的洋能供应被切断（魔力耗尽、环境洋能骤降、被外力强制驱散），那些依赖洋能维持的镜像层大面积失效。镜像出现系统性缺失——身体还活着，但"自己"的定义框架塌了。

余下的半残镜像可能仍能维持基本的生命活动和反射行为——就像只剩根分区的操作系统还能亮灯、风扇还在转。但"人"已经不在了。

这就是为什么沉渊后的个体无法被治疗魔法修复。`fsck` 需要一个源数据校验基准。镜像本身是那个基准。基准碎了。

> 在洋能被命名之初（约60年前），人们对沉渊的理解只有"出事了"和"不知道怎么办"。这恰好对应：他们不知道镜像的存在，更不知道镜像可以被覆写。

## 脑死亡 = 镜像死亡

在法律和医学定义上，人的死亡仍然是脑死亡——大脑不可逆地丧失全部功能。这与镜像机制完美一致：镜像存在于大脑中。大脑的物理死亡意味着镜像存储的物理介质消失。

这解释了为什么身体器官可以移植但人不可以——移植的器官是 CoW 层覆盖的硬件，但镜像寄存在大脑里，不能搬家。

## 人类的 Btrfs 功能边界

人类的镜像系统只有以下功能：

| 功能 | 说明 |
|------|------|
| 原始镜像 | 大脑中的一次性只读快照，定义"完整的自己" |
| CoW 差分层 | 每次改变生成新层，不覆写原始块 |
| 引用计数 | 每个底层块被多少上层引用计数，归零后释放 |
| 洋能覆写（跨越者） | 已释放块被洋能依赖层占据 → 失去洋能即镜像缺失 |
| fsck（治疗魔法） | 以镜像为基准修复物理损伤 |

**人类没有的功能：**
- 不能主动创建快照——原始镜像是出生/成胎时一次性生成的
- 不能主动回滚——无法"回到某个历史状态"
- 不能读取镜像内容——无法"看到自己完整的样子"。镜像是一个无形的校验基准，不是一本可翻阅的书
- 不能外部备份——镜像只在自身大脑中

## 洋能生物

对于洋能生物（亚空间原生或泡宇宙中的纯洋能物种），镜像系统的完整度因物种而异。没有"统一的洋能生物 Btrfs 标准"——每一种生物在进化或形成过程中发展出了自己的一套。

有些洋能生物具有人类不具备的高级特性——比如主动快照管理、部分回滚能力、甚至在主体死亡后以镜像为基础重建身体的可能性（取决于该物种的等级和生理构造）。但这些都是特例，不能泛化。

人类的 Btrfs 版本就是它唯一的版本。残缺。但足够跑一个文明了。

## 智慧断崖

智慧断崖——智力在"猫"附近存在一道无法跨越的断层——现在有了一个来自镜像机制的底层解释。

运行镜像 Btrfs 是有计算成本的。引用计数的维护、CoW 差分的映射、块释放和清理——这些需要足够强的神经基础来支撑。就像一台没有操作系统的单片机可以正常工作，但你不能在上面跑一个文件系统。

大多数生物的神经系统 ≈ 单片机或 MCU——有反射、有本能、有简单学习，但没有足够资源维持一个完整的镜像文件系统。

猫恰好卡在门槛附近。猫同时具有大量脊髓反射（不需要镜像参与）和一个还算聪明的大脑。它的大脑能跑一部分镜像功能——但不够完整。这就是为什么猫聪明但跨不过智慧断崖：它的镜像系统足够让它认识主人、记住习惯、产生偏好，但不足以支撑自我意识的涌现和累积。

能跑完整 Btrfs 镜像的物种，才具备发展出自我意识、累积文明的理论前提。在整个泡宇宙中，神经系统硬件能达到这个门槛的物种屈指可数。这就是智慧断崖的底层原因。

> 智慧不是某种神圣的火。是物理。是神经基础够不够跑文件系统。
