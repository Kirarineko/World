# 武力体系 · 金玄归天


> **适用范围**：仅本地泡。这是本泡人类在漫长历史中自主发展出的肉身修炼体系：在不知晓其他泡存在、不知道洋能真正来源的情况下，独立建立的本土框架。其底层能量（魔力）本质上是亚空间洋能，与跨泡通用的能层能级体系（K-Q 能层 + s-i 能级）共享同一能量本源。金玄归天是本地文明对洋能的"地方化表达"，不是跨泡通用标准。在极少数场合（如与非本泡出身的洋能使用者交流），请使用能层能级体系对标，而不是报"归一武者"。

> 本地泡的肉身修炼体系。与魔法体系相辅相成，共享同一能量本源。


## 体系概览



武力修炼以肉身为根基，以魔力为进阶之梯。四境十二阶，倒序计数：数字越小，境界越高。


```
金三 → 金二 → 金一   纯肉身
玄三 → 玄二 → 玄一   魔力介入
归三 → 归二 → 归一   魔力融合
天三 → 天二 → 天一   人类极限
```


> 金三起步，天一封顶。每一境的"一"是该境圆满，"三"是该境入门。


## 四境



### 金境：锻体如金


| 阶位 | 说明                                                 |
| -- | -------------------------------------------------- |
| 金三 | 入门。肉身开始系统锤炼，体能超越普通人的极限门槛                           |
| 金二 | 筋骨坚实，五感锐化。徒手击碎金石，短程爆发力可追猎豹                         |
| 金一 | 纯肉身顶峰。不依赖任何外部能量，仅凭身体达到的极限状态。此后，肉身再无寸进：剩下的路，需要别的东西 |


> 金境不需要魔力，也感知不到魔力。大多数武者终其一生留在金境。


### 玄境：入玄窥妙


| 阶位 | 说明                                                                        |
| -- | ------------------------------------------------------------------------- |
| 玄三 | 魔力觉醒期。第一次感知到环境中魔力的存在，尝试吸收。此时魔力尚浅，仍可凭肉身底蕴推进                                |
| 玄二 | **必须魔力支撑。** 肉身已越过纯物理的极限线，没有魔力灌注便无法维持，遑论精进。这是武者的第一道滤网：大量金一武者卡在此处，终生无法踏入玄二 |
| 玄一 | 魔力与肉身初成循环。攻防开始附带魔力属性，速度与力量不再受纯物理约束。此时武者开始面对魔力的心智影响                        |


### 归境：万法归宗


| 阶位 | 说明                                                              |
| -- | --------------------------------------------------------------- |
| 归三 | 魔力与肉身深度融合。不再是"用魔力强化身体"，而是"身体开始以魔力为组成成分"。突破归三需经历**归渊**：武者的心理临界点 |
| 归二 | 魔力循环自成体系，可在战斗中持续自我补充。身体的魔力密度达到峰值                                |
| 归一 | 归境圆满。肉身与魔力的界限模糊到几乎不可区分。此时每一拳都是物理打击也是魔力释放                        |


> 归境武者极为稀少。踏入归境本身需要魔力转化效率达到正常水平：仅此一条就筛掉了绝大多数人。


### 天境：天人之境


| 阶位 | 说明                                                         |
| -- | ---------------------------------------------------------- |
| 天三 | 人类范畴内肉身与魔力的终极平衡。举手投足间魔力和肌肉不再有先后，是同步的、同一的                   |
| 天二 | 开始触及"人类"这个概念的边缘。魔力占比已超过肉身，但生命形式仍维持碳基。此时武者必须刻意维持自己的人性，不是比喻 |
| 天一 | 人类武学的理论终点。再往前一步（魔力占比再高一点、生命形式再偏离一点）就不再是人类                |


> **天境的魔力门槛**：踏入天境，魔力修为至少需要达到 **烈·贯（M·3p）**。这是天境区别于归境的硬性分界，不是魔力总量够多就行，而是必须掌握轴向定向的贯穿形态，将魔力与武学攻击融为一体。达不到烈·贯的归境武者，无论肉身多强，天境的门不会开。
>
> 天境武者带着常人看不见的疤。每一个天一都是和他自己搏斗的幸存者。

## 魔力普及度



魔力存在于环境中：来自亚空间的洋能渗透，但能否吸收、吸收后能转化多少，因人而异。


| 群体  | 大致比例    | 特征                                      |
| --- | ------- | --------------------------------------- |
| 无感者 | 绝大多数    | 魔力就在身边，但身体无法吸收。如同不会呼吸空气的鱼               |
| 低转者 | 有感者中的多数 | 能吸收，但转化效率极低。修炼速度以十年为单位。终其一生，魔力积累不足以支撑玄二 |
| 正常者 | 有感者中的少数 | 转化效率足以支撑真正的魔力修行。魔法师、玄境以上武者，都来自这个池子      |


### 社会后果


* 金境武者数量可观（纯肉身修炼不依赖魔力，门槛相对低


* 玄二以上武者断崖式下降）卡在魔力这道天然的生物筛选上



* 魔法师的绝对数量比玄境武者更少（纯魔法路线对转化效率的要求更高


* 魔力知识不普及）大多数人既无需求也无接触途径


## 魔力量化感知


> 一般人对魔力只有模糊的直觉，无法量化感知。


魔法学术界使用的能层能级体系（微、明、烈、炽……浑、贯、散、构……）是一套精密的理论框架，但它是学者在实验室里构建的产物。在实际生活中：


| 群体      | 感知能力                                                        |
| ------- | ----------------------------------------------------------- |
| 普通人     | 顶多感到"这个人很强""那个地方有点不对劲"。说不出强在哪、有多强。就像普通人能感到冷热，但说不出具体几度       |
| 低阶武者/法师 | 能模糊分辨魔力的大致强弱，但无法精确定量。判断"这人魔力比我多"靠的是经验直觉，不是读数                |
| 高阶修行者   | 感知更敏锐，但仍以经验和直觉为主。能分辨对方魔力的大致层次，但对精细能级的判断需要专门训练：且即便训练过，也可能出错 |
| 学术检测    | 唯一能精确量化魔力的手段是专业设备和标准化测试。但这套东西只在少数机构中存在                      |



> 两个武者在街头对峙，谁也看不清对方的准确修为。他们判断对方能不能打，靠的是气场、肌肉、站姿、眼神：以及动手之后的第一秒。这和动物判断对手的方式没有本质区别。


这意味着：


* 能层能级是学术分类，不是战斗读数。没人头顶飘着等级标签


* 魔力强弱可以被模糊感知，但不能被精确判断。示弱和示强都是可行的战术


* 关于某人的"真实水平"，永远存在争议空间


## 与魔法体系的关系


### 相辅相成



武力与魔法不是两条互斥的路。同一个人可以同时修炼：前提是转化效率达标。


| 关系   | 说明                                             |
| ---- | ---------------------------------------------- |
| 能量同源 | 武力和魔法使用的魔力，本质上是同一种东西：来自亚空间的洋能                 |
| 路径不同 | 武力以肉身为容器和引擎，魔法以精神/意识为框架和管道                     |
| 互相促进 | 武力提升扩大魔力容量，魔法提升提高魔力精度。两者不是加法，是乘法，但前提是时间精力允许双修 |
| 代价共享 | 魔力影响心智，无论是武者还是法师都要面对。武者额外承担归渊，法师额外承担能级突破时的意识冲击 |


### 双系统参考映射


```
武力              魔法能层      备注
─────────────────────────────────────
金三~金一          未觉醒
玄三~玄一          微 ~ 明
归三~归一          明 ~ 烈
天三~天一          ≥ 3s(烈·散)    ← 天境硬门槛
```


> 映射不严格。一个归一武者魔力总量可能达到烈层，但在能级形态上几乎为空白：武者不练浑/贯/散/构的形态分化，那是魔法师的领域。武者把魔力灌进拳头，魔法师把魔力编织成结构。
>
> 但天境例外：天境要求武者必须掌握贯态，不是为了施法，而是为了让魔力沿着攻击轴线定向爆发。这是武学对魔力形态提出的唯一硬性要求。


## 魔力对心智的影响


魔力影响心智。这不是道德命题，不是"力量使人堕落"，而是魔力作为一种外来能量的生物心理学副作用。



| 表现   | 说明                                                                                                                                |
| ---- | --------------------------------------------------------------------------------------------------------------------------------- |
| 执念放大 | 修炼者的性格特质被魔力持续强化。坚毅渐变偏执，谨慎渐变多疑，热爱渐变痴迷。不是变成了另一个人，是原来的自己变得更极端                                                                        |
| 共情偏移 | 对非修炼者的情绪共鸣持续减弱。不是冷酷，是"感受不到"。高阶武者需要刻意维持与普通人的情感连接：一个天一可能对家人的爱完全是意识层面的选择，因为他已经感觉不到那种爱了                                              |
| 魔力回声 | 停止使用魔力时，意识深处残留某种"回响"。不是幻听，更像耳鸣：一段不属于自己的感知残余，在寂静时浮现。魔力使用越频繁、强度越高，回声越清晰                                                            |
| 归渊   | 突破归三时必须经历的心理临界。魔力占比越过某个阈值，意识的基底结构开始重组。在此期间修炼者面对自身所有被放大的执念、所有被削弱的共情、所有累积的回声：同时面对。撑过去，踏入归境。撑不过去，心智不可逆受损。归渊的失败者被称为"沉渊者"：身体活着，人已不完整 |


### 天境武者的心理画像


天境武者不是疯子。但他们每个人都在和某种常人不理解的东西共存。


* 有些天境选择沉默寡言，不是冷漠，是怕说出的话已不再来自"自己"


* 有些天境刻意维持琐碎的日常习惯（每天逛菜市场、养猫、种花）用普通人的仪式对抗共情偏移



* 有些天境在归渊之后性格发生了可见的改变，不是变坏了，是某些东西被留在了渊里


> 普通人畏惧天境武者，让他们不安的不是一拳碎楼的力量，而是直视他们的眼睛时，总觉得镜子里的自己在发颤。

## 人类的定义


在本地泡，是否为"人类"由两个条件判定：


| 条件   | 说明              |
| ---- | --------------- |
| 碳基生命 | 生命形式仍为碳基。这是硬边界  |
| 肉体完整 | 物理身体未发生不可逆的非人转化 |


> 天一的魔力占比极高，但他依然是碳基生物，依然是人类。跨过炽→虚的魔法师魔力总量也许不如天一，但生命形式已经改变：不再是碳基，不再被认为是人类。


跨越者与天境武者的本质区别不在力量大小，在**身体的基础化学**。这也是人类洁癖派的底层逻辑：他们捍卫的不是某种文化身份，是一条生物边界。
