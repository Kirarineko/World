# 武力体系 · 金玄归天

> **适用范围**：仅本地泡。这套体系是本泡人类在漫长历史里自己摸出来的肉身修炼路数。那时的人并不知道别的泡，也不知道洋能真正从哪里来，只能按本地经验把它整理成一套能练、能传、能比较的框架。它底层用的还是亚空间洋能，和跨泡通用的能层能级体系共用同一种能量本源。金玄归天是本地文明对洋能的本地叫法，不是跨泡通用标准。和别的泡出身的人谈的时候，还是拿能层能级对标更清楚，别直接报“归一武者”。

> 本地泡的肉身修炼体系。与魔法体系相辅相成，共享同一能量本源。

## 体系概览

武力修炼先落在肉身上，再一点点借魔力往上抬。四境十二阶，倒着数，数字越小，境界越高。

```
金三 → 金二 → 金一   纯肉身
玄三 → 玄二 → 玄一   魔力介入
归三 → 归二 → 归一   魔力融合
天三 → 天二 → 天一   人类极限
```
> 金三起步，天一封顶。每一境的“一”是圆满，“三”是入门。

## 四境

### 金境：锻体如金

| 阶位 | 说明 |
| -- | -- |
| 金三 | 入门。肉身开始系统锤炼，体能超越普通人的极限门槛 |
| 金二 | 筋骨坚实，五感锐化。徒手击碎金石，短程爆发力可及老虎 |
| 金一 | 不依赖任何外部能量，仅凭身体达到的极限状态。 |

### 玄境：入玄窥妙

| 阶位 | 说明 |
| -- | -- |
| 玄三 | 魔力觉醒期。此时魔力尚浅，仍可凭肉身底蕴推进 |
| 玄二 | **必须魔力支撑。** 肉身已越过纯物理的极限线，没有魔力灌注便无法维持，遑论精进。这是武者的第一道滤网：大量金一武者卡在此处，终生无法踏入玄二 |
| 玄一 | 魔力与肉身初成循环。攻防开始附带魔力属性，速度与力量不再受纯物理约束。此时武者开始面对魔力带来的心智负担 |

### 归境：万法归宗

| 阶位 | 说明 |
| -- | -- |
| 归三 | 魔力与肉身深度融合。 |
| 归二 | 魔力循环自成体系。身体的魔力密度达到峰值 |
| 归一 | 归境圆满。肉身与魔力的界限模糊到几乎不可区分。此时每一拳都是物理打击也是魔力释放 |

### 天境：天人之境

| 阶位 | 说明                                                         |
| -- | -- |
| 天三 | 人类范畴内肉身与魔力的终极平衡。举手投足间魔力和肌肉不再有先后，是同步的、同一的 |
| 天二 | 开始触及碳基人类的边缘。极易在不经意间成为跨越者 |
| 天一 | 人类武学的理论终点。 |

> 碳基基底的人类对魔力的利用率较低，相对亚空间的洋能生物的平均水平只有七八成，一旦社会上的偏见解除，六成武者会选择成为跨越者

## 魔力普及度

魔力就是亚空间渗进来的洋能。只是能不能吸收、吸收了能不能用得看个人。

* 金境武者数量可观纯肉身修炼不依赖魔力，门槛相对低

* 玄二以上武者断崖式下降，卡在魔力这道天然的生物筛选上

* 魔法师的绝对数量比玄境武者，更少纯魔法路线对转化效率的要求更高

* 魔力知识不普及，大多数人既无需求也无接触途径

## 魔力量化感知

一般人对魔力只有模糊的直觉。

魔法学术界用的能层能级体系（微、明、烈、炽 · 浑、贯、散、构……）是学者在实验室里慢慢搭出来的。在日常里，它没那么直白

普通人顶多感到"这个人很强""那个地方有点不对劲"。说不出强在哪、有多强。这个感觉就像就像普通人能感到冷热，但说不出具体几度

低阶武者/法师能模糊分辨魔力的大致强弱，但无法精确定量。判断"这人魔力比我多"靠的是经验直觉。

* 魔力强弱可以被模糊感知，但不能被精确判断。示弱和示强都是可行的战术

* 关于某人的"真实水平"，永远存在争议空间

## 与魔法体系的关系

### 相辅相成

武力和魔法不是两条互相排斥的路。同一个人可以同时修。

### 双系统参考映射

```
武力              魔法能层      备注
─────────────────────────────────────
金三~金一          未觉醒
玄三~玄一          微 ~ 明
归三~归一          明 ~ 烈
天三~天一          ≥ 3p(烈·贯) 
```

> 映射不严格。一个归一武者魔力总量可能达到烈层，但在能级形态上几乎为空白。

## 魔力对心智的影响

魔力会影响心智。它不是“力量使人堕落”那套说法，外来能量本身带来的生理和心理副作用。

> 详见 `洋能.md`。洋能被编码时，使用者的情绪、记忆、感官快照等元数据会自动写入。长期修炼意味着长期接触这些被编码的洋能，元数据碎片不断渗入心智。天境武者的心理偏移，很大程度上来自数十年间累积的、不属于自己的情绪和感知碎片。

### 天境武者的心理画像

天境武者不是疯子。但他们每个人都在和某种常人不理解的东西共存。

* 有些天境选择沉默寡言，怕说出的话已不再来自"自己"

* 有些天境刻意维持琐碎的日常习惯（每天逛菜市场、养猫、种花）用普通人的仪式对抗共情偏移

## 人类的定义


在本地泡0，是否为"人类"由两个条件判定：


| 条件 | 说明 |
| ---- | --------------- |
| 碳基生命 | 生命形式仍为碳基。这是硬边界  |
| 肉体完整 | 物理身体未发生不可逆的非人转化 |

跨越者与天境武者的本质区别不在力量大小，在**身体的基础化学**。这也是人类洁癖派的底层逻辑：他们捍卫的不是某种文化身份，是一条生物边界。
