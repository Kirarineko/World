# 洋能峰值时代

> 那二十一年，天与地之间的距离比现在近得多。 今人称之为神话。当事人不知道自己活在神话里。


## 概述

约两千年前，本泡的洋能渗透率出现了一次剧烈的、短暂的异常峰值。

峰值持续了约二十一年。

在这二十一年中，洋能浓度远超当代水平。大量超自然现象涌现，亚空间生物穿越维度界面进入泡内，人类中出现了能够驾驭洋能的个体，地理与气候在洋能场的影响下发生局部畸变。

峰值结束后，洋能浓度急剧衰退。那些曾经可以驾驭洋能的人失去了力量，那些穿越而来的亚空间生物要么消亡、要么退回、要么以残缺的形态留存于传说中。二十一年的"奇迹"在随后的数百年中被口述、被神化、被改写，最终变成了今人所知的神话与传说。

没有人知道峰值为何发生。也没有人知道它为何结束。

## 峰值期间的现象

### 亚空间生物入侵

高洋能环境下，维度界面的稳定性下降。泡内的复用区出现大量指针偏移——裂缝。

亚空间生物通过裂缝进入泡内。多数为低智慧的浮游层与巡游者种群，但有零星记录暗示，某些更大型、更智慧的存在也曾短暂出现于泡内。

这些生物在峰值结束后多数无法存活：泡内的洋能浓度虽然在峰值期间极高，但仍低于亚空间。当峰值衰退，洋能浓度骤降，依赖洋能维持生命的亚空间生物迅速消亡。它们的残骸、它们的行为模式、它们给人类留下的印象，被编织进了后世的传说中。

> 沧溟渡传说中那头"比船体大得多，表面没有鳞片，有缓慢流动的光纹"的巨兽，可能是某种深渊种在高洋能环境下进入浅洋甚至海面的罕见记录。它没有攻击那个渡海之人：它可能根本没注意到他。

### 人类中的洋能驾驭者

峰值期间，人类中出现了一批能够感知并驾驭洋能的个体。

他们不是现代意义上的"跨越者"——那个概念建立在学术理解之上，而两千年前没有这种理解。他们是什么？

一种可能是：在极高洋能环境下，某些人类的镜像系统在自然轮换中被洋能覆写的概率大幅上升。峰值期间的"跨越者"可能比现代多出数个数量级。

另一种可能是：峰值期间的洋能浓度高到可以被人类直接感知和调用，无需跨越镜像的门槛。就像一个人站在齐腰深的水中，不需要学会游泳也能被水托起。

无论哪种情况，这些人被当时的社会称为——

英雄。神。天选者。圣者。

名称因地域而异，含义相同：能做到常人做不到的事的人。


## "英雄"

> 他们不是传说中那样。但传说也没有完全说错。

### 能力

峰值期间的洋能驾驭者，能力范围远超现代的低阶跨越者。这并非因为他们更强，而是因为环境洋能浓度极高——就像一个现代的游泳冠军和一个古代的渔民在暴风雨中的海里，技术差距被环境抹平了。

| 能力类型 | 推测范围 |
| - | - |
| 肉身强化 | 远超玄境极限。可能接近现代的归境 |
| 魔力操控 | 粗糙但强大。能级可能达到烈（M）甚至炽（N）层 |
| 感知扩展 | 可以感知到远处的生命体、地形、甚至亚空间裂缝的位置 |
| 环境影响 | 局部改变气候、影响动植物生长、修复或破坏地形 |
| 跨维度感知 | 零星记录暗示，部分个体可以"看到"或"感觉到"亚空间的存在，但无法命名或理解 |


这些能力在现代的能层能级体系中可以被对标，但对标本身没有意义——他们不知道自己在"对标"什么。

### 结局

峰值结束后，洋能浓度急剧衰退。洋能驾驭者们经历了什么？

| 可能性 | 说明 |
| - | - |
| 能力消失 | 最普遍的情况。洋能衰退后，镜像覆写层失去能量供给，镜像出现破损，但记忆仍在 |
| 身体崩溃 | 部分个体可能因洋能依赖过深，在能量断供后出现严重的镜像缺失。这些人可能在峰值结束后的数年内死亡 |
| 隐退 | 少数个体可能保留了部分能力（残余洋能足够维持基本功能），选择隐居，将经历口述给后人 |
| 消亡 | 极少数可能在峰值期间已彻底亚空间化（生命形式转变），峰值结束后无法在低洋能环境中存活 |


> 没有"英雄不死"的结局。他们中的大多数，在二十一年的奇迹结束后，变成了普通人。少数变成了传说。


## 历史记录的失真

### 为什么失真

| 原因 | 说明 |
| - | - |
| 时间太短 | 21年不足以建立稳定的文字记录体系。当时的文明可能处于口述传统为主、文字记录为辅的阶段 |
| 来得太突然 | 峰值的起始没有预兆。当人们意识到"世界变了"的时候，峰值可能已经过了数年 |
| 结束得太快 | 峰值的衰退可能是急剧的。当人们还在适应"新世界"的时候，旧世界已经回来了 |
| 社会动荡 | 峰值结束后的数百年中，社会经历了剧烈动荡（无名碑所记载的边境战争可能与此有关）。文字记录在战乱中散失 |
| 神化倾向 | 人类倾向于将无法解释的现象神化。21年的"奇迹"在口述中被不断放大，最终变成了神话 |


### 失真的形式

| 形式 | 说明 |
| - | - |
| 时间错位 | 事件被移置到不同的年代。一个发生在峰值第3年的事件，可能被记录为"上古时期" |
| 人物合并 | 多个洋能驾驭者的事迹被合并到一个人身上，形成"超级英雄"式的传说 |
| 因果倒置 | 人们用结果解释原因。"他能飞是因为他是神"，而不是"他能飞是因为洋能浓度足够高" |
| 细节丢失 | 具体的技术细节（如何使用洋能、如何感知亚空间）在口述中丢失，只留下"他做到了什么"的叙事 |
| 地域混淆 | 不同地区的传说互相渗透，形成矛盾的地理描述 |


> 今人读到的每一个前魔力时代传说，都是峰值时代的一个碎片。碎片与碎片之间无法拼合，因为它们来自不同的地域、不同的口述者、不同的神化方向。
