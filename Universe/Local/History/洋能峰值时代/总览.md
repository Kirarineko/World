# 洋能峰值时代

> 那二十一年，天与地之间的距离比现在近得多。 今人称之为神话。就像后羿射日、女娲补天、山海经里的奇禽异兽。 没有人把这些传说当真。

## 概述

约两千年前，本泡的洋能渗透率出现了一次剧烈的、短暂的异常峰值。

峰值发生时，人类已有早期文明。彼时的文明处于口述传统为主、文字记录为辅的阶段。

峰值持续了约二十一年。

在这二十一年中，洋能浓度远超当代水平。大量超自然现象涌现：亚空间物质穿越维度界面进入泡内，人类中出现了能够驾驭洋能的个体，地理与气候在洋能场的影响下发生局部畸变。

峰值结束后，洋能浓度急剧衰退。那些曾经可以驾驭洋能的人失去了力量，那些穿越而来的亚空间物质要么消散、要么退化、要么以残缺的形态留存于传说中。二十一年的"奇迹"在随后的数百年中被口述、被神化、被改写，最终变成了今人所知的神话与传说。

**人类完全不知道当时存在洋能。**

这些传说从未被当真。后羿射日、女娲补天、山海经——它们是"古人的想象"，是"文化符号"，是"不能用科学解释的上古记忆"。没有人认为它们是真实发生过的事。

## 成因：亚空间堰塞湖决堤

### 机制

1. **复用区**：亚空间中存在一片较大的复用区，与本地泡共享物理坐标。这片复用区指向本地泡的地球

2. **隐蔽裂隙**：复用区附近已经存在大量裂隙。但这些裂隙大部分在深山深海中，有的甚至被压在山体下，当时人类**无法发现**，完全没有记录

3. **堰塞湖**：复用区附近存在一个巨大的"堰塞湖

4. **决堤**：堰塞湖决堤了。大量富含洋能的水通过那些隐蔽的裂隙冲入本地泡的地球

### 洋能释放

亚空间的一切物质都含有大量洋能，因为亚空间的"空气"就是洋能。堰塞湖里的就是水，亚空间的水。

当这些水冲入本地泡后，由于泡内环境洋能水平极低，亚空间介质开始**疯狂释放洋能**。

就像一块烧红的铁掉进冷水里：铁会把热量传给水，直到温度平衡。亚空间的水掉进泡内：水会把洋能传给环境，直到浓度平衡。

但堰塞湖的水量太大了。释放的洋能远远超过了泡内环境的正常水平。整个泡内的洋能浓度飙升，持续了约二十一年，直到堰塞湖的水释放殆尽，洋能浓度才急剧回落。

### 时间线

| 阶段 | 说明 |
| - | - |
| 决堤前 | 泡内洋能浓度**远低于现代**。裂隙存在但隐蔽，无人发现。已存在早期人类文明 |
| 决堤瞬间 | 大量亚空间介质通过隐蔽裂隙涌入泡内。可能伴随地震、海啸、气候异变 |
| 峰值期（21年） | 亚空间介质持续释放洋能，泡内浓度极高。超自然现象涌现，"英雄"出现 |
| 衰退期 | 堰塞湖的水释放殆尽，洋能浓度急剧回落。"英雄"失去能力，亚空间物质消散 |
| 峰值后 | 洋能浓度回落，但仍高于决堤前。此后缓慢上升，至现代仍处于不均匀的上升趋势中 |


## 地理位置

裂缝所在的位置，现在被称为：

| 地点 | 说明 |
| **星银山脉** | 高山 |
| **洛神洋** | 大海 |
| 位置关系 | 高山紧邻着大海 |



- 星银山脉向北约**170km**就是飑髓的发现地点，即Lynx空难的发生地点

### 山脉的异常地质特征

- 存在大量**无法解释的冲刷形成的结构**

- 存在**极其怪异的已经干枯的地下暗河**

- 这些特征是**堰塞湖决堤的物理痕迹**，那是两千年前亚空间水冲入泡内时留下的冲刷印记

### 现状：裂缝仍在，洋能偏高

- 那些裂缝**依然存在**，没有闭合

- 洋能仍在从亚空间持续渗透

- 星银山脉/洛神洋区域的**洋能水平依然偏高**

- 这解释了为什么飑髓会出现在那个区域——它们可能被洋能吸引，或者洋能环境有利于它们的生存

## 为什么没有记录

| 原因 | 说明 |
| - | - |
| 裂隙隐蔽 | 裂隙在深山深海、甚至被压在山体下，人类无法发现 |
| 时间太短 | 21年不足以建立稳定的文字记录体系 |
| 来得太突然 | 峰值的起始没有预兆，当人们意识到"世界变了"的时候，峰值可能已经过了数年 |
| 结束得太快 | 峰值的衰退是急剧的。当人们还在适应"新世界"的时候，旧世界已经回来了 |
| 社会动荡 | 峰值结束后的数百年中，社会经历了剧烈动荡，文字记录在战乱中散失 |
| 神化倾向 | 人类倾向于将无法解释的现象神化。21年的"奇迹"在口述中被不断放大，最终变成了神话 |
| 不被当真 | 就像现实中的上古神话，没有人认为这些传说是真实发生过的事 |

### 失真的形式

| 形式 | 说明 |
| - | - |
| 时间错位 | 事件被移置到不同的年代。一个发生在峰值第3年的事件，可能被记录为"上古时期" |
| 人物合并 | 多个天选者的事迹被合并到一个人身上，形成"超级英雄"式的传说 |
| 因果倒置 | 人们用结果解释原因。"他能飞是因为他是神"，而不是"他能飞是因为洋能浓度足够高" |
| 细节丢失 | 具体的技术细节（如何使用洋能、如何感知亚空间）在口述中丢失，只留下"他做到了什么"的叙事 |
| 地域混淆 | 不同地区的传说互相渗透，形成矛盾的地理描述 |

> 今人读到的每一个前魔力时代传说，都是峰值时代的一个碎片。碎片与碎片之间无法拼合，因为它们来自不同的地域、不同的口述者、不同的神化方向。


## 天选者

> 设定文档中也使用"英雄"或"洋能驾驭者"称呼同一群体。

峰值期间出现的洋能驾驭者，在现代流传的神话中被称为"天选者"。

他们不是现代意义上的"跨越者"——那个概念建立在镜像机制的学术理解之上，而两千年前没有这种理解。他们是什么？没人知道。

一种可能是：在极高洋能环境下，某些人类的镜像系统在自然轮换中被洋能覆写的概率大幅上升。

另一种可能是：峰值期间的洋能浓度高到可以被人类直接感知和调用，无需跨越镜像的门槛。就像一个人站在齐腰深的水中，不需要学会游泳也能被水托起。

无论哪种情况，这些人被当时的社会称为：英雄。神。天选者。圣者。名称因地域而异，含义相同：能做到常人做不到的事的人。

### 能力

峰值期间的天选者，能力范围远超现代的低阶跨越者。这并非因为他们更强，而是因为环境洋能浓度极高——就像一个现代的游泳冠军和一个古代的渔民在暴风雨中的海里，技术差距被环境抹平了。

| 能力类型 | 推测范围 |
| - | - |
| 肉身强化 | 远超玄境极限。可能接近现代的归境 |
| 魔力操控 | 粗糙但强大。能级可能达到烈（M）层 |
| 感知扩展 | 可以感知到远处的生命体、地形、甚至亚空间裂缝的位置 |
| 环境影响 | 局部改变气候、影响动植物生长、修复或破坏地形 |
| 跨维度感知 | 零星记录暗示，部分个体可以"看到"或"感觉到"亚空间的存在，但无法命名或理解 |

这些能力在现代的能层能级体系中可以被对标，但对标本身没有意义，他们不知道自己在"对标"什么。

### 结局

峰值结束后，洋能浓度急剧衰退。天选者们经历了什么？

| 可能性 | 说明 |
| - | - |
| 能力消失 | 最普遍的情况。洋能衰退后，镜像覆写层失去能量供给，镜像出现破损，但记忆仍在 |
| 身体崩溃 | 部分个体可能因洋能依赖过深，在能量断供后出现严重的镜像缺失。可能在峰值结束后的数年内死亡 |
| 隐退 | 少数个体可能保留了部分能力（残余洋能足够维持基本功能），选择隐居，将经历口述给后人 |
| 消亡 | 极少数可能在峰值期间已彻底亚空间化（生命形式转变），峰值结束后无法在低洋能环境中存活 |

> 没有"英雄不死"的结局。他们中的大多数，在二十一年的奇迹结束后，变成了普通人。少数变成了传说。

### 已知

- 有天选者**去了亚空间**

### 留白

- 进入亚空间的天选者有多少？他们在那里经历了什么？

## 神话地位

洋能峰值时代留下的传说，在后世的地位类似现实中的后羿射日、女娲补天、山海经——是"古人的想象"，是"文化符号"，是"不能用科学解释的上古记忆"。

没有人把这些传说当真。

它们与后来的历史事件无关。那些是各自独立的历史，发生在洋能峰值结束很久之后。
