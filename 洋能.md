# 洋能编码

> 洋能不只是能量。它是一张白纸，使用者需要往上写字。

## 概述

洋能可以被编码携带信息。使用者施展法术时，需要将法术所需的变量、参数、目标指向等数据存入洋能中。洋能本身只是载体，不具有执行能力，执行仍依赖使用者的操作。

大部分使用者并不知道自己在编码。就像人不会察觉自己正在分泌肾上腺素一样，编码是洋能使用中的底层生理过程，被意识察觉的只是最终效果。

## 编码

### 信息与元数据

每个被编码的洋能单元包含两层内容：

| 层级 | 内容 | 写入方式 |
|------|------|----------|
| 内容 | 法术变量、参数、目标指向等使用者有意存入的信息 | 可控制（仅少部分生物） |
| 元数据 | 使用者当时的情绪、记忆片段、感官快照、思维碎片 | 可以抑制，内容不可控 |

元数据始终伴随编码产生，大部分生物无法关闭。它像数字照片的 EXIF 信息：拍摄时自动记录，拍摄者不一定知道它存在。法术执行完毕后，数据包和元数据应当被清除，但总有遗漏。

### 控制层级

编码的控制能力因物种而异：

大部分生物完全自动，它们无法控制编码内容，如同心跳，身体自行完成。这类生物通常也无法得知洋能存有编码这件事。

少部分生物可控制内容，可主动决定存入什么信息，但仍无法控制元数据写入。感觉类似呼吸，通常由身体静默调节，但可以手动接管。

元数据方面，极少数生物可以选择不写入元数据，但无法控制写入什么。这是一个开关：写入或不写入，没有中间态。不写入元数据的洋能基本无法用于大部分施法用途——它失去了与使用者心智的耦合，变成一具没有灵魂的能量壳，无法被使用者的意图精确驱动。因此绝大多数施法者不会去抑制它。

### 编码的无意识性

大部分洋能使用者（包括大多数武者和法师）在施展能力时，不会感知到编码过程本身。他们感知到的是"我要让火球飞向那边"，而不是"我要在洋能中写入坐标参数和燃烧强度"。

但这些信息确实被写入了。连同他们当时的心理状态一起。

## 解码

### 被动解码

部分非洋能生物接触残留洋能时，洋能中携带的信息会被不受控地写入大脑。这个过程没有防火墙，没有过滤器，信息直接进入感知层。

零零碎碎的情绪片段、记忆碎片、感官快照被大脑当作自身的输入信号处理。结果是幻觉、错觉、情绪波动、莫名的恐惧或熟悉感。当事人不会察觉这些感觉不属于自己——它们被直接输出到了感官，和自身的感知没有边界。

### 解码抵抗

不同群体对环境残留洋能的解码行为差异显著：

原生洋能生物不会像人类那样被动解码环境洋能。

跨越者（大部分）跨越后解码行为发生质变，不再无脑接收环境洋能中的信息。特殊情况仍有少量渗入，日常影响微乎其微，但战斗中面对高密度编码洋能时风险骤升

部分非洋能生物无抵抗能力。接触即解码|

### 第六感

洋能生物（包括跨越者）在日常活动中部分行为会释放微量洋能，可能会有残留。这些残留洋能中携带的元数据往往包含了释放者当下的感知状态。

具体过程：

1. 洋能生物感知到周围有什么（比如"有人在看我"），这个感知被作为元数据写入了它自然散发的洋能中
2. 附近的人类接触这些残留洋能，被动解码
3. 大脑收到了"有东西在看我"的信号，但这个信号不是接收者自己的，是释放者的
4. 由于直接输出到感官，接收者下意识认为这就是自己的直觉
5. 潜意识发现自己接受到了一些奇怪的东西，于是产生了"这里不太对""有人在盯着我"的感觉，却说不出为什么

这就是人类对身边潜伏的跨越者或洋能生物产生"第六感"的本质：**你以为是自己的感觉，其实是别人的情绪碎片。**

这种机制的普遍性远超预期。一个洋能生物在暗处观察一个人时，它散发的洋能中携带的"我在看"这个元数据，在目标人物同样被解码为"我在看"。

反过来，一个放松的、没有警觉的洋能生物（比如在树上趴着的茉璃），散发的洋能中携带的是平静的元数据，路人接触后会觉得"这个角落让人平静"，却不知道为什么。

### 精神污染与控制

洋能编码机制的副产品之一是精神污染的可能性。

对洋能生物实施精神污染需要复杂的木马注入，需要绕过对方的解码防线，将恶意信息伪装成合法输入。这在亚空间中是一项需要高超技巧的攻击手段。

对本地泡普通生物则简单得多。由于它们无法区分环境中洋能信息的来源，攻击者只需将要注入的内容直接写入洋能，目标会自行解码并将外来信息当作自身感知处理。跨越者的解码行为在跨越后有所改善，但防护力度仍远不及洋能生物，属于中等难度的目标。

### 交流

亚空间洋能生物没有语言。它们的交流方式是利用洋能编码直接传递信息，将自己的知道的信息原封不动地编码进洋能中，发送给对方。对方解码后获得的是发送者的第一视角体验，而非经过语言压缩的描述，由于带宽受限，所以虽然信息的准确度极高，但是交流特别慢，可控编码的生物可以选择性发送需要的信息。

能够控制编码内容的生物可以精确选择发送什么信息。无法控制编码内容的生物同样可以交流，只是它们不知道自己交流的原理，也不知道具体传递了哪些类型的信息。

不同生物的编码形式在细节上可能有差异，但只要差异不太离谱，经过熟悉后都可以互相翻译。这使得亚空间中不同物种之间存在一种基于洋能的底层沟通能力。

## 残留与持久度

法术执行完毕后，洋能中的数据包和元数据应当被清除。但多余洋能释放后未被使用的情况，由于未被使用，数据自然没有被清除，如果使用者没有回收，这些残留洋能带着编码信息留在环境中。

残留洋能的持久度因法术类型和使用者的具体操作而异，没有统一规律。某些法术释放的洋能衰减极快，信息在数分钟内消散；另一些可能持续数月时甚至更久。一般而言，释放时越"用力"、信息密度越高的洋能，残留时间越长。

环境中长期存在的残留洋能形成了某种"信息背景噪声"。在洋能浓度较高或频繁有修炼者活动的区域，这种噪声尤为明显。长期生活在这种环境中的人，可能发展出对编码洋能的微弱适应性。

## BUG

本地泡大部分生物在使用洋能时存在一个独特的bug：它们无法区分环境中解码到的洋能信息是自己释放的还是他人释放的。这意味着使用者自己释放的洋能中携带的元数据，会被自己的身体当作外部输入再次解码。

结果是递归。

内心的想法被编码进洋能，释放后又被自己解码回来，当作新的感知输入，强化原始想法，再编码循环往复。左脚踩右脚上天。

这个缺陷源于本地泡精简镜像系统的演化选择：省下的神经资源换来了远超平均水平的智力，但也丢掉了"区分自我与外部信号"的过滤链。

### 递归的三级表现

| 程度 | 表现 |
|------|------|
| 轻度 | 行为极端化、固执。原本模糊的情绪被递归放大为明确的执念。一个只是"有点嫉妒"的人可能在数轮递归后变成"我必须毁掉他" |
| 中度 | 被自己的野性淹没。理智层被递归的情绪洪流冲垮，退化为本能驱动。行为不再受理性约束 |
| 重度 | 扰乱镜像系统的指针。指针定位异常乱跳，在不同差分层之间疯狂跳跃。|

轻度递归在修炼者中极为常见，几乎每个使用洋能的本地泡生物都会经历。大部分情况下递归在轻度就停止了，可能是使用者的情绪自然平复，或者洋能耗尽，循环断开。轻度递归的外在表现是"这个人今天有点不对劲"，不会被当作异常。

中度递归较为罕见，通常发生在情绪本就极端的状态下（暴怒、极度恐惧、深仇大恨）。当事人的行为会明显脱离理性轨道。

重度递归本身仍然不等于沉渊。它只是打开了通往沉渊的门。

### 沉渊的触发

重度递归是沉渊的必要条件，但不是充分条件。沉渊的触发需要一连串小概率事件同时发生：

1. 重度递归发生
2. 递归期间，镜像系统恰好发生了一次覆写——某个原始块的引用计数归零，被新的差分层覆盖
3. 使用者恰好解码了覆写发生前释放的洋能
4. 那段洋能中恰好存在被覆写处的旧地址信息
5. 指针跟着这个旧地址去寻找数据
6. 被覆写处已经是空指针——数据已被覆盖，地址失效
7. 💥

六个环节，任何一个断开，递归就回到轻度或中度，不会演变成沉渊。

### 现代沉渊率低的原因

沉渊战争之后，人类对洋能影响心智的认知大幅提升。这种认知本身构成了一道防线：当使用者下意识对洋能保持警惕时，递归的起始强度被削弱，从轻度递归升级为中度的概率下降，重度递归变得极为罕见。

换言之，人类用"知道自己可能被骗"这件事，部分抵消了"无法区分自我与外部信号"的演化缺陷。这不是彻底的修复，但足以让沉渊从战前的高频灾难变成如今的偶发事故。

## 对普通生物的影响

本地泡的普通动物接触亚空间洋能时，可能发生一种与递归效应不同的现象。

亚空间洋能中可能携带镜像特性的信息，是镜像系统本身的结构参数。当这些信息在生物体内"下载"完成，如果恰好凑齐了一组完整的特性模块，生物的身体可能会"安装"这个模块。

安装后的表现：

| 变化 | 说明 |
|------|------|
| 降智 | 神经资源被新安装的镜像特性占用，分配给通用智力的部分减少 |
| 生命力增强 | 镜像系统变得更完整，身体的自我修复、抗损伤能力提升 |

这不是跨越。是生物的镜像系统被外来信息补全了原本缺失的结构，代价是智力退让。

人类的镜像系统经过本地泡特有的精简演化，安装模块所需的接口已被阉割，不会发生这种情况。但成为跨越者本身就会提升身体素质和生命力，这是镜像底层被洋能覆写后的附带效果，与模块安装无关。

再加上递归效应的叠加，虽然动物发生的递归普遍不深，但足够造成明显的症状：异常暴躁、过度警觉、攻击性增强、行为模式固化。两者叠加后的动物往往表现为"比以前更难杀，也比以前更蠢，而且脾气很差"。

野外偶尔出现的异常野兽目击，可能与此有关。

### 模块安装能力

部分洋能生物具备帮助其他生物安装模块的能力。但在亚空间中这种能力极少被使用，具备这种能力的生物不一定知道自己有这个能力，知道的也不一定理解它能做什么，更多的是不知道还有这种能力。亚空间没有"给邻居装模块"的文化或需求，这项能力基本处于沉睡状态。
