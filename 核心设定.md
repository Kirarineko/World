# 基础设定

## 世界观概括

宇宙像一串彼此隔开的封闭气泡。每个泡都有自己的边界，边界会随着大质量天体的移动而鼓起。泡与泡之间没有空间关系，物质要过去，只能走亚空间。

## 一、泡宇宙模型

· 泡与泡之间在物理空间上互不相邻。不存在“两个泡之间隔着一段亚空间洋体”这种空间关系。
· 每个泡的泡壁之外，没有另一个泡。泡壁之外是什么？或许是虚空，或许是绝对的维度边界，这个问题本身在物理上无意义。
· 亚空间不是泡之间的“填充物”，是另一个维度的整个存在：它连接所有泡，但不与任何泡共享物理空间坐标。

### 1.1 泡的形态与结构

| 条目 | 设定 |
|------|------|
| 形态 | **不规则立体多边形**，非球体 |
| 大小 | **动态变化**，随泡内大质量天体位置实时改变 |
| 理论上限 | 可无限大。一个泡的内部空间可以容纳一整个可观测宇宙：泡的大小仅在决定其物理尺度的同时，不设上限 |
| 泡壁性质 | 一层实时跟随大质量天体的弹性"膜" |

### 1.2 泡壁机制

| 规则 | 说明 |
|------|------|
| 触发条件 | 大质量物体靠近泡壁时，泡壁被向外推开 |
| 响应速度 | **实时**，零延迟、零惯性。天体跑多快，泡壁跑多快 |
| 小质量物体 | 靠近泡壁时泡壁不动，无法影响泡壁位置 |
| 质量阈值 | 每个泡对"多大质量算大"有不同标准，不在基础设定中定义 |
| 靠近距离 | 不同泡对"多近算靠近"标准不同，不在基础设定中定义 |
| 收缩 | 大质量天体远离泡壁方向时，泡壁**同步实时收缩** |
| 收缩推力 | 泡壁收缩时，路径上的小质量物体被推回泡内 |

### 1.3 泡壁的穿越与推力

小质量物体可以穿越泡壁进入外层真空，穿越后会被一股推力推回泡内，推力作用于每一个基本粒子。

离泡内越远，推力越大，呈递增趋势。

大质量物体，无法穿越泡壁，它推着泡壁移动

## 二、高斯泼溅星空

| 条目 | 设定 |
|------|------|
| 位置 | 附着于泡壁内表面 |
| 形态 | 无数光点以密度分布形式呈现：有浓有淡、有聚有散，非均匀分布 |
| 存在时间 | 始终存在 |
| 可见性 | 白天因阳光遮蔽不可见，夜晚肉眼可见，如同真实星空 |
| 动态 | 实时映射，光点分布/亮度会实时变化 |
| 泡壁外光点 | 泡壁之外亦散布高斯泼溅光点，充当局域隔离层，不可穿越 |


## 三、亚空间

平行于现实空间的维度，连接所有泡，这是唯一物质跨泡通道

亚空间有风浪，存在可预见的危险

亚空间能量通过泡壁维度界面向泡内渗透。渗透率因泡而异：从零到丰沛不等。 渗透率为零的泡不存在洋能，也就没有魔法等力量体系，纯粹是物理规则下的世界 


## 五、力量本原

来自亚空间渗透的能量，泡宇宙中唯一统一的能量本源

**能层能级**。这是人类发现洋能后做的学术分类：能量本源相同。由于规律在所有泡中统一，其他文明总结出粗来的可能不一样，但是大差不差

少数有智慧文明的泡可能发展出自己本土的力量命名和修炼框架。这些本地文明的产物不与通用框架冲突，是通用框架的地方化表达
