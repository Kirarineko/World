# 基础设定

> **定位**: 本文件定义世界观底层规则，一切推导的起点。其他设定文件均引用本文件。
> **最后更新**: 待填


## 世界观概括

宇宙像一串彼此隔开的封闭气泡。每个泡都在自己的边界里呼吸，边界会随着大质量天体的移动而微微鼓起、收缩，夜空则像被人在膜上随手洒了一层细碎光点。泡与泡之间没有直通的路，物质要过去，只能借亚空间绕行；信息则另有别的走法。主角所在的泡是现代都市文明，武力修炼和魔法并行。


## 一、泡宇宙模型

· 泡与泡之间在物理空间上互不相邻。不存在“两个泡之间隔着一段亚空间洋体”这种空间关系。
· 每个泡的泡壁之外，没有“另一个泡”，也没有“一片汪洋”。泡壁之外是什么？或许是虚空，或许是绝对的维度边界，或许这个问题本身在物理上无意义。
· 亚空间不是泡之间的“填充物”，而是另一个维度的整个存在：它连接所有泡，但不与任何泡共享物理空间坐标。

### 1.1 泡的形态与结构

| 条目 | 设定 |
|------|------|
| 形态 | **不规则立体多边形**，非球体 |
| 大小 | **动态变化**，随泡内大质量天体位置实时改变 |
| 理论上限 | 可无限大，无法触碰到泡壁外的高斯泼溅光。一个泡的内部空间可以容纳一整个可观测宇宙：泡的大小仅在决定其物理尺度的同时，不设上限 |
| 泡壁性质 | 一层实时跟随大质量天体的弹性"膜" |

### 1.2 泡壁机制

| 规则 | 说明 |
|------|------|
| 触发条件 | 大质量物体靠近泡壁时，泡壁被向外推开 |
| 响应速度 | **实时**，零延迟、零惯性。天体跑多快，泡壁跑多快 |
| 小质量物体 | 靠近泡壁时泡壁不动，无法影响泡壁位置 |
| 质量阈值 | 每个泡对"多大质量算大"有不同标准，不在基础设定中定义 |
| 靠近距离 | 不同泡对"多近算靠近"标准不同，不在基础设定中定义 |
| 收缩 | 大质量天体远离泡壁方向时，泡壁**同步实时收缩** |
| 收缩推力 | 泡壁收缩时，路径上的小质量物体被推回泡内 |

### 1.3 泡壁的穿越与推力

| 条目 | 设定 |
|------|------|
| 小质量物体 | 可以穿越泡壁进入外层真空 |
| 穿越后遭遇 | 被一股推力**推回泡内** |
| 推力作用对象 | **每一个基本粒子**，无一例外 |
| 推力规律 | 离泡内越远，推力越大，呈递增趋势 |
| 大质量物体 | 无法穿越泡壁：它推动泡壁移动，但泡壁始终包裹着它 |

### 1.4 泡壁外空间

| 区域 | 内容 |
|------|------|
| 紧贴泡壁外侧 | 绝对真空：无粒子、无能量、纯虚空 |
| 泡与泡之间的空间 | 绝对真空 + 高斯泼溅光点 |

### 1.5 泡间关系

| 项目 | 状态 | 说明 |
|------|------|------|
| 物理穿越 | ❌ 不可行 | 小质量被推回，大质量出不去|
| 泡间碰撞 | ❌ 不可能 | 不同泡之间只有高斯泼溅光点，无碰撞条件 |
| 信息传递 | ✅ 可行 | 具体机制待定义 |
| 物质交通 | ✅ 仅亚空间 | 唯一物质通道，绕过泡壁与光点壁垒 |


## 二、高斯泼溅星空

| 条目 | 设定 |
|------|------|
| 位置 | 附着于泡壁内表面 |
| 形态 | 无数光点以密度分布形式呈现：有浓有淡、有聚有散，非均匀分布 |
| 存在时间 | **始终存在**，非仅夜晚 |
| 可见性 | 白天因阳光遮蔽不可见，夜晚肉眼可见，如同真实星空 |
| 动态 | **实时映射**，光点分布/亮度会实时变化 |
| 映射对象 | 待定（留白，作为故事深层谜团） |
| 泡壁外光点 | 泡壁之外亦散布高斯泼溅光点，充当局域隔离层，不可穿越 |

> **描写参考**：泡壁并非漆黑虚无，而是布满无数微小的光点。这些光点并非均匀分布，而是像墨水滴入水中那样，有浓有淡、有聚有散：某些区域密集如星团，某些区域稀疏如深空。它们固定在泡壁上，不旋转、不移动，与行星上的任何季节和方位无关。天文观测早已证实：那不是恒星，那是泡壁本身在发光。


## 三、亚空间

| 条目 | 设定 |
|------|------|
| 性质 | 平行于现实空间的温和维度，连接所有泡 |
| 地位 | **唯一物质跨泡通道** |
| 风险等级 | 相对温和：有风浪但不极端，存在可预见的危险但不至团灭 |
| 危险示例 | 迷路、亚空间生物、时间流速异常区域 |
| 洋能渗透 | 亚空间能量通过泡壁维度界面向泡内渗透。**渗透率因泡而异：从零到丰沛不等。** 渗透率为零的泡不存在洋能，也就没有魔法/高武等力量体系，纯粹是物理规则下的世界 |
| 进入方式 | 待定 |
| 与高武/魔法的关系 | 待定 |
| 详细信息 | 见 `亚空间志.md` |


## 五、力量本原

| 条目 | 设定 |
|------|------|
| 能量名称 | **洋能**：来自亚空间渗透的能量，泡宇宙中唯一统一的能量本源 |
| 力量类型 | 高武（武术/内力/修炼体系）+ 魔法 |
| 能量本源 | 亚空间洋能透过泡壁维度界面渗透至各泡。**洋能渗透并非必然：部分泡的渗透率为零，其内部不存在洋能，纯粹是物理规则下的世界。** 渗透率大于零的泡才有可能产生魔法/高武文明 |
| 通用框架 | **能层能级体系（K-Q 能层 + s-i 能级）**。这是对洋能规律的学术分类：能量本源相同，因此规律在所有泡中统一。但需要注意的是：绝大多数泡**不存在智慧文明**（见智慧断崖设定），因此这套框架在实际中仅在屈指可数的几个文明泡中被建立和使用 |
| 本地体系 | 少数有智慧文明的泡可能发展出自己本土的力量命名和修炼框架（如主角所在泡的"金玄归天"武力体系）。这些是本地文明的产物，不与通用框架冲突，而是通用框架的地方化表达 |
| 二者关系 | 能层能级 = 跨泡通用标准（仅有文明泡的学术界共享）；各泡本土体系 = 本地文明在不知晓其他泡存在的情况下独立建立的框架。一个泡可能根本不知道能层能级的存在，但其洋能使用者仍然符合该体系的客观规律 |
| 普及性 | 待定（人人可学？天赋限定？） |
| 上限 | 待定（个人战力层面？毁城？碎星？） |
| 跨泡有效性 | 通用。不同泡的洋能使用者可以用能层能级体系互相对标：即便他们的本地命名完全不同 |
| 详细信息 | 见 `能力体系.md`（通用框架）、各泡专项设定（本土体系） |
