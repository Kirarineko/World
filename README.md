# World · 泡宇宙世界观

> 宇宙由无数独立封闭的"泡空间"构成，泡壁随大质量天体实时变形，天空以高斯泼溅光点渲染星空。泡间绝无物理通道：信息可传递，物质只能走温和的亚空间。

---


一个泡即一个宇宙。主角所在泡为现代都市文明，高武与魔法并存，但泡内无人知晓自己身处泡中。

## 核心设定

- **泡宇宙模型**：每个泡都是一层封闭宇宙。泡壁会跟着大质量天体移动，小质量物体靠过去会被推回。泡壁外不是“别的泡”，也不是亚空间，而是一片更难用语言说清的空域。
- **高斯泼溅星空**：夜空里的光点不是随便挂上去的星星，而是贴在泡壁上的发光分布。白天看不见，夜里像真的一样。
- **亚空间**：它是另一层维度，不是泡之间的夹层。物质要跨泡，只能借它走一趟；它本身也有自己的洋流、风暴和生命。
- **洋能**：亚空间渗进来的能量。本质统一，但每个泡能接到多少不一样，接不到的地方就没有魔法，也没有高武。
- **能层能级体系**：K→Q 的能层，配上 s→i 的能级。N 到 O 不是普通升级，是身份开始变样的门槛。
- **智力断崖**：这个宇宙里，真正能把文明搭起来的智慧物种少得离谱。

## 主角：茉璃

茉璃是亚空间原生的洋能生命，长成了猫娘的样子，当前是炽·散（N·4d）。

二十年前，她在自己的领地边缘碰到一道新裂隙，好奇心把她推了过去。另一边是沼泽、平原、陌生的天空，还有一个把她带回星月湾的人。她在那里学会人类语言，给自己起了名字，也就顺势留下来了。

后来她常做的事很普通：帮HBI学生捞掉进海里的仪器，去漫展当吉祥物，偶尔接点湿件维修。句尾总会冒出一点“喵”，已经改不掉了。

> "若是繁星还没来，烟火也可映沧海。"
