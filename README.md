# World · 泡宇宙世界观

> 宇宙由无数独立封闭的"泡空间"构成，泡壁随大质量天体实时变形，天空以高斯泼溅光点渲染星空。泡间绝无物理通道：信息可传递，物质只能走温和的亚空间。

---


一个泡即一个宇宙。主角所在泡为现代都市文明，高武与魔法并存，但泡内无人知晓自己身处泡中。

## 核心设定

- **泡宇宙模型**：泡壁是一层实时跟随大质量天体的弹性膜，小质量物体穿越后会被推回。泡壁外是虚空+高斯泼溅光点（既不是另一个泡，也不是亚空间）
- **高斯泼溅星空**：附着于泡壁内表面的无数光点，以密度分布形式呈现，始终存在，昼夜均在。天文学层面与真实恒星无法区分（无人知晓自己看到的星星中哪些是投影）
- **亚空间**：平行于现实空间的独立维度，唯一物质跨泡通道。一片没有海面的汪洋，有流有浪有风暴，有独立的生态系统和称为"恶魔"的原生智慧生命
- **洋能**：来自亚空间渗透的能量，泡宇宙中唯一统一的能量本源。渗透率因泡而异（渗透率为零的泡不存在魔法/高武
- **能层能级体系**：K→Q 能层（微明烈炽虚极寂）+ s→i 能级（浑贯散构曲坍i留空）。N→O 是身份边界（跨过去的不再是纯粹的人类
- **智力断崖**：智力在"猫"附近存在断层，能建立文明的智慧物种在整个宇宙中屈指可数

## 主角：茉璃

亚空间原生洋能生物，猫娘。炽·散（N·4d）。

二十年前，她在领地边缘发现了一道崭新的裂隙。好奇心驱使下穿过裂隙，落入一片布满沼泽的广袤平原，被一个星月湾人捡到，带回了"圈里有公"的城市。她在星月湾学会了语言，给自己起了一个名字，在这座对跨越者最友好的城市里定居了二十年。

她帮HBI大学生下海捞实验仪器，去漫展当吉祥物，偶尔用假身份接湿件维修的零活。句末经常带"喵"，不是刻意，是习惯了。

> "若是繁星还没来，烟火也可映沧海。"
