
# 魔法力量体系 · 能层能级命名

> **适用范围**：跨泡通用标准。能量本源为亚空间洋能。洋能规律在物理层面客观统一，但需注意两点（① 洋能渗透率因泡而异，渗透率为零的泡不存在洋能，也就没有魔法文明）能层能级体系在那些泡中无实际意义；② 绝大多数泡**不存在智慧文明**（智慧断崖：智力在"猫"附近存在断层，能建立文明的智慧物种在整个宇宙中屈指可数）。因此能层能级体系在现实中仅被极少数的文明泡所建立和使用。各文明泡可能在不知晓彼此存在的情况下独立发展出本土的命名和修炼框架（如主角所在泡的"金玄归天"武力体系），但其底层能量学仍可纳入本体系对标。

参考现实电子能级

## 收敛

> 升层未必变强。有时退一步，形态才跟得上。

由于能级交错的存在（与电子轨道能级交叠同理：能量序并非始终按层递增，例如 4d < 5p < 6s < 4f），提升能层后能级的有效顺序会重组。一个炽·散（4d）的修行者在突破到虚层后，实际可用的能级可能退化为虚·贯（5p），在更极端的过渡期甚至退至极·浑（6s）。那些需要高精度控制的构态能力在这段空窗期内无法使用。

空窗期不短。修行者可以选择主动收敛：降低一层能层，换取能级上升一级。

收敛后强度受限于新能层的上限，但能量总量来自原能层，续航显著增强。比如虚·贯（5p）收敛至炽·散（4d）：力道仍是炽层水平，但能量池是虚层级别的，输出时间比原生炽·散长得多。

收敛可以连续进行多级，每次降一层能层、升一级能级。但每一级收敛都有 5%~7% 的能量损耗——仅折损续航，不影响强度。

```
能级交错顺序（升层时能级退化的方向）：
  4d < 5p < 6s < 4f < 5d < 6p < 7s < 5f < 6d ...

收敛方向（降层换升级的可操作路径）：
  6s ──→ 5p ──→ 4d
  P      O      N

  7p ──→ 6d ──→ 5f
  Q      P      O

  每步：层-1，级+1，损耗 5%~7%
```

| 收敛前 | 收敛后 | 说明 |
|--------|--------|------|
| 虚·贯（5p） | 炽·散（4d） | 一层一级，强度降为炽层，续航大幅增强 |
| 极·浑（6s） | 虚·贯（5p） | 同上模式 |
| 6s → 5p → 4d | 炽·散 | 连续收敛两级，过两次损耗，仍比原生持久 |

> 收敛不是退步。是把多出来的燃料倒进一个更趁手的引擎。

## 模拟

> 高能级可以表演低能级的动作，但不是原生的。

构态（f）可以部分再现散态（d）的扩散覆盖和贯态（p）的轴向突破，但效率、精度、范围均不如原生能级。模拟的保真度取决于两种能级之间的形态相似度：构模拟散比构模拟贯更自然——散与构同属复杂形态家族，贯则是截然不同的单轴逻辑。反过来，散模拟构几乎不可能：广域覆盖无法被压缩为精细编织。

> 模拟不是替代。一个 4f 模拟的 4d 和真正的 4d 之间的差距，就像一幅仿画和原作的差距：外行看不出，内行一眼了然。
