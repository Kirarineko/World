
# 魔法力量体系 · 能层能级命名

> **适用范围**：跨泡通用标准。能量本源为亚空间洋能。洋能规律在物理层面客观统一，但需注意两点（① 洋能渗透率因泡而异，渗透率为零的泡不存在洋能，也就没有魔法文明）能层能级体系在那些泡中无实际意义；② 绝大多数泡**不存在智慧文明**（智慧断崖：智力在"猫"附近存在断层，能建立文明的智慧物种在整个宇宙中屈指可数）。因此能层能级体系在现实中仅被极少数的文明泡所建立和使用。各文明泡可能在不知晓彼此存在的情况下独立发展出本土的命名和修炼框架（如主角所在泡的"金玄归天"武力体系），但其底层能量学仍可纳入本体系对标。

参考现实电子能级

## 能层（K→Q）

| 字母 | 名称 | 含义 |
|------|------|------|
| K | **微** | 微末之初，能量萌芽 |
| L | **明** | 明朗成形，稳定输出 |
| M | **烈** | 能量强度跃升，初具杀伤 |
| N | **炽** | 炽盛巅峰，人类路线的最高完整能层 |
| O | **虚** | 能量超越可见形态，虚实难辨 |
| P | **极** | 人类路线可触及的极限能层 |
| Q | **寂** | 能量归于寂然，无处不在而无所在 |

> **炽 → 虚 = 身份边界。** 炽是火焰的顶点，也是人类最后的温度。

## 能级（s→i）

| 字母 | 名称 | 形态描述 |
|------|------|----------|
| s | **浑** | 球形均衡，能量均匀展开 |
| p | **贯** | 轴向定向，贯穿突破 |
| d | **散** | 多瓣扩散，范围覆盖 |
| f | **构** | 复杂编织，精细结构 |
| g | **曲** | 时空扭曲，维度干扰 |
| h | **坍** | 能量内陷，湮灭归返 |
| i | **-** | 不可描述，不可命名 |

> i 形态的留空是体系的一部分：该形态无法被人类语言定义。知其在，不可说。

## 身份边界

### 边界不是一条线，是渐隐带

```
炽·N                          虚·O                      极·P            寂·Q
4f ── 最后的结构型 ──────┐
                          │  5f 5g ── 非人结构型开始
                          │
4d ── 最后的范围型 ──────────── 6d ── 范围型从此不再
                          │
5p ── 贯穿型尚存 ────────────── 6p ── 最后的贯穿 ──── 7p ── 非人贯穿开始
                          │
7s ── 最后的基础型（不稳定）
```

### 三道门槛

| 次序 | 名称 | 说明 |
|------|------|------|
| 第一 | **能量** | N→O之间，客观存在。突破极难，非绝对不可破，但筛掉绝大多数 |
| 第二 | **社会认同** | 近40年前开始与半魔化者和平共存，接受度大趋势走高。代际差异显著：老一辈仍有抵触，年轻一代日渐开放 |
| 第三 | **自我选择** | 跨过去，情感与自我意识完好保留，但不再属于人类范畴。不是堕落，不是成神，是身份的更迭 |
