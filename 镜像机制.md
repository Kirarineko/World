# 镜像机制

> 每个人脑中都有一个"最初的自己"：只读快照。每一次成长、愈合、衰老，都在上面叠加一层写时复制。只要原始镜像还在，"我"就还在。

> 镜像不是免费的。它吃掉多少神经资源，决定了生物还剩多少智力可花。本地泡把所有能省的都省了，换回了远在智慧断崖以上的风景。

## 原始镜像

每个智慧生物都有着一份"生命镜像"。它是身体刚长成时留下的那张验收照：骨头怎么排，器官怎么长，还有最核心的意识于记忆，都在这里面。只要这份镜像还在，就还剩这个人

镜像不是一整块糊在一起的东西，而是按块映射全身。骨骼、肌肉、器官、神经，各自都有自己的位置。每个块都只读，除非所有指向它的引用都被后续变化一点点带走。

## 写时复制

身体的每一次自然生长、受伤后的疤痕形成、肌肉锻炼的痕迹：每一次改变都生成一个新的"差分层"。差分层写入新的数据块，然后将对应的虚拟地址重新映射到新块上。

原始块的引用计数减一。但不会归零，直到所有引用它的差分层都已经被更上层的差异重定向替代。

``` 
原始镜像块────→ 引用计数=3      
   ├── 差分层A（童年生长）   ────→ 引用该块      
   ├── 差分层B（骨折愈合）   ────→ 引用该块      
   └── 差分层C（肌肉锻炼）   ────→ 引用该块      
      
当差分层C 的更高层差异替代了它对原始块的引用：      
  原始镜像块────→ 引用计数=2      
  差分层C 不再指向原始块
```

## 镜像的日常轮换

镜像不是静止的。就像人的细胞约每七年完成一轮完全更换一样，镜像中的块也在持续的日常代谢中被释放和新写。

旧块被释放，新的差分层写入同样的物理域，地址重新映射，身体在微观层面完成了翻新。一个人的镜像在今天和几年前已经不是同一组物理块，但它仍在映射同一个"自己"。就是因为映射关系不曾断裂。

块释放每天都在发生。衰老不是块消失，是轮换进入镜像底层的差分层可能存在缺陷，缺陷的差分层再导致新的差分层生成时可能出现更多缺陷，如此循环往复
## 治疗魔法

镜像可以作为一致性检查的基准。治疗魔法就是以镜像为基准的一次校验。

以镜像中对应块的校验和为基准，修复物理损伤。魔法能修复的，一定是"曾经完好的状态被破坏后的结果"。

它不能修复的东西也正好对应：

- **先天残疾和遗传病**：镜像本身就有这个"错误"。魔法生效了，但无法发现问题，那个"错误"在这个人身上就是正确的。
- **病毒感染**：外来物不在镜像中。魔法不识别为损坏，不予清除。

## 跨越者：洋能覆写原始镜像

镜像块在日常轮换中持续被释放和重新映射。正常情况下，重新写入这些位置的新差分层依赖人体自身的生物代谢来维持。

当一个人长期接触洋能，魔力回路和洋能代谢可能在镜像块的某次轮换中占据新写入的位置。这不是一个等待数年的缓慢过程：只需要原始镜像中某个特定块被自然释放的那一刻，洋能驱动的差分层恰好抢在生物代谢之前写入了同一地址。

如果这个块涉及某些功能性镜像（比如魔力回路的"传感器"区域），新差分层中混入了洋能依赖的映射关系，那么从这一刻起，这个人的镜像就不再纯粹依赖自身代谢。

这个过程可以在多个块上重复发生。每多一个洋能覆写的块，跨越的程度就深一层。

当洋能覆写的块足够多、足够核心，整个镜像的底层介质开始发生变化。一旦外部洋能停止供给，这些块就会因为引用计数混乱而出现功能缺失。

这就是跨越者的定义：**生命镜像底层出现依赖洋能的差分层覆写。** 一旦失去洋能，镜像出现缺失。

跨越者不是吸收了多少魔力，不是达到了某个能层，是原始的某个块被洋能依赖层改写。

## 镜像与智力的资源调度

镜像不是免费运行的。原始镜像维护、差分层映射、引用计数、块释放与垃圾回收、纠错校验，这些操作消耗神经计算资源。

每个物种的神经资源总量是有限的，镜像分走多少，智力就剩下多少。

在泡宇宙绝大多数泡中，生物把几乎所有神经资源投给了镜像基础

不过通用智力再低，也是智慧断崖之上的存在，于断崖之下那些几乎完全依赖条件反射的生物有质变

本地泡走了另一条路。镜像被大幅精。省下的资源被重新配置给了通用智力。

而在这条路上，不同物种的资源调度各有不同

### 写时复制与永生

写时复制是所有有镜像的生物共享的底层机制。差分层的资源消耗随时间增长，但增长曲线不是线性的。

这条曲线是永生的关键瓶颈。

曲线需要极长时间才能降到线性以下，因物种而异。在拐点到来之前，镜像的维护成本持续高于维护能力，缺陷开始累积，镜像逐渐崩溃。这就是为什么大多数生物不能永生，它们的寿命不够长，撑不到曲线平滑的那一天。

弱小生物的镜像系统更简单，理论上更容易达到拐点。但弱小意味着天敌更多、生存压力更大，活到那一天的概率更低。

写时复制的对数曲线是永生的必要条件之一，不是充分条件。能否真正实现永生还取决于纠错能力、外部威胁、代谢稳定性等诸多因素，因个体而异。

更重要的是：生物在曲线趋于平滑之前，无法知道自己是否能够永生。这是一个只有回望才能确认的事实。你活了几千年，回头看，发现维护成本增长速度开始放缓了，还要再撑过数百数千年，才能让镜像开始修复历史缺陷

很多生物无法察觉自身存在镜像，但这种生物一般更难出现长生种

### 计算资源载体

镜像的运行需要神经计算资源。

有肉体的生物依赖神经。人类使用碳基神经，洋能生物使用洋能基底的神经组织。两者组成不同，原理也可能不同。

灵魂生命的情况特殊。灵魂本身只能维持意识存在和简单思考，无法独立完成镜像维护所需的复杂计算。灵魂生命仍然需要某种神经载体来运行镜像系统。

这意味着"纯洋能生物"不等于"不需要神经"。它们的神经基底是洋能构成的，但计算功能本质相同。

### 亚空间生物的资源困境

高级特性各自资源消耗不同，叠加后总成本极高。

亚空间的高智慧生物本质上是数值怪，它们的生存能力来自于对镜像特性的投资，而这些投资的回报是活得更久。但投资本身也有成本，代价就是智力普遍不高。
