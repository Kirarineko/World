# 镜像机制

> 每个人脑中都有一个"最初的自己"：只读快照。每一次成长、愈合、衰老，都在上面叠加一层写时复制。只要原始镜像还在，"我"就还在。

> 镜像不是免费的。它吃掉多少神经资源，决定了生物还剩多少智力可花。本地泡把所有能省的都省了，换回了智慧断崖以上的风景。

## 原始镜像

人类大脑中保存着一份"生命镜像"：对全身组织的完整只读快照。这份镜像定义了"完整的自己"：骨骼的排列、器官的结构、神经回路的初始拓扑。它不是基因：基因是设计图，镜像是竣工时的验收照片。

镜像以块为单位映射全身。每个块对应某个特定组织或器官区域：比如一块骨骼的特定节段、一块心肌的特定分区。

镜像块是只读的：只要还有一个引用指向它。

## 写时复制

身体的每一次自然生长、受伤后的疤痕形成、肌肉锻炼的痕迹：每一次改变都生成一个新的"差分层"。差分层不覆盖原始块：它写入新的数据块，然后将对应的虚拟地址重新映射到新块上。

原始块的引用计数减一。但不会归零：除非所有引用它的差分层都已经被更上层的差异重定向替代。

``` 
原始镜像块────→ 引用计数=3      
   ├── 差分层A（童年生长）   ────→ 引用该块      
   ├── 差分层B（骨折愈合）   ────→ 引用该块      
   └── 差分层C（肌肉锻炼）   ────→ 引用该块      
      
当差分层C 的更高层差异替代了它对原始块的引用：      
  原始镜像块────→ 引用计数=2      
  差分层C 不再指向原始块
```

## 镜像的日常轮换

镜像不是静止的。就像人的细胞约每七年完成一轮完全更换一样，镜像中的块也在持续的日常代谢中被释放和新写。

这不是损坏：是正常维护。旧块被释放，新的差分层写入同样的物理域，地址重新映射，身体在微观层面完成了翻新。一个人的镜像在今天和七年前已经不是同一组物理块，但它仍在映射同一个"自己"。就是因为映射关系不曾断裂。

> 镜像的本质不是块，是映射。块可以换，指针对就行。

## 治疗魔法

镜像可以作为一致性检查的基准。治疗魔法就是以镜像为基准的一次校验。

以镜像中对应块的校验和为基准，修复物理损伤：骨折对齐、组织愈合、血管重接。魔法能修复的，一定是"曾经完好的状态被破坏后的结果"。

它不能修复的东西也正好对应：

- **先天残疾 / 遗传病**：镜像本身就有这个"错误"，不是损坏，是源数据如此。`魔法`跑一遍，校验通过：那个"错误"就是正确的。
- **病毒感染**：外来代码不在镜像中。`魔法`不识别为损坏，不予清除。

> `魔法`不是万能医生。它是校对员：只纠正在源数据基础上发生的偏移。

## 跨越者：洋能覆写原始镜像

镜像块在日常轮换中持续被释放和重新映射。正常情况下，重新写入这些位置的新差分层依赖人体自身的生物代谢来维持：由身体提供能量，不依赖外力。

当一个人长期接触洋能，魔力回路和洋能代谢可能在镜像块的某次轮换中占据新写入的位置。这不是一个等待数年的缓慢过程：只需要原始镜像中某个特定块被自然释放的那一刻，洋能驱动的差分层恰好抢在生物代谢之前写入了同一地址。

如果这个块涉及某些功能性镜像（比如魔力回路的"传感器"区域），新差分层中混入了洋能依赖的映射关系，那么从这一刻起，这个人的镜像就不再纯粹依赖自身代谢。

这个过程可以在多个块上重复发生。每多一个洋能覆写的块，跨越的程度就深一层。

当洋能覆写的块足够多、足够核心，整个镜像的底层介质开始发生变化。这不是"吸收了很多魔力"——是"原来的自己有一部分被换成了洋能写的副本"。一旦外部洋能停止供给，这些块就会因为引用计数混乱而出现功能缺失。

这就是跨越者的定义：**生命镜像底层出现依赖洋能的差分层覆写。** 一旦失去洋能，镜像出现缺失。

跨越者不是吸收了多少魔力，不是达到了某个能层：是原始的某个块被洋能依赖层改写。

纯洋能生物的镜像完全不依赖碳基代谢——它们天生活在镜像底层就是洋能驱动的那一侧。

人类的定义：碳基生命 + 肉体完整 + 镜像未出现洋能底层覆写。

## 快照

原始镜像（出生时的一次性只读快照）定义了"完整的自己"。但如果这个原始镜像本身已经缺失了呢？

部分物种的镜像系统不依赖原始镜像。它们没有"出生时的完整快照"，取而代之的是运行过程中周期性生成的不完整参照点：快照。快照不同于原始镜像：它不是全量，只覆盖某些特定维度。它不是出生时一次性生成的，而是在生命过程中持续写入——每一次快照都可能比上一次更完整，也可能更残缺。

快照的作用是在镜像块丢失时提供回退基准。当某个块的引用计数归零、原始数据不复存在时，最近的快照中该块对应的数据被调入，作为新的基准重新开始写入差分层。回退的不是"出生的样子"，是"上一次快照时的样子"。

这意味着：以快照为基准的生物，无法被治疗魔法以"原始镜像"的校验和修复。因为不存在原始镜像。治疗魔法只能以最近的快照为基准，而快照本身可能就包含了缺陷。

> 快照不是备份。是不完整的自我记录。对这类生物而言，魔法治疗的本质是"把你修回上一次记得自己是什么样的时候"，而不是"把你修回最初的样子"。

## 分布式镜像

人类的大脑是统一架构：所有镜像数据在同一块神经硬件上运行，由同一套寻址指针定位。但有些物种走了另一条路。

它们的神经系统是分布式的。镜像不存储在单一器官中，它们分散在多个独立但同时运行的神经节点上。每一个节点管理镜像的一个子集：一个节点管骨骼框架，一个节点管表皮结构，一个节点管内脏排列。每个节点有自己的局部引用计数、自己的局部分层链、自己的局部寻址指针。

节点之间可以互相访问对方的镜像数据，但延迟更高、带宽更低。一个节点需要查询另一个节点上的块时，信号需要穿越更长的神经通路。这不影响日常运行（大部分时间各节点各自维护自己的子域），但在需要跨节点一致性校验时（比如治疗魔法试图以全镜像为基准修复损伤），跨节点访问的延迟会导致校验效率显著下降。

> 分布式镜像物种的魔法治疗比其它物种更慢、更耗能。数据不在同一块主板上。

分布式架构的另一个后果是：局部脑损伤不致命。摧毁一个节点，丢失的是该节点管理的镜像子集，而非全体。如果该子集恰好不涉及核心生命功能（比如丢失的是"表皮纹理"的节点），该生物可以继续存活，只是从此再也长不出原来的皮肤纹路了。

拥有独立多大脑器官的物种（物理意义上的多脑并存，而非单脑分区）是最典型的分布式镜像架构携带者。本地泡某些爬行类、深海头足类可能具备此形态；亚空间中多脑并行更为常见。

## 镜像冗余

绝大多数物种只保留一份镜像副本。一个块归零，该块对应的身体信息永久丢失。

镜像冗余物种保留了不止一份。

它们同时运行着两套镜像副本，覆盖完全相同的身体区域。两套副本之间持续校验：副本 A 说这块骨头是 35.2 厘米，副本 B 说这块骨头是 35.1 厘米，系统取多数意见并纠正少数。纠错系统在比对中自动修复偏差，无需外部魔法干预。

但这个机制是有代价的：运行两套镜像需要双份存储空间、双份差分层写入、以及持续不间断的校验通信。这套同步协议的能耗足以拖垮一台不擅长计算的大脑。因此镜像冗余在自然演化中极为罕见：大多数携带镜像冗余的物种都是亚空间原生生物，它们的代谢不依赖碳氢化合物的氧化释放，可以承受人类碳基大脑无法承受的计算成本。

> 镜像冗余物种是镜像机制上的"豪华版"。代价是终日巨大的进食量——或者根本不吃东西，靠洋能驱动。

镜像冗余物种在面对镜像缺失时具有天然优势。即使某个块在所有存储体中同时被释放了（极少见，因为各存储体的轮换周期通常不严格同步），只要有一个存储体还保留着该块的合法数据，镜像就是完整的。

## 子卷

人类的镜像是单卷：一块平坦的地址空间，所有镜像数据存在于同一个卷中，共享同一套引用计数域。卷不可分割：你不能单独对骨骼镜像做回滚而不影响皮肤镜像。

某些物种的镜像被划分为多个独立的子卷。每一个子卷有自己的地址空间、自己的引用计数域、自己的快照链。子卷之间逻辑上完全隔离：卷 A 的块释放不影响卷 B 的引用计数，卷 B 的差分层写入不会覆盖卷 A 的数据。

子卷结构的实用意义在于：可以独立操作子卷而不干扰其它卷。某物种可以对自己的表皮子卷做一个临时快照，从这个快照生成一个全新的差分层让表皮变化，而镜像的其它部分完全不变。事后如果这块表皮被破坏，回滚也仅限于该子卷内部，不影响骨骼、肌肉、神经。

子卷为形态多变物种提供了镜像层面的支持。一个生命周期中经历了若干次"破茧重组"的物种，可能每一次破茧都是一个子卷的快照回收和新卷挂载。旧的子卷被卸载但数据保留，新的子卷从快照继承了基础模板然后独立演化。

> 人类的镜像只有一个 C 盘。子卷物种的镜像可以挂载多个分区。

## 镜像与智力的资源调度

镜像不是免费运行的。原始镜像维护、差分层映射、引用计数、块释放与垃圾回收、纠错校验，这些操作消耗神经计算资源。每个物种的神经资源总量是有限的，镜像分走多少，智力就剩下多少。

在泡宇宙绝大多数泡中，生物把几乎所有神经资源投给了镜像活得久、缺陷少、纠错强—但智力难以翻不过"猫"附近的断崖。

本地泡走了另一条路。镜像被大幅精简：纠错更弱、引用计数精度更低、块释放更粗糙。省下的资源被重新配置给了通用智力。

## 镜像资源复杂度

镜像的每一个组件、每一种高级特性都需要消耗神经计算资源。不同特性消耗不同，且增长方式差异巨大。

### 各特性资源消耗

| 特性 | 资源消耗特征 |
|------|-------------|
| 原始镜像 | 一次性成本，固定不变 |
| 写时复制差分层 | 随时间放大的对数增长，极其漫长才能降到线性以下 |
| 引用计数 | 几乎固定，不随规模显著增长 |
| 日常轮换 | 与其他机制耦合，无法独立计算 |
| 弱纠错 | 视具体情况 |
| 快照 | 历史版本数的阶乘级膨胀，除非能自由清除旧版本 |
| 镜像冗余 | 简单翻倍 |
| 子卷本体 | 固定成本，但子卷上执行的操作消耗不定 |

快照的资源消耗取决于历史版本数。每增加一个快照，与旧快照的组合关系阶乘级膨胀。能否自由清除旧快照因个体而异，有些生物可以主动丢弃过期快照，有些则不能，这取决于物种的镜像架构和个体的神经系统特性。

### 写时复制与永生

写时复制是所有有镜像的生物共享的底层机制。差分层的资源消耗随时间增长，但增长曲线不是线性的：它是放大的对数型。

这条曲线是永生的关键瓶颈。

曲线需要极长时间才能降到线性以下，因物种而异。在拐点到来之前，镜像的维护成本持续高于维护能力，缺陷开始累积，镜像逐渐崩溃。这就是为什么大多数生物不能永生，它们的寿命不够长，撑不到曲线平滑的那一天。

弱小生物的镜像系统更简单，理论上更容易达到拐点。但弱小意味着天敌更多、生存压力更大，活到那一天的概率更低。强者镜像更重、曲线更高、拐点更远；弱者镜像更轻、曲线更低、拐点更近，但弱者死得更快。

写时复制的对数曲线是永生的必要条件之一，不是充分条件。能否真正实现永生还取决于纠错能力、外部威胁、代谢稳定性等诸多因素，因个体而异。

更重要的是：生物在曲线趋于平滑之前，无法知道自己是否能够永生。这是一个只有回望才能确认的事实。你活了几千年，回头看，发现维护成本增长速度开始放缓了，但你在第一千年的时候并不知道这一点。

### 计算资源载体

镜像的运行需要神经计算资源。

有肉体的生物依赖神经。人类使用碳基神经，洋能生物使用洋能基底的神经组织。两者组成不同，原理也可能不同，碳基神经通过离子通道和突触传递信号，洋能基底神经的信号机制未必相同。但在功能层面，两者都承担着镜像维护、差分层映射、引用计数等计算任务。

灵魂生命的情况更特殊。灵魂本身只能维持意识存在和简单思考，无法独立完成镜像维护所需的复杂计算。灵魂生命仍然需要某种神经载体来运行镜像系统——这个载体可以是残留的肉体组织、洋能构成的临时结构、或者其他形式的计算基底。

这意味着"纯洋能生物"不等于"不需要神经"。它们的神经基底是洋能构成的，但计算功能本质相同。

### 亚空间生物的资源困境

高级特性各自资源消耗不同，叠加后总成本极高。

纯洋能生物的代谢不依赖碳基化学，能承受更高的计算消耗，因此更有可能携带高级特性。但即使是纯洋能生物，资源也不是无限的。同时拥有快照替代基准、分布式多节点、全冗余双存储体、以及独立子卷隔离的全功能镜像，总消耗极高，自然界几乎不存在。

亚空间的高智慧生物本质上是数值怪——它们的智力和生存能力来自于对镜像特性的投资，而这些投资的回报是活得更久、缺陷更少、纠错更强。但投资本身也有成本：每多一种特性，神经资源就被多分走一份。在资源总量有限的前提下，特性组合的选择决定了物种的生存策略。

## 亚空间生物的镜像生态

上文列举的高级特性（快照、分布式镜像、镜像冗余、子卷）构成了每个物种镜像系统的功能图谱。不同物种拥有不同的特性组合。

**人类的这些特性是全部缺失的。** 基础的单机镜像（原始镜像 + 写时复制差分层 + 引用计数 + 日常轮换 + 弱纠错）不属于高级特性，是所有有镜像的生物共享的底层框架。而快照、分布式、冗余、子卷，这些是人类镜像完全不支持的功能。

**亚空间生物中，这些特性全部缺失的同样是极少数。** 多数亚空间物种具备其中一到两项。比如一个洋体巡游者可能没有子卷、没有冗余，但它的镜像依赖快照作为基准（原始镜像已丢失），同时它的感应器官分布在身体各处，构成了轻度的分布式结构。

**全部特性齐备的亚空间生物更是凤毛麟角。** 同时拥有快照替代基准、分布式多节点、全冗余双存储体、以及独立子卷隔离的镜像系统，在自然界中几乎不存在。即使存在，维持这样一套全功能镜像的能耗也远非任何自然代谢能承受——只有最高阶的洋能驱动生物才可能触及这个理论组合。
