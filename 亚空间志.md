
# 亚空间

> 有关本地泡的部分设定非当前时间线，如果严格按照当前时间线，应该对亚空间一无所知

## 本质

亚空间不是泡和泡之间塞进去的一层东西。它自己就是另一层维度，和任何一个泡都不共享同一套物理坐标。

泡壁外面是什么，没人真知道。

## 海洋隐喻

亚空间没有海面，但它确实像海。也只像到这里为止。拿海来比喻它很方便，也很容易把人带偏。

| 像海洋的 | 不像海洋的 |
|----------|------------|
| 有流、有浪、有涌 | 没有海面，也没有“之上” |
| 有深浅、有暗礁 | 深浅不是从海面算，是从维度界面算 |
| 有独立生态链 | 没有“海岸”和“陆地”这套分法，岛屿除外 |
| 有风暴 | 风暴不是天气，是维度本身在抖 |
| 可以航行 | 方向没有绝对的上下 |

> 你可以把它想成一片越潜越深的海。只是你潜到最后，会发现“向下”本身也开始失去意义。


## 核心参数

| 条目 | 说明 |
|------|------|
| 介质 | 非水，非气。半透明流体，密度不均，有层有流。可被能量推动 |
| 能见度 | 有限。依赖介质自身微光和生物光，无自然光源 |
| 压力 | 不存在"水压"。离维度界面越远，介质对意识的侵蚀越强：称为**洋压** |
| 方向 | 无绝对上下。以泡的维度界面为参照：界面是唯一的"岸" |
| 文明 | **极为稀少**。亚空间内大部分生命是无智慧或低智慧的简单生物 |


## 智慧断崖

整个泡宇宙（泡内和亚空间）存在一个普遍的生物学规律：

比猫聪明的生物极为罕见。能使用工具、形成语言、建立文明的智慧物种，在整个宇宙中屈指可数。人类的智力水平并非进化树上的常规分支，而是统计学意义上的极端离群值。

亚空间内也是如此。它拥有丰富的生态，但绝大多数生物停留在本能和简单认知的层面。能够被称为"智慧生命"的存在，在亚空间内同样是异数。

## 地理

### 复用区

泡与亚空间的连接点。不是物理上的"洞口"，而是泡壁上的维度映射区域：在此处进行维度转换，可以从泡内进入亚空间，反之亦然。

### 浅洋

| 特征 | 说明 |
|------|------|
| 介质密度 | 稀 |
| 洋压 | 低，短时间暴露几乎无影响 |
| 生态 | 浮游生物、小型浅洋物种 |
| 航行 | 近界面常规活动区域 |

### 中洋。

| 特征 | 说明 |
|------|------|
| 介质密度 | 中 |
| 洋压 | 中等，需能量护罩或高能层抗性 |
| 生态 | 丰富的独立生态群，大型巡游生物 |
| 航行 | 界面间航线必经 |

### 深渊

远离任何维度界面的黑暗洋域。介质浓稠如胶。

| 特征 | 说明 |
|------|------|
| 介质密度 | 极稠 |
| 洋压 | 强烈。长时间暴露导致"洋溺" |
| 生态 | 几乎无可靠记录。深渊种的存在被确认但未被系统研究 |
| 航行 | 无人涉足并完整返回 |

### 暗礁

亚空间中的固定或半固定构造。由固化介质、古代残骸或未知物质堆积而成。形状不定，位置可能缓慢漂移。

| 类型 | 说明 |
|------|------|
| 介质礁 | 介质自然凝结成的固体结构。多孔，部分内部有微生态 |
| 残骸礁 | 失事船只/生物遗骸/不明物体堆积。偶有可打捞物 |
| 活礁 | 群居生物构筑的活体结构，会生长，会移动 |

### 岛屿

亚空间内存在**岛屿**：质量不均匀、形状不规则的行星体，悬浮于洋体之中。

| 特征 | 说明 |
|------|------|
| 形态 | 不规则。不遵循球形天体规律，形状各异 |
| 大小 | 从小型岩块到行星级不等 |
| 质量分布 | 不均匀。某些区域密度极高，某些区域近乎中空 |
| 生态 | 独立的原生生态系统。与洋体生态不同，岛屿有自己的生物链 |

## 恶魔

"恶魔"是亚空间内**智慧原生生命**的统称。

名称来自最早接触他们的泡内人类：在那个时代，一切来自亚空间的智慧存在都被视为威胁。现代学术界承认这个称呼带有历史偏见，但已经约定俗成。

| 条目 | 说明 |
|------|------|
| 本质 | 亚空间原生智慧生命，不是堕落者，不是外来者 |
| 智力 | 智慧断崖之上的极少数亚空间物种。具备语言、社会、文化 |
| 居住地 | 不固定。有生活在洋体中的，有定居于岛屿的 |
| 生态 | 各自独立。不同岛屿/洋域的恶魔可能彼此从未接触，演化出完全不同的形态与社会 |
| 态度 | **不一定是恶意的。** 有敌对的，有中立的，有愿意交流的。"恶魔"这个名字是偏见，不是定义 |
| 多样性 | 不同种群之间的差异可能大于人类与章鱼的差异。不存在一个统一的"恶魔种族" |

> 本地泡对恶魔的了解极为有限。以上分类框架本身可能就有严重缺陷。

## 洋流、涌变与海啸

### 洋流

| 类型 | 特征 |
|------|------|
| 常流 | 稳定、可预测。界面间航线依赖常流 |
| 季流 | 周期性出现和消失。可能受某些泡的天体运动影响 |
| 暗涌 | 突发、无规律、多来自深渊方向。危险 |
| 界面流 | 沿维度界面表面流动的薄层洋流，适合短途 |

### 风暴。

| 等级 | 描述 |
|------|------|
| 微涌 | 颠簸，不影响航行 |
| 中涌 | 航线偏移，需加固护罩 |
| 烈涌 | 船体受损风险，洋压骤升，能见度归零 |
| 渊涌 | 从深渊方向袭来。无完整生还记录 |

### 海啸

> 海啸不是涌变的最高等级。海啸是另一种东西。

| 条目 | 说明 |
|------|------|     
| 形态 | 一道墙。一面由极致密介质构成的、横贯洋域的移动壁垒 |
| 规模 | 可席卷多个维度界面之间的整个洋域 |
| 速度 | 极快。几乎没有规避时间 |
| 洋压 | 致死级。接触即洋溺 |
| 数量 | **同一时刻恰好存在三道。** 不多不少。无人知晓为何是三，而非一或七。 |
| 频率 | 亚空间在三轴上均无边界，三道墙各自横贯不同洋域。遭遇概率在数学上趋近于零，但不为零。上一次有记载的海啸距离本地泡的时间尺度以万年计 |
| 预兆 | 无已知预兆。这是它最危险的地方 |
| 成因 | 完全未知。假说包括：深渊中的巨型存在活动、维度结构本身的周期性调整、某个远古文明的遗留机制。三道海啸是否共享同一成因、是否存在某种守恒律使其恰好为三，均无从考证 |

## 生态

亚空间的生物大部分是原生种，不是从泡内掉落或逃逸的生物。它们生于这片洋，死于这片洋，无法在泡内长期存活。

### 形态多样性

亚空间生物不存在统一的形体蓝图。泡内生物的身体结构受制于各自的演化路径：骨架、甲壳、软组织、对称性，而亚空间生物不受此限。

| 条目 | 说明 |
|------|------|
| 形态范围 | 从无定形流体到几何结构体，从丝状网络到脉动团块。命名（如"骨鱼""幕鳐"）仅为泡内人类根据外观类比所取，不代表生物学亲缘 |
| 对称性 | 不存在普遍的身体对称法则。一个亚空间生物可能一侧有三只感光器，另一侧有一张进食口，毫无对称可言 |
| 体态稳定性 | 部分物种终身维持同一形态，部分物种随洋流密度、洋压、年龄改变体态。后者在分类学上极难处理：同一物种可能被记录为三种不同生物 |
| 人形存在 | **确认存在，但极为罕见。** 有零星的可靠目击记录，描述为外形接近人类轮廓的生物。是否具备智慧、是否与恶魔有关、是否与泡内人类存在某种未知关联：均无定论 |

> 现有的"浮游层/巡游者/深渊种"分类，不过是泡内学者根据体型、栖息深度和行为模式强行划分的框架。它有用，但不真实。亚空间不在乎林奈。

### 浮游层

| 生物 | 描述 |
|------|------|
| 光屑 | 食物链基础。微小光点，随流漂移，被滤食者吞食 |
| 流丝 | 细丝状半透明体，成群缠绕成网，捕捉碎屑 |
| 浮囊 | 小型囊状体，通过密度调节在介质中升降 |

### 巡游者

| 生物 | 描述 |
|------|------|
| 流影 | 扁平，滑行于流层之间。无害。常被用作导航参考：流影群总沿稳定洋流行进 |
| 骨鱼 | 命名源于其半透明体内可见的能量骨架。掠食性，攻击小型目标 |
| 幕鳐 | 宽大薄片状，如水中飞毯。温顺，偶被驯化用作短途载具 |
| 裂口鲸 | 航线主要威胁之一。体型巨大，吞吸式进食，可吞下小型船 |

### 深渊种

确认存在，几乎无可靠记录。

| 记录 | 描述 |
|------|------|
| 渊光 | 深渊中偶尔出现的巨大光斑，缓慢移动。生物或现象，无定论 |
| 古声 | 深渊方向传来的低频振动，似心跳、似吟唱。录音设备无法捕捉 |

### 岛屿生态

岛屿拥有独立于洋体的生态系统。目前已知的信息不足以建立系统分类。恶魔的栖息地生态是其主要研究方向，但受限于接触机会极少。

### 外来种

极少数生物并非亚空间原生，而是从某个泡内进入并存活下来。

| 条目 | 说明 |
|------|------|
| 来源 | 可能通过天然裂隙、界面事故或远古遗弃的载具进入亚空间 |
| 数量 | **极为稀少**。绝大部分泡内生物进入亚空间后无法存活：介质依赖倒置，洋压致命 |
| 存活条件 | 侥幸落在维度界面附近、介质极稀薄的浅洋区域；或发生了未知的适应性变异 |
| 已知记录 | 寥寥无几。多为小型生物，未发现泡内智慧生命在亚空间中建立稳定种群 |
| 学术价值 | 极高。每一个外来种标本都可能揭示泡内生物学与亚空间生态之间的深层关联 |

> 没有证据表明泡内智慧文明曾成功在亚空间中殖民或建立永久定居。岛屿上的恶魔生态是原生演化，与泡内无关。


## 航行

### 载具

| 类型 | 说明 |
|------|------|
| 介质船 | 主流。以能量推动，船体刻有稳定介质密度的铭文 |
| 肉艇 | 将大型亚空间生物改造/驯化为载具。少数势力使用 |
| 肉身 | 能层烈以上可短时在浅洋活动。中洋需更高能层或特殊体质（仅人类） |

### 导航

| 方法 | 说明 |
|------|------|
| 界面信标 | 固定于维度界面附近的信号装置。近界面导航的基础 |
| 洋流图 | 记录常流路径的航海图。经验积累的产物，代代修订 |
| 流影追踪 | 跟随流影群：它们沿稳定洋流迁移 |
| 密度感知 | 有经验者通过介质密度变化判断方向 |
| 深渊方向 | **永远不要朝密度持续增大的方向航行** |

### 风险汇总

| 风险 | 说明 |
|------|------|
| 洋溺 | 高洋压长时间暴露。意识模糊、失向、消散。肉身可能存活，人已不是原来的人 |
| 迷航 | 失去界面信标和洋流参照。在无参照介质中，方向感会在数小时内完全丧失。涌变后最常见 |
| 生物威胁 | 裂口鲸、掠食性鱼群、未知深渊种 |
| 暗礁碰撞 | 低速可规避，高速或涌变中致命 |
| 海啸 | 无规避手段。遭遇即终局 |
| 介质蚀 | 对船体和护罩的持续性消磨。长途航行必须定期维护 |

## 能量本源

> 所有泡内的魔法力量体系，能量最终都来自亚空间。

这是泡宇宙中唯一统一的物理规律（不论泡内文明如何独立演化，当他们发现并系统化魔法时，他们接触的是同一种能量）亚空间透过泡壁渗入的洋能。

| 推论 | 说明 |
|------|------|
| 能层能级体系在各泡通用 | 因为能量本源相同 |
| 跨越者在亚空间中更强 | 曲态（g）可感知洋流，坍态（h）传闻可制造临时空泡 |
| 洋压抗性 | 能层越高，抗性越强。本质上是对同源能量的适应性 |
| 能量锁（N→O） | 可能与亚空间能量对纯人类身体的深层改造阈值有关（穿越能量锁，本质上是身体开始"亚空间化" |
| 潮汐假说 | 学术界未经证实的猜想）涌变周期与某些泡内的高能魔法使用量存在关联 |

### 洋能渗透

亚空间能量透过泡壁的维度界面持续向泡内渗透。**渗透率因泡而异：从零到丰沛不等。** 渗透率为零的泡内部不存在洋能，其文明（如果有的话）永远无法发展出魔法或高武体系。不同泡的渗透率差异影响该泡的魔法文明发展速度和力量上限。
